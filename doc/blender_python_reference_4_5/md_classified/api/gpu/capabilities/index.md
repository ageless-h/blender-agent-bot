# GPU Capabilities Utilities (gpu.capabilities)[¶](#module-gpu.capabilities "Link to this heading")

This module provides access to the GPU capabilities.

gpu.capabilities.compute\_shader\_support\_get()[¶](#gpu.capabilities.compute_shader_support_get "Link to this definition")
:   Are compute shaders supported.

    Returns:
    :   True when supported, False when not supported.

    Return type:
    :   bool

gpu.capabilities.extensions\_get()[¶](#gpu.capabilities.extensions_get "Link to this definition")
:   Get supported extensions in the current context.

    Returns:
    :   Extensions.

    Return type:
    :   tuple[str]

gpu.capabilities.hdr\_support\_get()[¶](#gpu.capabilities.hdr_support_get "Link to this definition")
:   Return whether GPU backend supports High Dynamic range for viewport.

    > return:
    > :   HDR support available.
    >
    > rtype:
    > :   bool

gpu.capabilities.max\_batch\_indices\_get()[¶](#gpu.capabilities.max_batch_indices_get "Link to this definition")
:   Get maximum number of vertex array indices.

    Returns:
    :   Number of indices.

    Return type:
    :   int

gpu.capabilities.max\_batch\_vertices\_get()[¶](#gpu.capabilities.max_batch_vertices_get "Link to this definition")
:   Get maximum number of vertex array vertices.

    Returns:
    :   Number of vertices.

    Return type:
    :   int

gpu.capabilities.max\_images\_get()[¶](#gpu.capabilities.max_images_get "Link to this definition")
:   Get maximum supported number of image units.

    Returns:
    :   Number of image units.

    Return type:
    :   int

gpu.capabilities.max\_texture\_layers\_get()[¶](#gpu.capabilities.max_texture_layers_get "Link to this definition")
:   Get maximum number of layers in texture.

    Returns:
    :   Number of layers.

    Return type:
    :   int

gpu.capabilities.max\_texture\_size\_get()[¶](#gpu.capabilities.max_texture_size_get "Link to this definition")
:   Get estimated maximum texture size to be able to handle.

    Returns:
    :   Texture size.

    Return type:
    :   int

gpu.capabilities.max\_textures\_frag\_get()[¶](#gpu.capabilities.max_textures_frag_get "Link to this definition")
:   Get maximum supported texture image units used for
    accessing texture maps from the fragment shader.

    Returns:
    :   Texture image units.

    Return type:
    :   int

gpu.capabilities.max\_textures\_geom\_get()[¶](#gpu.capabilities.max_textures_geom_get "Link to this definition")
:   Get maximum supported texture image units used for
    accessing texture maps from the geometry shader.

    Returns:
    :   Texture image units.

    Return type:
    :   int

gpu.capabilities.max\_textures\_get()[¶](#gpu.capabilities.max_textures_get "Link to this definition")
:   Get maximum supported texture image units used for
    accessing texture maps from the vertex shader and the
    fragment processor.

    Returns:
    :   Texture image units.

    Return type:
    :   int

:   Get maximum supported texture image units used for

    Returns:
    :   Texture image units.

    Return type:
    :   int

:   Get maximum number of values held in uniform variable
    storage for a fragment shader.

    Returns:
    :   Number of values.

    Return type:
    :   int

gpu.capabilities.max\_uniforms\_vert\_get()[¶](#gpu.capabilities.max_uniforms_vert_get "Link to this definition")
:   Get maximum number of values held in uniform variable

    Returns:
    :   Number of values.

    Return type:
    :   int

    Returns:
    :   Number of variables.

    Return type:
    :   int

    :   int

    Return type:
    :   int
