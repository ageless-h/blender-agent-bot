# VertexWeightProximityModifier(Modifier)[¶](#vertexweightproximitymodifier-modifier "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`Modifier`](Modifier.md#bpy.types.Modifier "bpy.types.Modifier")

*class* bpy.types.VertexWeightProximityModifier(*Modifier*)[¶](#bpy.types.VertexWeightProximityModifier "Link to this definition")
:   Set the weights of vertices in a group from a target object’s distance

    falloff\_type[¶](#bpy.types.VertexWeightProximityModifier.falloff_type "Link to this definition")
    :   How weights are mapped to their new values

        * `LINEAR`
          Linear – Null action.
        * `CURVE`
          Custom Curve.
        * `SHARP`
          Sharp.
        * `SMOOTH`
          Smooth.
        * `ROOT`
          Root.
        * `ICON_SPHERECURVE`
          Sphere.
        * `RANDOM`
          Random.
        * `STEP`
          Median Step – Map all values below 0.5 to 0.0, and all others to 1.0.

        Type:
        :   enum in [`'LINEAR'`, `'CURVE'`, `'SHARP'`, `'SMOOTH'`, `'ROOT'`, `'ICON_SPHERECURVE'`, `'RANDOM'`, `'STEP'`], default `'LINEAR'`

    invert\_falloff[¶](#bpy.types.VertexWeightProximityModifier.invert_falloff "Link to this definition")
    :   Invert the resulting falloff weight

        Type:
        :   boolean, default False

    invert\_mask\_vertex\_group[¶](#bpy.types.VertexWeightProximityModifier.invert_mask_vertex_group "Link to this definition")
    :   Invert vertex group mask influence

        Type:
        :   boolean, default False

    map\_curve[¶](#bpy.types.VertexWeightProximityModifier.map_curve "Link to this definition")
    :   Custom mapping curve

        Type:
        :   [`CurveMapping`](CurveMapping.md#bpy.types.CurveMapping "bpy.types.CurveMapping"), (readonly)

    mask\_constant[¶](#bpy.types.VertexWeightProximityModifier.mask_constant "Link to this definition")
    :   Global influence of current modifications on vgroup

        Type:
        :   float in [-inf, inf], default 1.0

    mask\_tex\_map\_bone[¶](#bpy.types.VertexWeightProximityModifier.mask_tex_map_bone "Link to this definition")
    :   Which bone to take texture coordinates from

        Type:
        :   string, default “”, (never None)

    mask\_tex\_map\_object[¶](#bpy.types.VertexWeightProximityModifier.mask_tex_map_object "Link to this definition")
    :   Which object to take texture coordinates from

        Type:
        :   [`Object`](Object.md#bpy.types.Object "bpy.types.Object")

    mask\_tex\_mapping[¶](#bpy.types.VertexWeightProximityModifier.mask_tex_mapping "Link to this definition")
    :   Which texture coordinates to use for mapping

        * `LOCAL`
          Local – Use local generated coordinates.
        * `GLOBAL`
          Global – Use global coordinates.
        * `OBJECT`
          Object – Use local generated coordinates of another object.
        * `UV`
          UV – Use coordinates from a UV layer.

        Type:
        :   enum in [`'LOCAL'`, `'GLOBAL'`, `'OBJECT'`, `'UV'`], default `'LOCAL'`

    mask\_tex\_use\_channel[¶](#bpy.types.VertexWeightProximityModifier.mask_tex_use_channel "Link to this definition")
    :   Which texture channel to use for masking

        Type:
        :   enum in [`'INT'`, `'RED'`, `'GREEN'`, `'BLUE'`, `'HUE'`, `'SAT'`, `'VAL'`, `'ALPHA'`], default `'INT'`

    mask\_tex\_uv\_layer[¶](#bpy.types.VertexWeightProximityModifier.mask_tex_uv_layer "Link to this definition")
    :   UV map name

        Type:
        :   string, default “”, (never None)

    mask\_texture[¶](#bpy.types.VertexWeightProximityModifier.mask_texture "Link to this definition")
    :   Masking texture

        Type:
        :   [`Texture`](Texture.md#bpy.types.Texture "bpy.types.Texture")

    mask\_vertex\_group[¶](#bpy.types.VertexWeightProximityModifier.mask_vertex_group "Link to this definition")
    :   Masking vertex group name

        Type:
        :   string, default “”, (never None)

    max\_dist[¶](#bpy.types.VertexWeightProximityModifier.max_dist "Link to this definition")
    :   Distance mapping to weight 1.0

        Type:
        :   float in [0, inf], default 1.0

    min\_dist[¶](#bpy.types.VertexWeightProximityModifier.min_dist "Link to this definition")
    :   Distance mapping to weight 0.0

        Type:
        :   float in [0, inf], default 0.0

    normalize[¶](#bpy.types.VertexWeightProximityModifier.normalize "Link to this definition")
    :   Normalize the resulting weights (otherwise they are only clamped within 0.0 to 1.0 range)

        Type:
        :   boolean, default False

    proximity\_geometry[¶](#bpy.types.VertexWeightProximityModifier.proximity_geometry "Link to this definition")
    :   Use the shortest computed distance to target object’s geometry as weight

        * `VERTEX`
          Vertex – Compute distance to nearest vertex.
        * `EDGE`
          Edge – Compute distance to nearest edge.
        * `FACE`
          Face – Compute distance to nearest face.

        Type:
        :   enum set in {`'VERTEX'`, `'EDGE'`, `'FACE'`}, default {`'FACE'`}

    proximity\_mode[¶](#bpy.types.VertexWeightProximityModifier.proximity_mode "Link to this definition")
    :   Which distances to target object to use

        * `OBJECT`
          Object – Use distance between affected and target objects.
        * `GEOMETRY`
          Geometry – Use distance between affected object’s vertices and target object, or target object’s geometry.

        Type:
        :   enum in [`'OBJECT'`, `'GEOMETRY'`], default `'GEOMETRY'`

    target[¶](#bpy.types.VertexWeightProximityModifier.target "Link to this definition")
    :   Object to calculate vertices distances from

        Type:
        :   [`Object`](Object.md#bpy.types.Object "bpy.types.Object")

    vertex\_group[¶](#bpy.types.VertexWeightProximityModifier.vertex_group "Link to this definition")
    :   Vertex group name

        Type:
        :   string, default “”, (never None)
