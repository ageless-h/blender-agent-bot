# OceanModifier(Modifier)[¶](#oceanmodifier-modifier "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`Modifier`](Modifier.md#bpy.types.Modifier "bpy.types.Modifier")

*class* bpy.types.OceanModifier(*Modifier*)[¶](#bpy.types.OceanModifier "Link to this definition")
:   Simulate an ocean surface

    bake\_foam\_fade[¶](#bpy.types.OceanModifier.bake_foam_fade "Link to this definition")
    :   How much foam accumulates over time (baked ocean only)

        Type:
        :   float in [0, inf], default 0.98

    choppiness[¶](#bpy.types.OceanModifier.choppiness "Link to this definition")
    :   Choppiness of the wave’s crest (adds some horizontal component to the displacement)

        Type:
        :   float in [0, inf], default 1.0

    damping[¶](#bpy.types.OceanModifier.damping "Link to this definition")
    :   Damp reflected waves going in opposite direction to the wind

        Type:
        :   float in [0, 1], default 0.5

    depth[¶](#bpy.types.OceanModifier.depth "Link to this definition")
    :   Depth of the solid ground below the water surface

        Type:
        :   float in [-inf, inf], default 200.0

    fetch\_jonswap[¶](#bpy.types.OceanModifier.fetch_jonswap "Link to this definition")
    :   This is the distance from a lee shore, called the fetch, or the distance over which the wind blows with constant velocity. Used by ‘JONSWAP’ and ‘TMA’ models.

        Type:
        :   float in [0, inf], default 120.0

    filepath[¶](#bpy.types.OceanModifier.filepath "Link to this definition")
    :   Path to a folder to store external baked images

        Type:
        :   string, default “”, (never None, blend relative `//` prefix supported)

    foam\_coverage[¶](#bpy.types.OceanModifier.foam_coverage "Link to this definition")
    :   Amount of generated foam

        Type:
        :   float in [-inf, inf], default 0.0

    foam\_layer\_name[¶](#bpy.types.OceanModifier.foam_layer_name "Link to this definition")
    :   Name of the vertex color layer used for foam

        Type:
        :   string, default “”, (never None)

    frame\_end[¶](#bpy.types.OceanModifier.frame_end "Link to this definition")
    :   End frame of the ocean baking

        Type:
        :   int in [-inf, inf], default 250

    frame\_start[¶](#bpy.types.OceanModifier.frame_start "Link to this definition")
    :   Start frame of the ocean baking

        Type:
        :   int in [-inf, inf], default 1

    geometry\_mode[¶](#bpy.types.OceanModifier.geometry_mode "Link to this definition")
    :   Method of modifying geometry

        * `GENERATE`
          Generate – Generate ocean surface geometry at the specified resolution.
        * `DISPLACE`
          Displace – Displace existing geometry according to simulation.

        Type:
        :   enum in [`'GENERATE'`, `'DISPLACE'`], default `'GENERATE'`

    invert\_spray[¶](#bpy.types.OceanModifier.invert_spray "Link to this definition")
    :   Invert the spray direction map

        Type:
        :   boolean, default False

    is\_cached[¶](#bpy.types.OceanModifier.is_cached "Link to this definition")
    :   Whether the ocean is using cached data or simulating

        Type:
        :   boolean, default False, (readonly)

    random\_seed[¶](#bpy.types.OceanModifier.random_seed "Link to this definition")
    :   Seed of the random generator

        Type:
        :   int in [0, inf], default 0

    repeat\_x[¶](#bpy.types.OceanModifier.repeat_x "Link to this definition")
    :   Repetitions of the generated surface in X

        Type:
        :   int in [1, 1024], default 1

    repeat\_y[¶](#bpy.types.OceanModifier.repeat_y "Link to this definition")
    :   Repetitions of the generated surface in Y

        Type:
        :   int in [1, 1024], default 1

    resolution[¶](#bpy.types.OceanModifier.resolution "Link to this definition")
    :   Resolution of the generated surface for rendering and baking

        Type:
        :   int in [1, 1024], default 7

    sharpen\_peak\_jonswap[¶](#bpy.types.OceanModifier.sharpen_peak_jonswap "Link to this definition")
    :   Peak sharpening for ‘JONSWAP’ and ‘TMA’ models

        Type:
        :   float in [0, 1], default 0.0

    size[¶](#bpy.types.OceanModifier.size "Link to this definition")
    :   Surface scale factor (does not affect the height of the waves)

        Type:
        :   float in [0, inf], default 1.0

    spatial\_size[¶](#bpy.types.OceanModifier.spatial_size "Link to this definition")
    :   Size of the simulation domain (in meters), and of the generated geometry (in BU)

        Type:
        :   int in [-inf, inf], default 50

    spectrum[¶](#bpy.types.OceanModifier.spectrum "Link to this definition")
    :   Spectrum to use

        * `PHILLIPS`
          Turbulent Ocean – Use for turbulent seas with foam.
        * `PIERSON_MOSKOWITZ`
          Established Ocean – Use for a large area, established ocean (Pierson-Moskowitz method).
        * `JONSWAP`
          Established Ocean (Sharp Peaks) – Use for established oceans (‘JONSWAP’, Pierson-Moskowitz method) with peak sharpening.
        * `TEXEL_MARSEN_ARSLOE`
          Shallow Water – Use for shallow water (‘JONSWAP’, ‘TMA’ - Texel-Marsen-Arsloe method).

        Type:
        :   enum in [`'PHILLIPS'`, `'PIERSON_MOSKOWITZ'`, `'JONSWAP'`, `'TEXEL_MARSEN_ARSLOE'`], default `'PHILLIPS'`

    spray\_layer\_name[¶](#bpy.types.OceanModifier.spray_layer_name "Link to this definition")
    :   Name of the vertex color layer used for the spray direction map

        Type:
        :   string, default “”, (never None)

    time[¶](#bpy.types.OceanModifier.time "Link to this definition")
    :   Current time of the simulation

        Type:
        :   float in [0, inf], default 1.0

    use\_foam[¶](#bpy.types.OceanModifier.use_foam "Link to this definition")
    :   Generate foam mask as a vertex color channel

        Type:
        :   boolean, default False

    use\_normals[¶](#bpy.types.OceanModifier.use_normals "Link to this definition")
    :   Output normals for bump mapping - disabling can speed up performance if it’s not needed

        Type:
        :   boolean, default False

    use\_spray[¶](#bpy.types.OceanModifier.use_spray "Link to this definition")
    :   Generate map of spray direction as a vertex color channel

        Type:
        :   boolean, default False

    viewport\_resolution[¶](#bpy.types.OceanModifier.viewport_resolution "Link to this definition")
    :   Viewport resolution of the generated surface

        Type:
        :   int in [1, 1024], default 7

    wave\_alignment[¶](#bpy.types.OceanModifier.wave_alignment "Link to this definition")
    :   How much the waves are aligned to each other

        Type:
        :   float in [0, 1], default 0.0

    wave\_direction[¶](#bpy.types.OceanModifier.wave_direction "Link to this definition")
    :   Main direction of the waves when they are (partially) aligned

        Type:
        :   float in [-inf, inf], default 0.0

    wave\_scale[¶](#bpy.types.OceanModifier.wave_scale "Link to this definition")
    :   Scale of the displacement effect

        Type:
        :   float in [0, inf], default 1.0

    wave\_scale\_min[¶](#bpy.types.OceanModifier.wave_scale_min "Link to this definition")
    :   Shortest allowed wavelength

        Type:
        :   float in [0, inf], default 0.01

    wind\_velocity[¶](#bpy.types.OceanModifier.wind_velocity "Link to this definition")
    :   Wind speed

        Type:
        :   float in [-inf, inf], default 30.0

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.OceanModifier.bl_rna_get_subclass "Link to this definition")
