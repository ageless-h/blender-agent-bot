# PreferencesEdit(bpy\_struct)[¶](#preferencesedit-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.PreferencesEdit(*bpy\_struct*)[¶](#bpy.types.PreferencesEdit "Link to this definition")
:   Settings for interacting with Blender data

    auto\_keying\_mode[¶](#bpy.types.PreferencesEdit.auto_keying_mode "Link to this definition")
    :   Mode of automatic keyframe insertion for Objects and Bones (default setting used for new Scenes)

        Type:
        :   enum in [`'ADD_REPLACE_KEYS'`, `'REPLACE_KEYS'`], default `'ADD_REPLACE_KEYS'`

    collection\_instance\_empty\_size[¶](#bpy.types.PreferencesEdit.collection_instance_empty_size "Link to this definition")
    :   Display size of the empty when new collection instances are created

        Type:
        :   float in [0.001, inf], default 1.0

    connect\_strips\_by\_default[¶](#bpy.types.PreferencesEdit.connect_strips_by_default "Link to this definition")
    :   Connect newly added movie strips by default if they have multiple channels

        Type:
        :   boolean, default True

    fcurve\_new\_auto\_smoothing[¶](#bpy.types.PreferencesEdit.fcurve_new_auto_smoothing "Link to this definition")
    :   Auto Handle Smoothing mode used for newly added F-Curves

        Type:
        :   enum in [Fcurve Auto Smoothing Items](../../../appendix/bpy_types_enum_items/fcurve_auto_smoothing_items.md#rna-enum-fcurve-auto-smoothing-items), default `'CONT_ACCEL'`

    fcurve\_unselected\_alpha[¶](#bpy.types.PreferencesEdit.fcurve_unselected_alpha "Link to this definition")
    :   The opacity of unselected F-Curves against the background of the Graph Editor

        Type:
        :   float in [0.001, 1], default 0.25

    grease\_pencil\_default\_color[¶](#bpy.types.PreferencesEdit.grease_pencil_default_color "Link to this definition")
    :   Color of new annotation layers

        Type:
        :   float array of 4 items in [0, inf], default (0.38, 0.61, 0.78, 0.9)

    grease\_pencil\_eraser\_radius[¶](#bpy.types.PreferencesEdit.grease_pencil_eraser_radius "Link to this definition")
    :   Radius of eraser ‘brush’

        Type:
        :   int in [1, 500], default 25

    grease\_pencil\_euclidean\_distance[¶](#bpy.types.PreferencesEdit.grease_pencil_euclidean_distance "Link to this definition")
    :   Distance moved by mouse when drawing stroke to include

        Type:
        :   int in [0, 100], default 2

    grease\_pencil\_manhattan\_distance[¶](#bpy.types.PreferencesEdit.grease_pencil_manhattan_distance "Link to this definition")
    :   Pixels moved by mouse per axis when drawing stroke

        Type:
        :   int in [0, 100], default 1

    key\_insert\_channels[¶](#bpy.types.PreferencesEdit.key_insert_channels "Link to this definition")
    :   Which channels to insert keys at when no keying set is active

        Type:
        :   enum set in {`'LOCATION'`, `'ROTATION'`, `'SCALE'`, `'ROTATE_MODE'`, `'CUSTOM_PROPS'`}, default {`'CUSTOM_PROPS'`, `'LOCATION'`, `'ROTATION'`, `'SCALE'`}

    keyframe\_new\_handle\_type[¶](#bpy.types.PreferencesEdit.keyframe_new_handle_type "Link to this definition")
    :   Handle type for handles of new keyframes

        Type:
        :   enum in [Keyframe Handle Type Items](../../../appendix/bpy_types_enum_items/keyframe_handle_type_items.md#rna-enum-keyframe-handle-type-items), default `'AUTO_CLAMPED'`

    keyframe\_new\_interpolation\_type[¶](#bpy.types.PreferencesEdit.keyframe_new_interpolation_type "Link to this definition")
    :   Interpolation mode used for first keyframe on newly added F-Curves (subsequent keyframes take interpolation from preceding keyframe)

        Type:
        :   enum in [Beztriple Interpolation Mode Items](../../../appendix/bpy_types_enum_items/beztriple_interpolation_mode_items.md#rna-enum-beztriple-interpolation-mode-items), default `'BEZIER'`

    material\_link[¶](#bpy.types.PreferencesEdit.material_link "Link to this definition")
    :   Toggle whether the material is linked to object data or the object block

        * `OBDATA`
          Object Data – Toggle whether the material is linked to object data or the object block.
        * `OBJECT`
          Object – Toggle whether the material is linked to object data or the object block.

        Type:
        :   enum in [`'OBDATA'`, `'OBJECT'`], default `'OBDATA'`

    node\_margin[¶](#bpy.types.PreferencesEdit.node_margin "Link to this definition")
    :   Minimum distance between nodes for Auto-offsetting nodes

        Type:
        :   int in [0, 255], default 40

    node\_preview\_resolution[¶](#bpy.types.PreferencesEdit.node_preview_resolution "Link to this definition")
    :   Resolution used for Shader node previews (should be changed for performance convenience)

        Type:
        :   int in [50, 250], default 120

    node\_use\_insert\_offset[¶](#bpy.types.PreferencesEdit.node_use_insert_offset "Link to this definition")
    :   Automatically offset the following or previous nodes in a chain when inserting a new node

        Type:
        :   boolean, default True

    object\_align[¶](#bpy.types.PreferencesEdit.object_align "Link to this definition")
    :   The default alignment for objects added from a 3D viewport menu

        * `WORLD`
          World – Align newly added objects to the world coordinate system.
        * `VIEW`
          View – Align newly added objects to the active 3D view orientation.
        * `CURSOR`
          3D Cursor – Align newly added objects to the 3D Cursor’s rotation.

        Type:
        :   enum in [`'WORLD'`, `'VIEW'`, `'CURSOR'`], default `'WORLD'`

    sculpt\_paint\_overlay\_color[¶](#bpy.types.PreferencesEdit.sculpt_paint_overlay_color "Link to this definition")
    :   Color of texture overlay

        Type:
        :   [`mathutils.Color`](../../mathutils/index.md#mathutils.Color "mathutils.Color") of 3 items in [0, inf], default (0.0, 0.0, 0.0)

    show\_only\_selected\_curve\_keyframes[¶](#bpy.types.PreferencesEdit.show_only_selected_curve_keyframes "Link to this definition")
    :   Only keyframes of selected F-Curves are visible and editable

        Type:
        :   boolean, default False

    undo\_memory\_limit[¶](#bpy.types.PreferencesEdit.undo_memory_limit "Link to this definition")
    :   Maximum memory usage in megabytes (0 means unlimited)

        Type:
        :   int in [0, inf], default 0

    undo\_steps[¶](#bpy.types.PreferencesEdit.undo_steps "Link to this definition")

        Type:
        :   int in [0, 256], default 32

    use\_anim\_channel\_group\_colors[¶](#bpy.types.PreferencesEdit.use_anim_channel_group_colors "Link to this definition")

        Type:
        :   boolean, default False

    use\_auto\_keyframe\_insert\_needed[¶](#bpy.types.PreferencesEdit.use_auto_keyframe_insert_needed "Link to this definition")

        :   boolean, default True
