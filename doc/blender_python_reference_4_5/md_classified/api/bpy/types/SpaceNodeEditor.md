*class* bpy.types.SpaceNodeEditor(*Space*)[¶](#bpy.types.SpaceNodeEditor "Link to this definition")
:   Node editor space data

    backdrop\_channels[¶](#bpy.types.SpaceNodeEditor.backdrop_channels "Link to this definition")
    :   Channels of the image to draw

        * `COLOR_ALPHA`
          Color & Alpha – Display image with RGB colors and alpha transparency.
        * `COLOR`
          Color – Display image with RGB colors.
        * `ALPHA`
          Alpha – Display alpha transparency channel.
        * `RED`
          Red.
        * `GREEN`
          Green.
        * `BLUE`
          Blue.

        Type:
        :   enum in [`'COLOR_ALPHA'`, `'COLOR'`, `'ALPHA'`, `'RED'`, `'GREEN'`, `'BLUE'`], default `'COLOR'`

    backdrop\_offset[¶](#bpy.types.SpaceNodeEditor.backdrop_offset "Link to this definition")
    :   Backdrop offset

        Type:
        :   float array of 2 items in [-inf, inf], default (0.0, 0.0)

    backdrop\_zoom[¶](#bpy.types.SpaceNodeEditor.backdrop_zoom "Link to this definition")
    :   Backdrop zoom factor

        Type:
        :   float in [0.01, inf], default 1.0

    cursor\_location[¶](#bpy.types.SpaceNodeEditor.cursor_location "Link to this definition")
    :   Location for adding new nodes

        Type:
        :   [`mathutils.Vector`](../../mathutils/index.md#mathutils.Vector "mathutils.Vector") of 2 items in [-inf, inf], default (0.0, 0.0)

    edit\_tree[¶](#bpy.types.SpaceNodeEditor.edit_tree "Link to this definition")
    :   Node tree being displayed and edited

        Type:
        :   [`NodeTree`](NodeTree.md#bpy.types.NodeTree "bpy.types.NodeTree"), (readonly)

    geometry\_nodes\_tool\_tree[¶](#bpy.types.SpaceNodeEditor.geometry_nodes_tool_tree "Link to this definition")
    :   Node group to edit as node tool

        Type:
        :   [`NodeTree`](NodeTree.md#bpy.types.NodeTree "bpy.types.NodeTree")

    geometry\_nodes\_type[¶](#bpy.types.SpaceNodeEditor.geometry_nodes_type "Link to this definition")
    :   * `MODIFIER`
          Modifier – Edit node group from active object’s active modifier.
        * `TOOL`
          Tool – Edit any geometry node group for use as an operator.

        Type:
        :   enum in [`'MODIFIER'`, `'TOOL'`], default `'MODIFIER'`

    id[¶](#bpy.types.SpaceNodeEditor.id "Link to this definition")
    :   Data-block whose nodes are being edited

        Type:
        :   [`ID`](ID.md#bpy.types.ID "bpy.types.ID"), (readonly)

    id\_from[¶](#bpy.types.SpaceNodeEditor.id_from "Link to this definition")

        Type:
        :   [`ID`](ID.md#bpy.types.ID "bpy.types.ID"), (readonly)

    insert\_offset\_direction[¶](#bpy.types.SpaceNodeEditor.insert_offset_direction "Link to this definition")
    :   Direction to offset nodes on insertion

        Type:
        :   enum in [`'RIGHT'`, `'LEFT'`], default `'RIGHT'`

    node\_tree[¶](#bpy.types.SpaceNodeEditor.node_tree "Link to this definition")
    :   Base node tree from context

        Type:
        :   [`NodeTree`](NodeTree.md#bpy.types.NodeTree "bpy.types.NodeTree")

    overlay[¶](#bpy.types.SpaceNodeEditor.overlay "Link to this definition")
    :   Settings for display of overlays in the Node Editor

        Type:
        :   [`SpaceNodeOverlay`](SpaceNodeOverlay.md#bpy.types.SpaceNodeOverlay "bpy.types.SpaceNodeOverlay"), (readonly, never None)

    path[¶](#bpy.types.SpaceNodeEditor.path "Link to this definition")
    :   Path from the data-block to the currently edited node tree

        Type:
        :   [`SpaceNodeEditorPath`](SpaceNodeEditorPath.md#bpy.types.SpaceNodeEditorPath "bpy.types.SpaceNodeEditorPath") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`NodeTreePath`](NodeTreePath.md#bpy.types.NodeTreePath "bpy.types.NodeTreePath"), (readonly)

    pin[¶](#bpy.types.SpaceNodeEditor.pin "Link to this definition")
    :   Use the pinned node tree

        Type:
        :   boolean, default False

    shader\_type[¶](#bpy.types.SpaceNodeEditor.shader_type "Link to this definition")
    :   Type of data to take shader from

        * `OBJECT`
          Object – Edit shader nodes from Object.
        * `WORLD`
          World – Edit shader nodes from World.
        * `LINESTYLE`
          Line Style – Edit shader nodes from Line Style.

        Type:
        :   enum in [`'OBJECT'`, `'WORLD'`, `'LINESTYLE'`], default `'OBJECT'`

    show\_annotation[¶](#bpy.types.SpaceNodeEditor.show_annotation "Link to this definition")

        Type:

        :   boolean, default False

    texture\_type[¶](#bpy.types.SpaceNodeEditor.texture_type "Link to this definition")

          World – Edit texture nodes from World.

          Brush – Edit texture nodes from Brush.
        * `LINESTYLE`
          Line Style – Edit texture nodes from Line Style.

        Type:
        :   enum in [`'WORLD'`, `'BRUSH'`, `'LINESTYLE'`], default `'WORLD'`

    tree\_type[¶](#bpy.types.SpaceNodeEditor.tree_type "Link to this definition")
    :   Node tree type to display and edit

        * `GeometryNodeTree`
          Geometry Node Editor – Geometry nodes.

          Compositor – Compositing nodes.
        * `ShaderNodeTree`
          Shader Editor – Shader nodes.
        * `TextureNodeTree`
          Texture Node Editor – Texture nodes.

        Type:
        :   enum in [`'GeometryNodeTree'`, `'CompositorNodeTree'`, `'ShaderNodeTree'`, `'TextureNodeTree'`], default `'DEFAULT'`

    *classmethod* draw\_handler\_add(*callback*, *args*, *region\_type*, *draw\_type*)[¶](#bpy.types.SpaceNodeEditor.draw_handler_add "Link to this definition")
    :   Add a new draw handler to this space type.
        It will be called every time the specified region in the space type will be drawn.
        Note: All arguments are positional only for now.

        Parameters:
        :   * **callback** (*Callable**[**...**,* *Any**]*) – A function that will be called when the region is drawn.
              It gets the specified arguments as input, it’s return value is ignored.
            * **args** (*tuple**[**Any**,* *...**]*) – Arguments that will be passed to the callback.
            * **region\_type** (*str*) – The region type the callback draws in; usually `WINDOW`. ([`bpy.types.Region.type`](Region.md#bpy.types.Region.type "bpy.types.Region.type"))
            * **draw\_type** (*str*) – Usually `POST_PIXEL` for 2D drawing and `POST_VIEW` for 3D drawing. In some cases `PRE_VIEW` can be used. `BACKDROP` can be used for backdrops in the node editor.

        Returns:
        :   Handler that can be removed later on.

        Return type:
        :   object

    *classmethod* draw\_handler\_remove(*handler*, *region\_type*)[¶](#bpy.types.SpaceNodeEditor.draw_handler_remove "Link to this definition")
    :   Remove a draw handler that was added previously.

        Parameters:
        :   * **handler** (*object*) – The draw handler that should be removed.
            * **region\_type** (*str*) – Region type the callback was added to.
