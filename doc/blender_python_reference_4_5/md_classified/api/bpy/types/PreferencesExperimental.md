# PreferencesExperimental(bpy\_struct)[¶](#preferencesexperimental-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.PreferencesExperimental(*bpy\_struct*)[¶](#bpy.types.PreferencesExperimental "Link to this definition")
:   Experimental features

    override\_auto\_resync[¶](#bpy.types.PreferencesExperimental.override_auto_resync "Link to this definition")
    :   Disable library overrides automatic resync detection and process on file load (can be useful to help fixing broken files). Also see the –disable-liboverride-auto-resync command line option

        Type:
        :   boolean, default False

    show\_asset\_debug\_info[¶](#bpy.types.PreferencesExperimental.show_asset_debug_info "Link to this definition")
    :   Enable some extra fields in the Asset Browser to aid in debugging

        Type:
        :   boolean, default False

    use\_all\_linked\_data\_direct[¶](#bpy.types.PreferencesExperimental.use_all_linked_data_direct "Link to this definition")
    :   Forces all linked data to be considered as directly linked. Workaround for current issues/limitations in BAT (Blender studio pipeline tool)

        Type:
        :   boolean, default False

    use\_asset\_indexing[¶](#bpy.types.PreferencesExperimental.use_asset_indexing "Link to this definition")
    :   Disable the asset indexer, to force every asset library refresh to completely reread assets from disk

        Type:
        :   boolean, default False

    use\_attribute\_storage\_write[¶](#bpy.types.PreferencesExperimental.use_attribute_storage_write "Link to this definition")
    :   Instead of writing with the older “CustomData” format for forward compatibility, use the new “AttributeStorage” format

        Type:
        :   boolean, default False

    use\_bundle\_and\_closure\_nodes[¶](#bpy.types.PreferencesExperimental.use_bundle_and_closure_nodes "Link to this definition")
    :   Enables bundle and closure nodes in Geometry Nodes

        Type:
        :   boolean, default False

    use\_cycles\_debug[¶](#bpy.types.PreferencesExperimental.use_cycles_debug "Link to this definition")
    :   Enable Cycles debugging options for developers

        Type:
        :   boolean, default False

    use\_eevee\_debug[¶](#bpy.types.PreferencesExperimental.use_eevee_debug "Link to this definition")
    :   Enable EEVEE debugging options for developers

        Type:
        :   boolean, default False

    use\_extended\_asset\_browser[¶](#bpy.types.PreferencesExperimental.use_extended_asset_browser "Link to this definition")
    :   Enable Asset Browser editor and operators to manage regular data-blocks as assets, not just poses

        Type:
        :   boolean, default False

    use\_extensions\_debug[¶](#bpy.types.PreferencesExperimental.use_extensions_debug "Link to this definition")
    :   Extra debugging information & developer support utilities for extensions

        Type:
        :   boolean, default False

    use\_new\_curves\_tools[¶](#bpy.types.PreferencesExperimental.use_new_curves_tools "Link to this definition")
    :   Enable additional features for the new curves data block

        Type:
        :   boolean, default False

    use\_new\_volume\_nodes[¶](#bpy.types.PreferencesExperimental.use_new_volume_nodes "Link to this definition")
    :   Enables visibility of the new Volume nodes in the UI

        Type:
        :   boolean, default False

    use\_recompute\_usercount\_on\_save\_debug[¶](#bpy.types.PreferencesExperimental.use_recompute_usercount_on_save_debug "Link to this definition")
    :   Recompute all ID usercounts before saving to a blendfile. Allows to work around invalid usercount handling in code that may lead to loss of data due to wrongly detected unused data-blocks

        Type:
        :   boolean, default False

    use\_sculpt\_texture\_paint[¶](#bpy.types.PreferencesExperimental.use_sculpt_texture_paint "Link to this definition")
    :   Use texture painting in Sculpt Mode

        Type:
        :   boolean, default False

    use\_shader\_node\_previews[¶](#bpy.types.PreferencesExperimental.use_shader_node_previews "Link to this definition")
    :   Enables previews in the shader node editor

        Type:
        :   boolean, default False

    use\_socket\_structure\_type[¶](#bpy.types.PreferencesExperimental.use_socket_structure_type "Link to this definition")
    :   Enables new visualization of socket data compatibility in Geometry Nodes

        Type:
        :   boolean, default False

    use\_undo\_legacy[¶](#bpy.types.PreferencesExperimental.use_undo_legacy "Link to this definition")
    :   Use legacy undo (slower than the new default one, but may be more stable in some cases)

        Type:
        :   boolean, default False

    use\_viewport\_debug[¶](#bpy.types.PreferencesExperimental.use_viewport_debug "Link to this definition")
    :   Enable viewport debugging options for developers in the overlays pop-over

        Type:
        :   boolean, default False

    write\_large\_blend\_file\_blocks[¶](#bpy.types.PreferencesExperimental.write_large_blend_file_blocks "Link to this definition")
    :   Enables support for writing .blend files that contain buffers larger than 2 GB. If enabled, any saved files can not be opened by older Blender versions

        Type:
        :   boolean, default False

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.PreferencesExperimental.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass

    *classmethod* bl\_rna\_get\_subclass\_py(*id*, *default=None*, */*)[¶](#bpy.types.PreferencesExperimental.bl_rna_get_subclass_py "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The class or default when not found.

        Return type:
        :   type
