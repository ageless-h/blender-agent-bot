:   bimanual[¶](#bpy.types.XrActionMapItem.bimanual "Link to this definition")
    :   The action depends on the states/poses of both user paths

        Type:
        :   boolean, default False

    :   Bindings for the action map item, mapping the action to an XR input

        Type:
        :   [`XrActionMapBindings`](XrActionMapBindings.md#bpy.types.XrActionMapBindings "bpy.types.XrActionMapBindings") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`XrActionMapBinding`](XrActionMapBinding.md#bpy.types.XrActionMapBinding "bpy.types.XrActionMapBinding"), (readonly)

    haptic\_amplitude[¶](#bpy.types.XrActionMapItem.haptic_amplitude "Link to this definition")
    :   Intensity of the haptic vibration, ranging from 0.0 to 1.0

        Type:
        :   float in [0, 1], default 0.0

    haptic\_duration[¶](#bpy.types.XrActionMapItem.haptic_duration "Link to this definition")
    :   Haptic duration in seconds. 0.0 is the minimum supported duration.

        Type:
        :   float in [0, inf], default 0.0

    haptic\_frequency[¶](#bpy.types.XrActionMapItem.haptic_frequency "Link to this definition")
    :   Frequency of the haptic vibration in hertz. 0.0 specifies the OpenXR runtime’s default frequency.

        Type:
        :   float in [0, inf], default 0.0

    haptic\_match\_user\_paths[¶](#bpy.types.XrActionMapItem.haptic_match_user_paths "Link to this definition")
    :   Apply haptics to the same user paths for the haptic action and this action

        Type:
        :   boolean, default False

    haptic\_mode[¶](#bpy.types.XrActionMapItem.haptic_mode "Link to this definition")
    :   Haptic application mode

        * `PRESS`
          Press – Apply haptics on button press.
        * `RELEASE`
          Release – Apply haptics on button release.
        * `PRESS_RELEASE`
          Press Release – Apply haptics on button press and release.
        * `REPEAT`
          Repeat – Apply haptics repeatedly for the duration of the button press.

        Type:
        :   enum in [`'PRESS'`, `'RELEASE'`, `'PRESS_RELEASE'`, `'REPEAT'`], default `'PRESS'`

    haptic\_name[¶](#bpy.types.XrActionMapItem.haptic_name "Link to this definition")
    :   Name of the haptic action to apply when executing this action

        Type:
        :   string, default “”, (never None)

    name[¶](#bpy.types.XrActionMapItem.name "Link to this definition")
    :   Name of the action map item

        Type:
        :   string, default “”, (never None)

    op[¶](#bpy.types.XrActionMapItem.op "Link to this definition")
    :   Identifier of operator to call on action event

        Type:
        :   string, default “”, (never None)

    op\_mode[¶](#bpy.types.XrActionMapItem.op_mode "Link to this definition")
    :   Operator execution mode

        * `PRESS`
          Press – Execute operator on button press (non-modal operators only).
        * `RELEASE`
          Release – Execute operator on button release (non-modal operators only).
        * `MODAL`
          Modal – Use modal execution (modal operators only).

        Type:
        :   enum in [`'PRESS'`, `'RELEASE'`, `'MODAL'`], default `'PRESS'`

    op\_name[¶](#bpy.types.XrActionMapItem.op_name "Link to this definition")
    :   Name of operator (translated) to call on action event

        Type:
        :   string, default “”, (readonly, never None)

    op\_properties[¶](#bpy.types.XrActionMapItem.op_properties "Link to this definition")
    :   Properties to set when the operator is called

        Type:
        :   [`OperatorProperties`](OperatorProperties.md#bpy.types.OperatorProperties "bpy.types.OperatorProperties"), (readonly)

    pose\_is\_controller\_aim[¶](#bpy.types.XrActionMapItem.pose_is_controller_aim "Link to this definition")
    :   The action poses will be used for the VR controller aims

        Type:
        :   boolean, default False

    pose\_is\_controller\_grip[¶](#bpy.types.XrActionMapItem.pose_is_controller_grip "Link to this definition")
    :   The action poses will be used for the VR controller grips

        Type:
        :   boolean, default False

    selected\_binding[¶](#bpy.types.XrActionMapItem.selected_binding "Link to this definition")
    :   Currently selected binding

        Type:
        :   int in [-32768, 32767], default 0

    type[¶](#bpy.types.XrActionMapItem.type "Link to this definition")
    :   Action type

        * `FLOAT`
          Float – Float action, representing either a digital or analog button.
        * `VECTOR2D`
          Vector2D – 2D float vector action, representing a thumbstick or trackpad.
        * `POSE`
          Pose – 3D pose action, representing a controller’s location and rotation.
        * `VIBRATION`
          Vibration – Haptic vibration output action, to be applied with a duration, frequency, and amplitude.

        Type:
        :   enum in [`'FLOAT'`, `'VECTOR2D'`, `'POSE'`, `'VIBRATION'`], default `'FLOAT'`

    user\_paths[¶](#bpy.types.XrActionMapItem.user_paths "Link to this definition")
    :   OpenXR user paths

        :   [`XrUserPaths`](XrUserPaths.md#bpy.types.XrUserPaths "bpy.types.XrUserPaths") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`XrUserPath`](XrUserPath.md#bpy.types.XrUserPath "bpy.types.XrUserPath"), (readonly)
