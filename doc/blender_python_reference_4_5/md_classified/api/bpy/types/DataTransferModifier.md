# DataTransferModifier(Modifier)[¶](#datatransfermodifier-modifier "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`Modifier`](Modifier.md#bpy.types.Modifier "bpy.types.Modifier")

*class* bpy.types.DataTransferModifier(*Modifier*)[¶](#bpy.types.DataTransferModifier "Link to this definition")
:   Modifier transferring some data from a source mesh

    data\_types\_edges[¶](#bpy.types.DataTransferModifier.data_types_edges "Link to this definition")
    :   Which edge data layers to transfer

        * `SHARP_EDGE`
          Sharp – Transfer sharp mark.
        * `SEAM`
          UV Seam – Transfer UV seam mark.
        * `CREASE`
          Crease – Transfer subdivision crease values.
        * `BEVEL_WEIGHT_EDGE`
          Bevel Weight – Transfer bevel weights.
        * `FREESTYLE_EDGE`
          Freestyle – Transfer Freestyle edge mark.

        Type:
        :   enum set in {`'SHARP_EDGE'`, `'SEAM'`, `'CREASE'`, `'BEVEL_WEIGHT_EDGE'`, `'FREESTYLE_EDGE'`}, default {`'SHARP_EDGE'`}

    data\_types\_loops[¶](#bpy.types.DataTransferModifier.data_types_loops "Link to this definition")
    :   Which face corner data layers to transfer

        * `CUSTOM_NORMAL`
          Custom Normals – Transfer custom normals.
        * `COLOR_CORNER`
          Colors – Transfer color attributes.
        * `UV`
          UVs – Transfer UV layers.

        Type:
        :   enum set in {`'CUSTOM_NORMAL'`, `'COLOR_CORNER'`, `'UV'`}, default {`'CUSTOM_NORMAL'`}

    data\_types\_polys[¶](#bpy.types.DataTransferModifier.data_types_polys "Link to this definition")
    :   Which face data layers to transfer

        * `SMOOTH`
          Smooth – Transfer flat/smooth mark.
        * `FREESTYLE_FACE`
          Freestyle Mark – Transfer Freestyle face mark.

        Type:
        :   enum set in {`'SMOOTH'`, `'FREESTYLE_FACE'`}, default {`'SMOOTH'`}

    data\_types\_verts[¶](#bpy.types.DataTransferModifier.data_types_verts "Link to this definition")
    :   Which vertex data layers to transfer

        * `VGROUP_WEIGHTS`
          Vertex Groups – Transfer active or all vertex groups.
        * `BEVEL_WEIGHT_VERT`
          Bevel Weight – Transfer bevel weights.
        * `COLOR_VERTEX`
          Colors – Transfer color attributes.

        Type:
        :   enum set in {`'VGROUP_WEIGHTS'`, `'BEVEL_WEIGHT_VERT'`, `'COLOR_VERTEX'`}, default {`'VGROUP_WEIGHTS'`}

    edge\_mapping[¶](#bpy.types.DataTransferModifier.edge_mapping "Link to this definition")
    :   Method used to map source edges to destination ones

        Type:
        :   enum in [Dt Method Edge Items](../../../appendix/bpy_types_enum_items/dt_method_edge_items.md#rna-enum-dt-method-edge-items), default `'NEAREST'`

    invert\_vertex\_group[¶](#bpy.types.DataTransferModifier.invert_vertex_group "Link to this definition")
    :   Invert vertex group influence

        Type:
        :   boolean, default False

    islands\_precision[¶](#bpy.types.DataTransferModifier.islands_precision "Link to this definition")
    :   Factor controlling precision of islands handling (typically, 0.1 should be enough, higher values can make things really slow)

        Type:
        :   float in [0, 1], default 0.0

    layers\_uv\_select\_dst[¶](#bpy.types.DataTransferModifier.layers_uv_select_dst "Link to this definition")
    :   How to match source and destination layers

        Type:
        :   enum in [Dt Layers Select Dst Items](../../../appendix/bpy_types_enum_items/dt_layers_select_dst_items.md#rna-enum-dt-layers-select-dst-items), default `'NAME'`

    layers\_uv\_select\_src[¶](#bpy.types.DataTransferModifier.layers_uv_select_src "Link to this definition")
    :   Which layers to transfer, in case of multi-layers types

        Type:
        :   enum in [Dt Layers Select Src Items](../../../appendix/bpy_types_enum_items/dt_layers_select_src_items.md#rna-enum-dt-layers-select-src-items), default `'ALL'`

    layers\_vcol\_loop\_select\_dst[¶](#bpy.types.DataTransferModifier.layers_vcol_loop_select_dst "Link to this definition")
    :   How to match source and destination layers

        Type:
        :   enum in [Dt Layers Select Dst Items](../../../appendix/bpy_types_enum_items/dt_layers_select_dst_items.md#rna-enum-dt-layers-select-dst-items), default `'NAME'`

    layers\_vcol\_loop\_select\_src[¶](#bpy.types.DataTransferModifier.layers_vcol_loop_select_src "Link to this definition")
    :   Which layers to transfer, in case of multi-layers types

        Type:
        :   enum in [Dt Layers Select Src Items](../../../appendix/bpy_types_enum_items/dt_layers_select_src_items.md#rna-enum-dt-layers-select-src-items), default `'ALL'`

    layers\_vcol\_vert\_select\_dst[¶](#bpy.types.DataTransferModifier.layers_vcol_vert_select_dst "Link to this definition")
    :   How to match source and destination layers

        Type:
        :   enum in [Dt Layers Select Dst Items](../../../appendix/bpy_types_enum_items/dt_layers_select_dst_items.md#rna-enum-dt-layers-select-dst-items), default `'NAME'`

    layers\_vcol\_vert\_select\_src[¶](#bpy.types.DataTransferModifier.layers_vcol_vert_select_src "Link to this definition")
    :   Which layers to transfer, in case of multi-layers types

        Type:
        :   enum in [Dt Layers Select Src Items](../../../appendix/bpy_types_enum_items/dt_layers_select_src_items.md#rna-enum-dt-layers-select-src-items), default `'ALL'`

    layers\_vgroup\_select\_dst[¶](#bpy.types.DataTransferModifier.layers_vgroup_select_dst "Link to this definition")
    :   How to match source and destination layers

        Type:
        :   enum in [Dt Layers Select Dst Items](../../../appendix/bpy_types_enum_items/dt_layers_select_dst_items.md#rna-enum-dt-layers-select-dst-items), default `'NAME'`

    layers\_vgroup\_select\_src[¶](#bpy.types.DataTransferModifier.layers_vgroup_select_src "Link to this definition")
    :   Which layers to transfer, in case of multi-layers types

        Type:
        :   enum in [Dt Layers Select Src Items](../../../appendix/bpy_types_enum_items/dt_layers_select_src_items.md#rna-enum-dt-layers-select-src-items), default `'ALL'`

    loop\_mapping[¶](#bpy.types.DataTransferModifier.loop_mapping "Link to this definition")
    :   Method used to map source faces’ corners to destination ones

        Type:
        :   enum in [Dt Method Loop Items](../../../appendix/bpy_types_enum_items/dt_method_loop_items.md#rna-enum-dt-method-loop-items), default `'NEAREST_POLYNOR'`

    max\_distance[¶](#bpy.types.DataTransferModifier.max_distance "Link to this definition")
    :   Maximum allowed distance between source and destination element, for non-topology mappings

        Type:
        :   float in [0, inf], default 1.0

    mix\_factor[¶](#bpy.types.DataTransferModifier.mix_factor "Link to this definition")
    :   Factor to use when applying data to destination (exact behavior depends on mix mode, multiplied with weights from vertex group when defined)

        Type:
        :   float in [0, 1], default 0.0

    mix\_mode[¶](#bpy.types.DataTransferModifier.mix_mode "Link to this definition")
    :   How to affect destination elements with source values

        Type:
        :   enum in [Dt Mix Mode Items](../../../appendix/bpy_types_enum_items/dt_mix_mode_items.md#rna-enum-dt-mix-mode-items), default `'REPLACE'`

    object[¶](#bpy.types.DataTransferModifier.object "Link to this definition")
    :   Object to transfer data from

        Type:
        :   [`Object`](Object.md#bpy.types.Object "bpy.types.Object")

    poly\_mapping[¶](#bpy.types.DataTransferModifier.poly_mapping "Link to this definition")
    :   Method used to map source faces to destination ones

        Type:
        :   enum in [Dt Method Poly Items](../../../appendix/bpy_types_enum_items/dt_method_poly_items.md#rna-enum-dt-method-poly-items), default `'NEAREST'`

    ray\_radius[¶](#bpy.types.DataTransferModifier.ray_radius "Link to this definition")
    :   ‘Width’ of rays (especially useful when raycasting against vertices or edges)

        Type:
        :   float in [0, inf], default 0.0

    use\_edge\_data[¶](#bpy.types.DataTransferModifier.use_edge_data "Link to this definition")
    :   Enable edge data transfer

        Type:
        :   boolean, default False

    use\_loop\_data[¶](#bpy.types.DataTransferModifier.use_loop_data "Link to this definition")
    :   Enable face corner data transfer

        Type:
        :   boolean, default False

    use\_max\_distance[¶](#bpy.types.DataTransferModifier.use_max_distance "Link to this definition")
    :   Source elements must be closer than given distance from destination one

        Type:
        :   boolean, default False

    use\_object\_transform[¶](#bpy.types.DataTransferModifier.use_object_transform "Link to this definition")
    :   Evaluate source and destination meshes in global space

        Type:
        :   boolean, default True

    use\_poly\_data[¶](#bpy.types.DataTransferModifier.use_poly_data "Link to this definition")
    :   Enable face data transfer

        Type:
        :   boolean, default False

    use\_vert\_data[¶](#bpy.types.DataTransferModifier.use_vert_data "Link to this definition")
    :   Enable vertex data transfer

        Type:
        :   boolean, default False

    vert\_mapping[¶](#bpy.types.DataTransferModifier.vert_mapping "Link to this definition")
    :   Method used to map source vertices to destination ones

        Type:
        :   enum in [Dt Method Vertex Items](../../../appendix/bpy_types_enum_items/dt_method_vertex_items.md#rna-enum-dt-method-vertex-items), default `'NEAREST'`

    vertex\_group[¶](#bpy.types.DataTransferModifier.vertex_group "Link to this definition")
    :   Vertex group name for selecting the affected areas

        Type:
        :   string, default “”, (never None)
