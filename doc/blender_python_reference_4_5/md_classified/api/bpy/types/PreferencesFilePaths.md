# PreferencesFilePaths(bpy\_struct)[¶](#preferencesfilepaths-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.PreferencesFilePaths(*bpy\_struct*)[¶](#bpy.types.PreferencesFilePaths "Link to this definition")
:   Default paths for external files

    active\_asset\_library[¶](#bpy.types.PreferencesFilePaths.active_asset_library "Link to this definition")
    :   Index of the asset library being edited in the Preferences UI

        Type:
        :   int in [-32768, 32767], default 0

    animation\_player[¶](#bpy.types.PreferencesFilePaths.animation_player "Link to this definition")
    :   Path to a custom animation/frame sequence player

        Type:
        :   string, default “”, (never None)

    animation\_player\_preset[¶](#bpy.types.PreferencesFilePaths.animation_player_preset "Link to this definition")
    :   Preset configs for external animation players

        * `INTERNAL`
          Internal – Built-in animation player.
        * `DJV`
          DJV – Open source frame player.
        * `FRAMECYCLER`
          FrameCycler – Frame player from IRIDAS.
        * `RV`
          RV – Frame player from Tweak Software.
        * `MPLAYER`
          MPlayer – Media player for video and PNG/JPEG/SGI image sequences.
        * `CUSTOM`
          Custom – Custom animation player executable path.

        Type:
        :   enum in [`'INTERNAL'`, `'DJV'`, `'FRAMECYCLER'`, `'RV'`, `'MPLAYER'`, `'CUSTOM'`], default `'INTERNAL'`

    asset\_libraries[¶](#bpy.types.PreferencesFilePaths.asset_libraries "Link to this definition")
    :   Type:
        :   [`AssetLibraryCollection`](AssetLibraryCollection.md#bpy.types.AssetLibraryCollection "bpy.types.AssetLibraryCollection") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`UserAssetLibrary`](UserAssetLibrary.md#bpy.types.UserAssetLibrary "bpy.types.UserAssetLibrary"), (readonly)

    auto\_save\_time[¶](#bpy.types.PreferencesFilePaths.auto_save_time "Link to this definition")
    :   The time (in minutes) to wait between automatic temporary saves

        Type:
        :   int in [1, 60], default 2

    file\_preview\_type[¶](#bpy.types.PreferencesFilePaths.file_preview_type "Link to this definition")
    :   What type of blend preview to create

        * `NONE`
          None – Do not create blend previews.
        * `AUTO`
          Auto – Automatically select best preview type.
        * `SCREENSHOT`
          Screenshot – Capture the entire window.
        * `CAMERA`
          Camera View – Workbench render of scene.

        Type:
        :   enum in [`'NONE'`, `'AUTO'`, `'SCREENSHOT'`, `'CAMERA'`], default `'AUTO'`

    font\_directory[¶](#bpy.types.PreferencesFilePaths.font_directory "Link to this definition")
    :   The default directory to search for loading fonts

        Type:
        :   string, default “”, (never None, blend relative `//` prefix supported)

    i18n\_branches\_directory[¶](#bpy.types.PreferencesFilePaths.i18n_branches_directory "Link to this definition")
    :   The path to the ‘/branches’ directory of your local svn-translation copy, to allow translating from the UI

        Type:
        :   string, default “”, (never None)

    image\_editor[¶](#bpy.types.PreferencesFilePaths.image_editor "Link to this definition")
    :   Path to an image editor

        Type:
        :   string, default “”, (never None)

    recent\_files[¶](#bpy.types.PreferencesFilePaths.recent_files "Link to this definition")
    :   Maximum number of recently opened files to remember

        Type:
        :   int in [0, 30], default 20

    render\_cache\_directory[¶](#bpy.types.PreferencesFilePaths.render_cache_directory "Link to this definition")
    :   Where to cache raw render results

        Type:
        :   string, default “”, (never None, blend relative `//` prefix supported)

    render\_output\_directory[¶](#bpy.types.PreferencesFilePaths.render_output_directory "Link to this definition")
    :   The default directory for rendering output, for new scenes

        Type:
        :   string, default “”, (never None, blend relative `//` prefix supported)

    save\_version[¶](#bpy.types.PreferencesFilePaths.save_version "Link to this definition")
    :   The number of old versions to maintain in the current directory, when manually saving

        Type:
        :   int in [0, 32], default 1

    script\_directories[¶](#bpy.types.PreferencesFilePaths.script_directories "Link to this definition")
    :   Type:
        :   [`ScriptDirectoryCollection`](ScriptDirectoryCollection.md#bpy.types.ScriptDirectoryCollection "bpy.types.ScriptDirectoryCollection") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`ScriptDirectory`](ScriptDirectory.md#bpy.types.ScriptDirectory "bpy.types.ScriptDirectory"), (readonly)

    show\_hidden\_files\_datablocks[¶](#bpy.types.PreferencesFilePaths.show_hidden_files_datablocks "Link to this definition")
    :   Show files and data-blocks that are normally hidden

        Type:
        :   boolean, default True

    show\_recent\_locations[¶](#bpy.types.PreferencesFilePaths.show_recent_locations "Link to this definition")
    :   Show Recent locations list in the File Browser

        Type:
        :   boolean, default True

    show\_system\_bookmarks[¶](#bpy.types.PreferencesFilePaths.show_system_bookmarks "Link to this definition")
    :   Show System locations list in the File Browser

        Type:
        :   boolean, default True

    sound\_directory[¶](#bpy.types.PreferencesFilePaths.sound_directory "Link to this definition")
    :   The default directory to search for sounds

        Type:
        :   string, default “”, (never None, blend relative `//` prefix supported)

    temporary\_directory[¶](#bpy.types.PreferencesFilePaths.temporary_directory "Link to this definition")
    :   The directory for storing temporary save files. The path must reference an existing directory or it will be ignored

        Type:
        :   string, default “”, (never None)

    text\_editor[¶](#bpy.types.PreferencesFilePaths.text_editor "Link to this definition")
    :   Command to launch the text editor, either a full path or a command in $PATH.
        Use the internal editor when left blank

        Type:
        :   string, default “”, (never None)

    text\_editor\_args[¶](#bpy.types.PreferencesFilePaths.text_editor_args "Link to this definition")
    :   Defines the specific format of the arguments with which the text editor opens files. The supported expansions are as follows:

        $filepath The absolute path of the file.
        $line The line to open at (Optional).
        $column The column to open from the beginning of the line (Optional).
        $line0 & column0 start at zero.
        Example: -f $filepath -l $line -c $column

        Type:
        :   string, default “”, (never None)

    texture\_directory[¶](#bpy.types.PreferencesFilePaths.texture_directory "Link to this definition")
    :   The default directory to search for textures

        Type:
        :   string, default “”, (never None, blend relative `//` prefix supported)

    use\_auto\_save\_temporary\_files[¶](#bpy.types.PreferencesFilePaths.use_auto_save_temporary_files "Link to this definition")
    :   Automatic saving of temporary files in temp directory, uses process ID.
        Warning: Sculpt and edit mode data won’t be saved

        Type:
        :   boolean, default True

    use\_extension\_online\_access\_handled[¶](#bpy.types.PreferencesFilePaths.use_extension_online_access_handled "Link to this definition")
    :   The user has been shown the “Online Access” prompt and made a choice

        Type:
        :   boolean, default False

    use\_file\_compression[¶](#bpy.types.PreferencesFilePaths.use_file_compression "Link to this definition")
    :   Enable file compression when saving .blend files

        Type:
        :   boolean, default False

    use\_filter\_files[¶](#bpy.types.PreferencesFilePaths.use_filter_files "Link to this definition")
    :   Enable filtering of files in the File Browser

        Type:
        :   boolean, default True

    use\_load\_ui[¶](#bpy.types.PreferencesFilePaths.use_load_ui "Link to this definition")
    :   Load user interface setup when loading .blend files

        Type:
        :   boolean, default True

    use\_relative\_paths[¶](#bpy.types.PreferencesFilePaths.use_relative_paths "Link to this definition")
    :   Default relative path option for the file selector, when no path is defined yet

        Type:
        :   boolean, default True

    use\_scripts\_auto\_execute[¶](#bpy.types.PreferencesFilePaths.use_scripts_auto_execute "Link to this definition")
    :   Allow any .blend file to run scripts automatically (unsafe with blend files from an untrusted source)

        Type:
        :   boolean, default False

    use\_tabs\_as\_spaces[¶](#bpy.types.PreferencesFilePaths.use_tabs_as_spaces "Link to this definition")
    :   Automatically convert all new tabs into spaces for new and loaded text files

        Type:
        :   boolean, default True
