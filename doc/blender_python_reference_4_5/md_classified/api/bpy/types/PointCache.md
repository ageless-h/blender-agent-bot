# PointCache(bpy\_struct)[¶](#pointcache-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.PointCache(*bpy\_struct*)[¶](#bpy.types.PointCache "Link to this definition")
:   Active point cache for physics simulations

    compression[¶](#bpy.types.PointCache.compression "Link to this definition")
    :   Compression method to be used

        * `NO`
          None – No compression.
        * `LIGHT`
          Lite – Fast but not so effective compression.
        * `HEAVY`
          Heavy – Effective but slow compression.

        Type:
        :   enum in [`'NO'`, `'LIGHT'`, `'HEAVY'`], default `'NO'`

    filepath[¶](#bpy.types.PointCache.filepath "Link to this definition")
    :   Cache file path

        Type:
        :   string, default “”, (never None, blend relative `//` prefix supported)

    frame\_end[¶](#bpy.types.PointCache.frame_end "Link to this definition")
    :   Frame on which the simulation stops

        Type:
        :   int in [1, 1048574], default 0

    frame\_start[¶](#bpy.types.PointCache.frame_start "Link to this definition")
    :   Frame on which the simulation starts

        Type:
        :   int in [-1048574, 1048574], default 0

    frame\_step[¶](#bpy.types.PointCache.frame_step "Link to this definition")
    :   Number of frames between cached frames

        Type:
        :   int in [1, 20], default 0

    index[¶](#bpy.types.PointCache.index "Link to this definition")
    :   Index number of cache files

        Type:
        :   int in [-1, 100], default 0

    info[¶](#bpy.types.PointCache.info "Link to this definition")
    :   Info on current cache status

        Type:
        :   string, default “”, (readonly, never None)

    is\_baked[¶](#bpy.types.PointCache.is_baked "Link to this definition")
    :   The cache is baked

        Type:
        :   boolean, default False, (readonly)

    is\_baking[¶](#bpy.types.PointCache.is_baking "Link to this definition")
    :   The cache is being baked

        Type:
        :   boolean, default False, (readonly)

    is\_frame\_skip[¶](#bpy.types.PointCache.is_frame_skip "Link to this definition")
    :   Some frames were skipped while baking/saving that cache

        Type:
        :   boolean, default False, (readonly)

    is\_outdated[¶](#bpy.types.PointCache.is_outdated "Link to this definition")
    :   Type:
        :   boolean, default False, (readonly)

    name[¶](#bpy.types.PointCache.name "Link to this definition")
    :   Cache name

        Type:
        :   string, default “”, (never None)

    point\_caches[¶](#bpy.types.PointCache.point_caches "Link to this definition")
    :   Type:
        :   [`PointCaches`](PointCaches.md#bpy.types.PointCaches "bpy.types.PointCaches") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`PointCacheItem`](PointCacheItem.md#bpy.types.PointCacheItem "bpy.types.PointCacheItem"), (readonly)

    use\_disk\_cache[¶](#bpy.types.PointCache.use_disk_cache "Link to this definition")
    :   Save cache files to disk (.blend file must be saved first)

        Type:
        :   boolean, default False

    use\_external[¶](#bpy.types.PointCache.use_external "Link to this definition")
    :   Read cache from an external location

        Type:
        :   boolean, default False

    use\_library\_path[¶](#bpy.types.PointCache.use_library_path "Link to this definition")
    :   Use this file’s path for the disk cache when library linked into another file (for local bakes per scene file, disable this option)

        Type:
        :   boolean, default False

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.PointCache.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass
