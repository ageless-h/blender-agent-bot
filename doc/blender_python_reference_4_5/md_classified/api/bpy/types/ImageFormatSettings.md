# ImageFormatSettings(bpy\_struct)[¶](#imageformatsettings-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.ImageFormatSettings(*bpy\_struct*)[¶](#bpy.types.ImageFormatSettings "Link to this definition")
:   Settings for image formats

    cineon\_black[¶](#bpy.types.ImageFormatSettings.cineon_black "Link to this definition")
    :   Log conversion reference blackpoint

        Type:
        :   int in [0, 1024], default 0

    cineon\_gamma[¶](#bpy.types.ImageFormatSettings.cineon_gamma "Link to this definition")
    :   Log conversion gamma

        Type:
        :   float in [0, 10], default 0.0

    cineon\_white[¶](#bpy.types.ImageFormatSettings.cineon_white "Link to this definition")
    :   Log conversion reference whitepoint

        Type:
        :   int in [0, 1024], default 0

    color\_depth[¶](#bpy.types.ImageFormatSettings.color_depth "Link to this definition")
    :   Bit depth per channel

        Type:
        :   enum in [Image Color Depth Items](../../../appendix/bpy_types_enum_items/image_color_depth_items.md#rna-enum-image-color-depth-items), default `'8'`

    color\_management[¶](#bpy.types.ImageFormatSettings.color_management "Link to this definition")
    :   Which color management settings to use for file saving

        Type:
        :   enum in [`'FOLLOW_SCENE'`, `'OVERRIDE'`], default `'FOLLOW_SCENE'`

    color\_mode[¶](#bpy.types.ImageFormatSettings.color_mode "Link to this definition")
    :   Choose BW for saving grayscale images, RGB for saving red, green and blue channels, and RGBA for saving red, green, blue and alpha channels

        Type:
        :   enum in [Image Color Mode Items](../../../appendix/bpy_types_enum_items/image_color_mode_items.md#rna-enum-image-color-mode-items), default `'BW'`

    compression[¶](#bpy.types.ImageFormatSettings.compression "Link to this definition")
    :   Amount of time to determine best compression: 0 = no compression with fast file output, 100 = maximum lossless compression with slow file output

        Type:
        :   int in [0, 100], default 15

    display\_settings[¶](#bpy.types.ImageFormatSettings.display_settings "Link to this definition")
    :   Settings of device saved image would be displayed on

        Type:
        :   [`ColorManagedDisplaySettings`](ColorManagedDisplaySettings.md#bpy.types.ColorManagedDisplaySettings "bpy.types.ColorManagedDisplaySettings"), (readonly)

    exr\_codec[¶](#bpy.types.ImageFormatSettings.exr_codec "Link to this definition")
    :   Compression codec settings for OpenEXR

        Type:
        :   enum in [Exr Codec Items](../../../appendix/bpy_types_enum_items/exr_codec_items.md#rna-enum-exr-codec-items), default `'NONE'`

    file\_format[¶](#bpy.types.ImageFormatSettings.file_format "Link to this definition")
    :   File format to save the rendered images as

        Type:
        :   enum in [Image Type Items](../../../appendix/bpy_types_enum_items/image_type_items.md#rna-enum-image-type-items), default `'PNG'`

    has\_linear\_colorspace[¶](#bpy.types.ImageFormatSettings.has_linear_colorspace "Link to this definition")
    :   File format expects linear color space

        Type:
        :   boolean, default False, (readonly)

    jpeg2k\_codec[¶](#bpy.types.ImageFormatSettings.jpeg2k_codec "Link to this definition")
    :   Codec settings for JPEG 2000

        Type:
        :   enum in [`'JP2'`, `'J2K'`], default `'JP2'`

    linear\_colorspace\_settings[¶](#bpy.types.ImageFormatSettings.linear_colorspace_settings "Link to this definition")
    :   Output color space settings

        Type:
        :   [`ColorManagedInputColorspaceSettings`](ColorManagedInputColorspaceSettings.md#bpy.types.ColorManagedInputColorspaceSettings "bpy.types.ColorManagedInputColorspaceSettings"), (readonly)

    quality[¶](#bpy.types.ImageFormatSettings.quality "Link to this definition")
    :   Quality for image formats that support lossy compression

        Type:
        :   int in [0, 100], default 90

    stereo\_3d\_format[¶](#bpy.types.ImageFormatSettings.stereo_3d_format "Link to this definition")
    :   Settings for stereo 3D

        Type:
        :   [`Stereo3dFormat`](Stereo3dFormat.md#bpy.types.Stereo3dFormat "bpy.types.Stereo3dFormat"), (readonly, never None)

    tiff\_codec[¶](#bpy.types.ImageFormatSettings.tiff_codec "Link to this definition")
    :   Compression mode for TIFF

        Type:
        :   enum in [`'NONE'`, `'DEFLATE'`, `'LZW'`, `'PACKBITS'`], default `'DEFLATE'`

    use\_cineon\_log[¶](#bpy.types.ImageFormatSettings.use_cineon_log "Link to this definition")
    :   Convert to logarithmic color space

        Type:
        :   boolean, default False

    use\_jpeg2k\_cinema\_48[¶](#bpy.types.ImageFormatSettings.use_jpeg2k_cinema_48 "Link to this definition")
    :   Use OpenJPEG Cinema Preset (48fps)

        Type:
        :   boolean, default False

    use\_jpeg2k\_cinema\_preset[¶](#bpy.types.ImageFormatSettings.use_jpeg2k_cinema_preset "Link to this definition")
    :   Use OpenJPEG Cinema Preset

        Type:
        :   boolean, default False

    use\_jpeg2k\_ycc[¶](#bpy.types.ImageFormatSettings.use_jpeg2k_ycc "Link to this definition")
    :   Save luminance-chrominance-chrominance channels instead of RGB colors

        Type:
        :   boolean, default False

    use\_preview[¶](#bpy.types.ImageFormatSettings.use_preview "Link to this definition")
    :   When rendering animations, save JPG preview images in same directory

        Type:
        :   boolean, default False

    view\_settings[¶](#bpy.types.ImageFormatSettings.view_settings "Link to this definition")
    :   Color management settings applied on image before saving

        Type:
        :   [`ColorManagedViewSettings`](ColorManagedViewSettings.md#bpy.types.ColorManagedViewSettings "bpy.types.ColorManagedViewSettings"), (readonly)

    views\_format[¶](#bpy.types.ImageFormatSettings.views_format "Link to this definition")
    :   Format of multiview media

        Type:
        :   enum in [Views Format Multiview Items](../../../appendix/bpy_types_enum_items/views_format_multiview_items.md#rna-enum-views-format-multiview-items), default `'INDIVIDUAL'`

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.ImageFormatSettings.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass

    *classmethod* bl\_rna\_get\_subclass\_py(*id*, *default=None*, */*)[¶](#bpy.types.ImageFormatSettings.bl_rna_get_subclass_py "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The class or default when not found.

        Return type:
        :   type
