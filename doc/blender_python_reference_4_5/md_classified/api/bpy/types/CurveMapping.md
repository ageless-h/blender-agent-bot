*class* bpy.types.CurveMapping(*bpy\_struct*)[¶](#bpy.types.CurveMapping "Link to this definition")
:   Curve mapping to map color, vector and scalar values to other values using a user defined curve

    black\_level[¶](#bpy.types.CurveMapping.black_level "Link to this definition")
    :   For RGB curves, the color that black is mapped to

        Type:
        :   [`mathutils.Color`](../../mathutils/index.md#mathutils.Color "mathutils.Color") of 3 items in [-inf, inf], default (0.0, 0.0, 0.0)

    :   Type:

    :   Type:

    extend[¶](#bpy.types.CurveMapping.extend "Link to this definition")
    :   Extrapolate the curve or extend it horizontally

        Type:
        :   enum in [`'HORIZONTAL'`, `'EXTRAPOLATED'`], default `'HORIZONTAL'`

    tone[¶](#bpy.types.CurveMapping.tone "Link to this definition")
    :   Tone of the curve

        * `STANDARD`
          Standard – Combined curve is applied to each channel individually, which may result in a change of hue.
        * `FILMLIKE`
          Filmlike – Keeps the hue constant.

        Type:
        :   enum in [`'STANDARD'`, `'FILMLIKE'`], default `'STANDARD'`

    use\_clip[¶](#bpy.types.CurveMapping.use_clip "Link to this definition")
    :   Force the curve view to fit a defined boundary

        Type:
        :   boolean, default False

    white\_level[¶](#bpy.types.CurveMapping.white_level "Link to this definition")
    :   For RGB curves, the color that white is mapped to

        Type:
        :   [`mathutils.Color`](../../mathutils/index.md#mathutils.Color "mathutils.Color") of 3 items in [-inf, inf], default (0.0, 0.0, 0.0)

    update()[¶](#bpy.types.CurveMapping.update "Link to this definition")
    :   Update curve mapping after making changes

    reset\_view()[¶](#bpy.types.CurveMapping.reset_view "Link to this definition")
    :   Reset the curve mapping grid to its clipping size

    initialize()[¶](#bpy.types.CurveMapping.initialize "Link to this definition")

        Returns:
        :   Value, Value of curve at given location

        Return type:
        :   float in [-inf, inf]
