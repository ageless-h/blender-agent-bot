# FluidFlowSettings(bpy\_struct)[¶](#fluidflowsettings-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.FluidFlowSettings(*bpy\_struct*)[¶](#bpy.types.FluidFlowSettings "Link to this definition")
:   Fluid flow settings

    density[¶](#bpy.types.FluidFlowSettings.density "Link to this definition")
    :   Type:
        :   float in [0, 10], default 1.0

    density\_vertex\_group[¶](#bpy.types.FluidFlowSettings.density_vertex_group "Link to this definition")
    :   Name of vertex group which determines surface emission rate

        Type:
        :   string, default “”, (never None)

    flow\_behavior[¶](#bpy.types.FluidFlowSettings.flow_behavior "Link to this definition")
    :   Change flow behavior in the simulation

        * `INFLOW`
          Inflow – Add fluid to simulation.
        * `OUTFLOW`
          Outflow – Delete fluid from simulation.
        * `GEOMETRY`
          Geometry – Only use given geometry for fluid.

        Type:
        :   enum in [`'INFLOW'`, `'OUTFLOW'`, `'GEOMETRY'`], default `'GEOMETRY'`

    flow\_source[¶](#bpy.types.FluidFlowSettings.flow_source "Link to this definition")
    :   Change how fluid is emitted

        Type:
        :   enum in [`'NONE'`], default `'NONE'`

    flow\_type[¶](#bpy.types.FluidFlowSettings.flow_type "Link to this definition")
    :   Change type of fluid in the simulation

        * `SMOKE`
          Smoke – Add smoke.
        * `BOTH`
          Fire + Smoke – Add fire and smoke.
        * `FIRE`
          Fire – Add fire.
        * `LIQUID`
          Liquid – Add liquid.

        Type:
        :   enum in [`'SMOKE'`, `'BOTH'`, `'FIRE'`, `'LIQUID'`], default `'SMOKE'`

    fuel\_amount[¶](#bpy.types.FluidFlowSettings.fuel_amount "Link to this definition")
    :   Type:
        :   float in [0, 10], default 1.0

    noise\_texture[¶](#bpy.types.FluidFlowSettings.noise_texture "Link to this definition")
    :   Texture that controls emission strength

        Type:
        :   [`Texture`](Texture.md#bpy.types.Texture "bpy.types.Texture")

    particle\_size[¶](#bpy.types.FluidFlowSettings.particle_size "Link to this definition")
    :   Particle size in simulation cells

        Type:
        :   float in [0.1, inf], default 1.0

    particle\_system[¶](#bpy.types.FluidFlowSettings.particle_system "Link to this definition")
    :   Particle systems emitted from the object

        Type:
        :   [`ParticleSystem`](ParticleSystem.md#bpy.types.ParticleSystem "bpy.types.ParticleSystem")

    smoke\_color[¶](#bpy.types.FluidFlowSettings.smoke_color "Link to this definition")
    :   Color of smoke

        Type:
        :   [`mathutils.Color`](../../mathutils/index.md#mathutils.Color "mathutils.Color") of 3 items in [0, inf], default (0.7, 0.7, 0.7)

    subframes[¶](#bpy.types.FluidFlowSettings.subframes "Link to this definition")
    :   Number of additional samples to take between frames to improve quality of fast moving flows

        Type:
        :   int in [0, 200], default 0

    surface\_distance[¶](#bpy.types.FluidFlowSettings.surface_distance "Link to this definition")
    :   Height (in domain grid units) of fluid emission above the mesh surface. Higher values result in emission further away from the mesh surface. If this value and the emitter size are smaller than the domain grid unit, fluid will not be created

        Type:
        :   float in [0, 10], default 1.0

    temperature[¶](#bpy.types.FluidFlowSettings.temperature "Link to this definition")
    :   Temperature difference to ambient temperature

        Type:
        :   float in [-10, 10], default 1.0

    texture\_map\_type[¶](#bpy.types.FluidFlowSettings.texture_map_type "Link to this definition")
    :   Texture mapping type

        * `AUTO`
          Generated – Generated coordinates centered to flow object.
        * `UV`
          UV – Use UV layer for texture coordinates.

        Type:
        :   enum in [`'AUTO'`, `'UV'`], default `'AUTO'`

    texture\_offset[¶](#bpy.types.FluidFlowSettings.texture_offset "Link to this definition")
    :   Z-offset of texture mapping

        Type:
        :   float in [0, 200], default 0.0

    texture\_size[¶](#bpy.types.FluidFlowSettings.texture_size "Link to this definition")
    :   Size of texture mapping

        Type:
        :   float in [0.01, 10], default 1.0

    use\_absolute[¶](#bpy.types.FluidFlowSettings.use_absolute "Link to this definition")
    :   Only allow given density value in emitter area and will not add up

        Type:
        :   boolean, default True

    use\_inflow[¶](#bpy.types.FluidFlowSettings.use_inflow "Link to this definition")
    :   Control when to apply fluid flow

        Type:
        :   boolean, default True

    use\_initial\_velocity[¶](#bpy.types.FluidFlowSettings.use_initial_velocity "Link to this definition")
    :   Fluid has some initial velocity when it is emitted

        Type:
        :   boolean, default False

    use\_particle\_size[¶](#bpy.types.FluidFlowSettings.use_particle_size "Link to this definition")
    :   Set particle size in simulation cells or use nearest cell

        Type:
        :   boolean, default True

    use\_plane\_init[¶](#bpy.types.FluidFlowSettings.use_plane_init "Link to this definition")
    :   Treat this object as a planar and unclosed mesh. Fluid will only be emitted from the mesh surface and based on the surface emission value.

        Type:
        :   boolean, default False

    use\_texture[¶](#bpy.types.FluidFlowSettings.use_texture "Link to this definition")
    :   Use a texture to control emission strength

        Type:
        :   boolean, default False

    uv\_layer[¶](#bpy.types.FluidFlowSettings.uv_layer "Link to this definition")
    :   UV map name

        Type:
        :   string, default “”, (never None)

    velocity\_coord[¶](#bpy.types.FluidFlowSettings.velocity_coord "Link to this definition")
    :   Additional initial velocity in X, Y and Z direction (added to source velocity)

        Type:
        :   [`mathutils.Vector`](../../mathutils/index.md#mathutils.Vector "mathutils.Vector") of 3 items in [-1000.1, 1000.1], default (0.0, 0.0, 0.0)

    velocity\_factor[¶](#bpy.types.FluidFlowSettings.velocity_factor "Link to this definition")
    :   Multiplier of source velocity passed to fluid (source velocity is non-zero only if object is moving)

        Type:
        :   float in [-100, 100], default 1.0

    velocity\_normal[¶](#bpy.types.FluidFlowSettings.velocity_normal "Link to this definition")
    :   Amount of normal directional velocity

        Type:
        :   float in [-100, 100], default 0.0

    velocity\_random[¶](#bpy.types.FluidFlowSettings.velocity_random "Link to this definition")
    :   Amount of random velocity

        Type:
        :   float in [0, 10], default 0.0

    volume\_density[¶](#bpy.types.FluidFlowSettings.volume_density "Link to this definition")
    :   Controls fluid emission from within the mesh (higher value results in greater emissions from inside the mesh)

        Type:
        :   float in [0, 1], default 0.0

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.FluidFlowSettings.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass

    *classmethod* bl\_rna\_get\_subclass\_py(*id*, *default=None*, */*)[¶](#bpy.types.FluidFlowSettings.bl_rna_get_subclass_py "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        :   The class or default when not found.

        :   type
