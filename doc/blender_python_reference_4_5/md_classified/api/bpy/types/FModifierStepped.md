# FModifierStepped(FModifier)[¶](#fmodifierstepped-fmodifier "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`FModifier`](FModifier.md#bpy.types.FModifier "bpy.types.FModifier")

*class* bpy.types.FModifierStepped(*FModifier*)[¶](#bpy.types.FModifierStepped "Link to this definition")
:   Hold each interpolated value from the F-Curve for several frames without changing the timing

    frame\_end[¶](#bpy.types.FModifierStepped.frame_end "Link to this definition")
    :   Frame that modifier’s influence ends (if applicable)

        Type:
        :   float in [-inf, inf], default 0.0

    frame\_offset[¶](#bpy.types.FModifierStepped.frame_offset "Link to this definition")
    :   Reference number of frames before frames get held (use to get hold for ‘1-3’ vs ‘5-7’ holding patterns)

        Type:
        :   float in [-inf, inf], default 0.0

    frame\_start[¶](#bpy.types.FModifierStepped.frame_start "Link to this definition")
    :   Frame that modifier’s influence starts (if applicable)

        Type:
        :   float in [-inf, inf], default 0.0

    frame\_step[¶](#bpy.types.FModifierStepped.frame_step "Link to this definition")
    :   Number of frames to hold each value

        Type:
        :   float in [-inf, inf], default 2.0

    use\_frame\_end[¶](#bpy.types.FModifierStepped.use_frame_end "Link to this definition")
    :   Restrict modifier to only act before its ‘end’ frame

        Type:
        :   boolean, default False

    use\_frame\_start[¶](#bpy.types.FModifierStepped.use_frame_start "Link to this definition")
    :   Restrict modifier to only act after its ‘start’ frame

        Type:
        :   boolean, default False

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.FModifierStepped.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass

    *classmethod* bl\_rna\_get\_subclass\_py(*id*, *default=None*, */*)[¶](#bpy.types.FModifierStepped.bl_rna_get_subclass_py "Link to this definition")
    :   Parameters:
