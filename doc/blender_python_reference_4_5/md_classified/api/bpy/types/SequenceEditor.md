        Type:
        :   int in [-inf, inf], default 0, (readonly)

    cache\_raw\_size[¶](#bpy.types.SequenceEditor.cache_raw_size "Link to this definition")
    :   Size of raw source images cache in megabytes

        Type:
        :   int in [-inf, inf], default 0, (readonly)

    channels[¶](#bpy.types.SequenceEditor.channels "Link to this definition")
    :   Type:
        :   [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`SequenceTimelineChannel`](SequenceTimelineChannel.md#bpy.types.SequenceTimelineChannel "bpy.types.SequenceTimelineChannel"), (readonly)

        Type:
        :   int in [-inf, inf], default 0

    proxy\_dir[¶](#bpy.types.SequenceEditor.proxy_dir "Link to this definition")
    :   Type:
        :   string, default “”, (never None, blend relative `//` prefix supported)

    proxy\_storage[¶](#bpy.types.SequenceEditor.proxy_storage "Link to this definition")
    :   How to store proxies for this project

        * `PER_STRIP`
          Per Strip – Store proxies using per strip settings.
        * `PROJECT`
          Project – Store proxies using project directory.

        Type:
        :   enum in [`'PER_STRIP'`, `'PROJECT'`], default `'PER_STRIP'`

    selected\_retiming\_keys[¶](#bpy.types.SequenceEditor.selected_retiming_keys "Link to this definition")
    :   Type:
        :   boolean, default False, (readonly)

    sequences[¶](#bpy.types.SequenceEditor.sequences "Link to this definition")
    :   (Deprecated: Replaced by ‘.strips’) Top-level strips only

        Type:
        :   [`StripsTopLevel`](StripsTopLevel.md#bpy.types.StripsTopLevel "bpy.types.StripsTopLevel") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`Strip`](Strip.md#bpy.types.Strip "bpy.types.Strip"), (readonly)

    sequences\_all[¶](#bpy.types.SequenceEditor.sequences_all "Link to this definition")
    :   (Deprecated: Replaced by ‘.strips\_all’) All strips, recursively including those inside metastrips

        :   [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`Strip`](Strip.md#bpy.types.Strip "bpy.types.Strip"), (readonly)

    :   Render missing images/movies with a solid magenta color

        Type:
        :   boolean, default False

    show\_overlay\_frame[¶](#bpy.types.SequenceEditor.show_overlay_frame "Link to this definition")
    :   Partial overlay on top of the sequencer with a frame offset

        Type:
        :   boolean, default False

    strips[¶](#bpy.types.SequenceEditor.strips "Link to this definition")
    :   Top-level strips only

        Type:
        :   [`StripsTopLevel`](StripsTopLevel.md#bpy.types.StripsTopLevel "bpy.types.StripsTopLevel") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`Strip`](Strip.md#bpy.types.Strip "bpy.types.Strip"), (readonly)

    strips\_all[¶](#bpy.types.SequenceEditor.strips_all "Link to this definition")
    :   All strips, recursively including those inside metastrips

        Type:
        :   [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`Strip`](Strip.md#bpy.types.Strip "bpy.types.Strip"), (readonly)

    use\_cache\_final[¶](#bpy.types.SequenceEditor.use_cache_final "Link to this definition")
    :   Cache final image for each frame

        Type:
        :   boolean, default False

    use\_cache\_raw[¶](#bpy.types.SequenceEditor.use_cache_raw "Link to this definition")
    :   Cache raw images read from disk, for faster tweaking of strip parameters at the cost of memory usage

        Type:
        :   boolean, default False

    use\_overlay\_frame\_lock[¶](#bpy.types.SequenceEditor.use_overlay_frame_lock "Link to this definition")
    :   Type:
        :   boolean, default False

    use\_prefetch[¶](#bpy.types.SequenceEditor.use_prefetch "Link to this definition")
    :   Render frames ahead of current frame in the background for faster playback

        Type:
        :   boolean, default False
