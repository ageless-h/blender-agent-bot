# KinematicConstraint(Constraint)[¶](#kinematicconstraint-constraint "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`Constraint`](Constraint.md#bpy.types.Constraint "bpy.types.Constraint")

*class* bpy.types.KinematicConstraint(*Constraint*)[¶](#bpy.types.KinematicConstraint "Link to this definition")
:   Inverse Kinematics

    chain\_count[¶](#bpy.types.KinematicConstraint.chain_count "Link to this definition")
    :   How many bones are included in the IK effect - 0 uses all bones

        Type:
        :   int in [0, 255], default 0

    distance[¶](#bpy.types.KinematicConstraint.distance "Link to this definition")
    :   Radius of limiting sphere

        Type:
        :   float in [0, 100], default 0.0

    ik\_type[¶](#bpy.types.KinematicConstraint.ik_type "Link to this definition")
    :   Type:
        :   enum in [`'COPY_POSE'`, `'DISTANCE'`], default `'COPY_POSE'`

    iterations[¶](#bpy.types.KinematicConstraint.iterations "Link to this definition")
    :   Maximum number of solving iterations

        Type:
        :   int in [0, 10000], default 0

    limit\_mode[¶](#bpy.types.KinematicConstraint.limit_mode "Link to this definition")
    :   Distances in relation to sphere of influence to allow

        * `LIMITDIST_INSIDE`
          Inside – The object is constrained inside a virtual sphere around the target object, with a radius defined by the limit distance.
        * `LIMITDIST_OUTSIDE`
          Outside – The object is constrained outside a virtual sphere around the target object, with a radius defined by the limit distance.
        * `LIMITDIST_ONSURFACE`
          On Surface – The object is constrained on the surface of a virtual sphere around the target object, with a radius defined by the limit distance.

        Type:
        :   enum in [`'LIMITDIST_INSIDE'`, `'LIMITDIST_OUTSIDE'`, `'LIMITDIST_ONSURFACE'`], default `'LIMITDIST_INSIDE'`

    lock\_location\_x[¶](#bpy.types.KinematicConstraint.lock_location_x "Link to this definition")
    :   Constraint position along X axis

        Type:
        :   boolean, default False

    lock\_location\_y[¶](#bpy.types.KinematicConstraint.lock_location_y "Link to this definition")
    :   Constraint position along Y axis

        Type:
        :   boolean, default False

    lock\_location\_z[¶](#bpy.types.KinematicConstraint.lock_location_z "Link to this definition")
    :   Constraint position along Z axis

        Type:
        :   boolean, default False

    lock\_rotation\_x[¶](#bpy.types.KinematicConstraint.lock_rotation_x "Link to this definition")
    :   Constraint rotation along X axis

        Type:
        :   boolean, default False

    lock\_rotation\_y[¶](#bpy.types.KinematicConstraint.lock_rotation_y "Link to this definition")
    :   Constraint rotation along Y axis

        Type:
        :   boolean, default False

    lock\_rotation\_z[¶](#bpy.types.KinematicConstraint.lock_rotation_z "Link to this definition")
    :   Constraint rotation along Z axis

        Type:
        :   boolean, default False

    orient\_weight[¶](#bpy.types.KinematicConstraint.orient_weight "Link to this definition")
    :   For Tree-IK: Weight of orientation control for this target

        Type:
        :   float in [0.01, 1], default 0.0

    pole\_angle[¶](#bpy.types.KinematicConstraint.pole_angle "Link to this definition")
    :   Pole rotation offset

        Type:
        :   float in [-3.14159, 3.14159], default 0.0

    pole\_subtarget[¶](#bpy.types.KinematicConstraint.pole_subtarget "Link to this definition")
    :   Type:
        :   string, default “”, (never None)

    pole\_target[¶](#bpy.types.KinematicConstraint.pole_target "Link to this definition")
    :   Object for pole rotation

        Type:
        :   [`Object`](Object.md#bpy.types.Object "bpy.types.Object")

    reference\_axis[¶](#bpy.types.KinematicConstraint.reference_axis "Link to this definition")
    :   Constraint axis Lock options relative to Bone or Target reference

        Type:
        :   enum in [`'BONE'`, `'TARGET'`], default `'BONE'`

    subtarget[¶](#bpy.types.KinematicConstraint.subtarget "Link to this definition")
    :   Armature bone, mesh or lattice vertex group, …

        Type:
        :   string, default “”, (never None)

    target[¶](#bpy.types.KinematicConstraint.target "Link to this definition")
    :   Target object

        Type:
        :   [`Object`](Object.md#bpy.types.Object "bpy.types.Object")

    use\_location[¶](#bpy.types.KinematicConstraint.use_location "Link to this definition")
    :   Chain follows position of target

        Type:
        :   boolean, default False

    use\_rotation[¶](#bpy.types.KinematicConstraint.use_rotation "Link to this definition")
    :   Chain follows rotation of target

        Type:
        :   boolean, default False

    use\_stretch[¶](#bpy.types.KinematicConstraint.use_stretch "Link to this definition")
    :   Enable IK Stretching

        Type:
        :   boolean, default False

    use\_tail[¶](#bpy.types.KinematicConstraint.use_tail "Link to this definition")
    :   Include bone’s tail as last element in chain

        Type:
        :   boolean, default False

    weight[¶](#bpy.types.KinematicConstraint.weight "Link to this definition")
    :   For Tree-IK: Weight of position control for this target

        Type:
        :   float in [0.01, 1], default 0.0
