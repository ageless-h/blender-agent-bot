# FreestyleSettings(bpy\_struct)[¶](#freestylesettings-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.FreestyleSettings(*bpy\_struct*)[¶](#bpy.types.FreestyleSettings "Link to this definition")
:   Freestyle settings for a ViewLayer data-block

    as\_render\_pass[¶](#bpy.types.FreestyleSettings.as_render_pass "Link to this definition")
    :   Renders Freestyle output to a separate pass instead of overlaying it on the Combined pass

        Type:
        :   boolean, default False

    crease\_angle[¶](#bpy.types.FreestyleSettings.crease_angle "Link to this definition")
    :   Angular threshold for detecting crease edges

        Type:
        :   float in [0, 3.14159], default 0.0

    kr\_derivative\_epsilon[¶](#bpy.types.FreestyleSettings.kr_derivative_epsilon "Link to this definition")
    :   Kr derivative epsilon for computing suggestive contours

        Type:
        :   float in [-1000, 1000], default 0.0

    linesets[¶](#bpy.types.FreestyleSettings.linesets "Link to this definition")
    :   Type:
        :   [`Linesets`](Linesets.md#bpy.types.Linesets "bpy.types.Linesets") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`FreestyleLineSet`](FreestyleLineSet.md#bpy.types.FreestyleLineSet "bpy.types.FreestyleLineSet"), (readonly)

    mode[¶](#bpy.types.FreestyleSettings.mode "Link to this definition")
    :   Select the Freestyle control mode

        * `SCRIPT`
          Python Scripting – Advanced mode for using style modules written in Python.
        * `EDITOR`
          Parameter Editor – Basic mode for interactive style parameter editing.

        Type:
        :   enum in [`'SCRIPT'`, `'EDITOR'`], default `'SCRIPT'`

    modules[¶](#bpy.types.FreestyleSettings.modules "Link to this definition")
    :   A list of style modules (to be applied from top to bottom)

        Type:
        :   [`FreestyleModules`](FreestyleModules.md#bpy.types.FreestyleModules "bpy.types.FreestyleModules") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`FreestyleModuleSettings`](FreestyleModuleSettings.md#bpy.types.FreestyleModuleSettings "bpy.types.FreestyleModuleSettings"), (readonly)

    sphere\_radius[¶](#bpy.types.FreestyleSettings.sphere_radius "Link to this definition")
    :   Sphere radius for computing curvatures

        Type:
        :   float in [0, 1000], default 1.0

    use\_culling[¶](#bpy.types.FreestyleSettings.use_culling "Link to this definition")
    :   If enabled, out-of-view edges are ignored

        Type:
        :   boolean, default False

    use\_material\_boundaries[¶](#bpy.types.FreestyleSettings.use_material_boundaries "Link to this definition")
    :   Enable material boundaries

        Type:
        :   boolean, default False

    use\_ridges\_and\_valleys[¶](#bpy.types.FreestyleSettings.use_ridges_and_valleys "Link to this definition")
    :   Enable ridges and valleys

        Type:
        :   boolean, default False

    use\_smoothness[¶](#bpy.types.FreestyleSettings.use_smoothness "Link to this definition")
    :   Take face smoothness into account in view map calculation

        Type:
        :   boolean, default False

    use\_suggestive\_contours[¶](#bpy.types.FreestyleSettings.use_suggestive_contours "Link to this definition")
    :   Enable suggestive contours

        Type:
        :   boolean, default False

    use\_view\_map\_cache[¶](#bpy.types.FreestyleSettings.use_view_map_cache "Link to this definition")
    :   Keep the computed view map and avoid recalculating it if mesh geometry is unchanged

        Type:
        :   boolean, default False

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.FreestyleSettings.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.
