bpy.ops.lattice.flip(*\**, *axis='U'*)[¶](#bpy.ops.lattice.flip "Link to this definition")
:   Mirror all control points without inverting the lattice deform

    Parameters:
    :   **axis** (enum in [`'U'`, `'V'`, `'W'`], (optional)) – Flip Axis, Coordinates along this axis get flipped

bpy.ops.lattice.make\_regular()[¶](#bpy.ops.lattice.make_regular "Link to this definition")
:   Set UVW control points a uniform distance apart

bpy.ops.lattice.select\_all(*\**, *action='TOGGLE'*)[¶](#bpy.ops.lattice.select_all "Link to this definition")
:   Change selection of all UVW control points

    Parameters:
    :   **action** (enum in [`'TOGGLE'`, `'SELECT'`, `'DESELECT'`, `'INVERT'`], (optional)) –

        Action, Selection action to execute

        * `TOGGLE`
          Toggle – Toggle selection for all elements.
        * `SELECT`
          Select – Select all elements.
        * `DESELECT`
          Deselect – Deselect all elements.
        * `INVERT`
          Invert – Invert selection of all elements.

bpy.ops.lattice.select\_less()[¶](#bpy.ops.lattice.select_less "Link to this definition")
:   Deselect vertices at the boundary of each selection region

bpy.ops.lattice.select\_mirror(*\**, *axis={'X'}*, *extend=False*)[¶](#bpy.ops.lattice.select_mirror "Link to this definition")
:   Select mirrored lattice points

    Parameters:
    :   * **axis** (enum set in [Axis Flag Xyz Items](../../../appendix/bpy_types_enum_items/axis_flag_xyz_items.md#rna-enum-axis-flag-xyz-items), (optional)) – Axis
        * **extend** (*boolean**,* *(**optional**)*) – Extend, Extend the selection

bpy.ops.lattice.select\_more()[¶](#bpy.ops.lattice.select_more "Link to this definition")
:   Select vertex directly linked to already selected ones

bpy.ops.lattice.select\_random(*\**, *ratio=0.5*, *seed=0*, *action='SELECT'*)[¶](#bpy.ops.lattice.select_random "Link to this definition")
:   Randomly select UVW control points

    Parameters:
    :   * **ratio** (*float in* *[**0**,* *1**]**,* *(**optional**)*) – Ratio, Portion of items to select randomly
        * **seed** (*int in* *[**0**,* *inf**]**,* *(**optional**)*) – Random Seed, Seed for the random number generator
        * **action** (enum in [`'SELECT'`, `'DESELECT'`], (optional)) –

          Action, Selection action to execute

          + `SELECT`
            Select – Select all elements.
          + `DESELECT`
            Deselect – Deselect all elements.

bpy.ops.lattice.select\_ungrouped(*\**, *extend=False*)[¶](#bpy.ops.lattice.select_ungrouped "Link to this definition")
:   Select vertices without a group

    Parameters:
    :   **extend** (*boolean**,* *(**optional**)*) – Extend, Extend the selection
