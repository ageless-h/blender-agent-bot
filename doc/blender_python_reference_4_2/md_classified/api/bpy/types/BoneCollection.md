# BoneCollection(bpy\_struct)[#](#bonecollection-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.BoneCollection(*bpy\_struct*)[#](#bpy.types.BoneCollection "Link to this definition")
:   Bone collection in an Armature data-block

    bones[#](#bpy.types.BoneCollection.bones "Link to this definition")
    :   Bones assigned to this bone collection. In armature edit mode this will always return an empty list of bones, as the bone collection memberships are only synchronized when exiting edit mode

        Type:
        :   [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`Bone`](Bone.md#bpy.types.Bone "bpy.types.Bone"), (readonly)

    child\_number[#](#bpy.types.BoneCollection.child_number "Link to this definition")
    :   Index of this collection into its parent’s list of children. Note that finding this index requires a scan of all the bone collections, so do access this with care

        Type:
        :   int in [-inf, inf], default 0

    children[#](#bpy.types.BoneCollection.children "Link to this definition")
    :   Type:
        :   [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`BoneCollection`](#bpy.types.BoneCollection "bpy.types.BoneCollection"), (readonly)

    index[#](#bpy.types.BoneCollection.index "Link to this definition")
    :   Index of this bone collection in the armature.collections\_all array. Note that finding this index requires a scan of all the bone collections, so do access this with care

        Type:
        :   int in [-inf, inf], default 0, (readonly)

    is\_editable[#](#bpy.types.BoneCollection.is_editable "Link to this definition")
    :   This collection is owned by a local Armature, or was added via a library override in the current blend file

        Type:
        :   boolean, default False, (readonly)

    is\_expanded[#](#bpy.types.BoneCollection.is_expanded "Link to this definition")
    :   This bone collection is expanded in the bone collections tree view

        Type:
        :   boolean, default False

    is\_local\_override[#](#bpy.types.BoneCollection.is_local_override "Link to this definition")
    :   This collection was added via a library override in the current blend file

        Type:
        :   boolean, default False, (readonly)

    is\_solo[#](#bpy.types.BoneCollection.is_solo "Link to this definition")
    :   Show only this bone collection, and others also marked as ‘solo’

        Type:
        :   boolean, default False

    is\_visible[#](#bpy.types.BoneCollection.is_visible "Link to this definition")
    :   Bones in this collection will be visible in pose/object mode

        Type:
        :   boolean, default False

    is\_visible\_ancestors[#](#bpy.types.BoneCollection.is_visible_ancestors "Link to this definition")
    :   True when all of the ancestors of this bone collection are marked as visible; always True for root bone collections

        Type:
        :   boolean, default False, (readonly)

    is\_visible\_effectively[#](#bpy.types.BoneCollection.is_visible_effectively "Link to this definition")
    :   Whether this bone collection is effectively visible in the viewport. This is True when this bone collection and all of its ancestors are visible, or when it is marked as ‘solo’

        Type:
        :   boolean, default False, (readonly)

    name[#](#bpy.types.BoneCollection.name "Link to this definition")
    :   Unique within the Armature

        Type:
        :   string, default “”, (never None)

    parent[#](#bpy.types.BoneCollection.parent "Link to this definition")
    :   Parent bone collection. Note that accessing this requires a scan of all the bone collections to find the parent

        Type:
        :   [`BoneCollection`](#bpy.types.BoneCollection "bpy.types.BoneCollection")

    bones\_recursive[#](#bpy.types.BoneCollection.bones_recursive "Link to this definition")
    :   A set of all bones assigned to this bone collection and its child collections.

        (readonly)

    assign(*bone*)[#](#bpy.types.BoneCollection.assign "Link to this definition")
    :   Assign the given bone to this collection

        Parameters:
        :   **bone** ([`AnyType`](AnyType.md#bpy.types.AnyType "bpy.types.AnyType")) – Bone to assign to this collection. This must be a Bone, PoseBone, or EditBone

        Returns:
        :   Assigned, Whether the bone was actually assigned; will be false if the bone was already member of the collection

        Return type:
        :   boolean

    unassign(*bone*)[#](#bpy.types.BoneCollection.unassign "Link to this definition")
    :   Remove the given bone from this collection

        Parameters:
        :   **bone** ([`AnyType`](AnyType.md#bpy.types.AnyType "bpy.types.AnyType")) – Bone to remove from this collection. This must be a Bone, PoseBone, or EditBone

        Returns:
        :   Unassigned, Whether the bone was actually removed; will be false if the bone was not a member of the collection to begin with

        Return type:
        :   boolean

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*)[#](#bpy.types.BoneCollection.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*string*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass

    *classmethod* bl\_rna\_get\_subclass\_py(*id*, *default=None*)[#](#bpy.types.BoneCollection.bl_rna_get_subclass_py "Link to this definition")
    :   Parameters:
        :   **id** (*string*) – The RNA type identifier.

        Returns:
        :   The class or default when not found.

        Return type:
        :   type
