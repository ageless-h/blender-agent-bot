# ClothSettings(bpy\_struct)[#](#clothsettings-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.ClothSettings(*bpy\_struct*)[#](#bpy.types.ClothSettings "Link to this definition")
:   Cloth simulation settings for an object

    air\_damping[#](#bpy.types.ClothSettings.air_damping "Link to this definition")
    :   Air has normally some thickness which slows falling things down

        Type:
        :   float in [0, 10], default 1.0

    bending\_damping[#](#bpy.types.ClothSettings.bending_damping "Link to this definition")
    :   Amount of damping in bending behavior

        Type:
        :   float in [0, 1000], default 0.5

    bending\_model[#](#bpy.types.ClothSettings.bending_model "Link to this definition")
    :   Physical model for simulating bending forces

        * `ANGULAR`
          Angular – Cloth model with angular bending springs.
        * `LINEAR`
          Linear – Cloth model with linear bending springs (legacy).

        Type:
        :   enum in [‘ANGULAR’, ‘LINEAR’], default ‘ANGULAR’

    bending\_stiffness[#](#bpy.types.ClothSettings.bending_stiffness "Link to this definition")
    :   How much the material resists bending

        Type:
        :   float in [0, 10000], default 0.5

    bending\_stiffness\_max[#](#bpy.types.ClothSettings.bending_stiffness_max "Link to this definition")
    :   Maximum bending stiffness value

        Type:
        :   float in [0, 10000], default 0.5

    collider\_friction[#](#bpy.types.ClothSettings.collider_friction "Link to this definition")
    :   Type:
        :   float in [0, 1], default 0.0

    compression\_damping[#](#bpy.types.ClothSettings.compression_damping "Link to this definition")
    :   Amount of damping in compression behavior

        Type:
        :   float in [0, 50], default 5.0

    compression\_stiffness[#](#bpy.types.ClothSettings.compression_stiffness "Link to this definition")
    :   How much the material resists compression

        Type:
        :   float in [0, 10000], default 15.0

    compression\_stiffness\_max[#](#bpy.types.ClothSettings.compression_stiffness_max "Link to this definition")
    :   Maximum compression stiffness value

        Type:
        :   float in [0, 10000], default 15.0

    density\_strength[#](#bpy.types.ClothSettings.density_strength "Link to this definition")
    :   Influence of target density on the simulation

        Type:
        :   float in [0, 1], default 0.0

    density\_target[#](#bpy.types.ClothSettings.density_target "Link to this definition")
    :   Maximum density of hair

        Type:
        :   float in [0, 10000], default 0.0

    effector\_weights[#](#bpy.types.ClothSettings.effector_weights "Link to this definition")
    :   Type:
        :   [`EffectorWeights`](EffectorWeights.md#bpy.types.EffectorWeights "bpy.types.EffectorWeights"), (readonly)

    fluid\_density[#](#bpy.types.ClothSettings.fluid_density "Link to this definition")
    :   Density (kg/l) of the fluid contained inside the object, used to create a hydrostatic pressure gradient simulating the weight of the internal fluid, or buoyancy from the surrounding fluid if negative

        Type:
        :   float in [-inf, inf], default 0.0

    goal\_default[#](#bpy.types.ClothSettings.goal_default "Link to this definition")
    :   Default Goal (vertex target position) value, when no Vertex Group used

        Type:
        :   float in [0, 1], default 0.0

    goal\_friction[#](#bpy.types.ClothSettings.goal_friction "Link to this definition")
    :   Goal (vertex target position) friction

        Type:
        :   float in [0, 50], default 0.0

    goal\_max[#](#bpy.types.ClothSettings.goal_max "Link to this definition")
    :   Goal maximum, vertex group weights are scaled to match this range

        Type:
        :   float in [0, 1], default 1.0

    goal\_min[#](#bpy.types.ClothSettings.goal_min "Link to this definition")
    :   Goal minimum, vertex group weights are scaled to match this range

        Type:
        :   float in [0, 1], default 0.0

    goal\_spring[#](#bpy.types.ClothSettings.goal_spring "Link to this definition")
    :   Goal (vertex target position) spring stiffness

        Type:
        :   float in [0, 0.999], default 1.0

    gravity[#](#bpy.types.ClothSettings.gravity "Link to this definition")
    :   Gravity or external force vector

        Type:
        :   [`mathutils.Vector`](../../mathutils/index.md#mathutils.Vector "mathutils.Vector") of 3 items in [-100, 100], default (0.0, 0.0, -9.81)

    internal\_compression\_stiffness[#](#bpy.types.ClothSettings.internal_compression_stiffness "Link to this definition")
    :   How much the material resists compression

        Type:
        :   float in [0, 10000], default 15.0

    internal\_compression\_stiffness\_max[#](#bpy.types.ClothSettings.internal_compression_stiffness_max "Link to this definition")
    :   Maximum compression stiffness value

        Type:
        :   float in [0, 10000], default 15.0

    internal\_friction[#](#bpy.types.ClothSettings.internal_friction "Link to this definition")
    :   Type:
        :   float in [0, 1], default 0.0

    internal\_spring\_max\_diversion[#](#bpy.types.ClothSettings.internal_spring_max_diversion "Link to this definition")
    :   How much the rays used to connect the internal points can diverge from the vertex normal

        Type:
        :   float in [0, 0.785398], default 0.785398

    internal\_spring\_max\_length[#](#bpy.types.ClothSettings.internal_spring_max_length "Link to this definition")
    :   The maximum length an internal spring can have during creation. If the distance between internal points is greater than this, no internal spring will be created between these points. A length of zero means that there is no length limit

        Type:
        :   float in [0, 1000], default 0.0

    internal\_spring\_normal\_check[#](#bpy.types.ClothSettings.internal_spring_normal_check "Link to this definition")
    :   Require the points the internal springs connect to have opposite normal directions

        Type:
        :   boolean, default True

    internal\_tension\_stiffness[#](#bpy.types.ClothSettings.internal_tension_stiffness "Link to this definition")
    :   How much the material resists stretching

        Type:
        :   float in [0, 10000], default 15.0

    internal\_tension\_stiffness\_max[#](#bpy.types.ClothSettings.internal_tension_stiffness_max "Link to this definition")
    :   Maximum tension stiffness value

        Type:
        :   float in [0, 10000], default 15.0

    mass[#](#bpy.types.ClothSettings.mass "Link to this definition")
    :   The mass of each vertex on the cloth material

        Type:
        :   float in [0, inf], default 0.3

    pin\_stiffness[#](#bpy.types.ClothSettings.pin_stiffness "Link to this definition")
    :   Pin (vertex target position) spring stiffness

        Type:
        :   float in [0, 50], default 1.0

    pressure\_factor[#](#bpy.types.ClothSettings.pressure_factor "Link to this definition")
    :   Ambient pressure (kPa) that balances out between the inside and outside of the object when it has the target volume

        Type:
        :   float in [0, 10000], default 1.0

    quality[#](#bpy.types.ClothSettings.quality "Link to this definition")
    :   Quality of the simulation in steps per frame (higher is better quality but slower)

        Type:
        :   int in [1, inf], default 5

    rest\_shape\_key[#](#bpy.types.ClothSettings.rest_shape_key "Link to this definition")
    :   Shape key to use the rest spring lengths from

        Type:
        :   [`ShapeKey`](ShapeKey.md#bpy.types.ShapeKey "bpy.types.ShapeKey")

    sewing\_force\_max[#](#bpy.types.ClothSettings.sewing_force_max "Link to this definition")
    :   Maximum sewing force

        Type:
        :   float in [0, 10000], default 0.0

    shear\_damping[#](#bpy.types.ClothSettings.shear_damping "Link to this definition")
    :   Amount of damping in shearing behavior

        Type:
        :   float in [0, 50], default 5.0

    shear\_stiffness[#](#bpy.types.ClothSettings.shear_stiffness "Link to this definition")
    :   How much the material resists shearing

        Type:
        :   float in [0, 10000], default 5.0

    shear\_stiffness\_max[#](#bpy.types.ClothSettings.shear_stiffness_max "Link to this definition")
    :   Maximum shear scaling value

        Type:
        :   float in [0, 10000], default 5.0

    shrink\_max[#](#bpy.types.ClothSettings.shrink_max "Link to this definition")
    :   Max amount to shrink cloth by

        Type:
        :   float in [-inf, 1], default 0.0

    shrink\_min[#](#bpy.types.ClothSettings.shrink_min "Link to this definition")
    :   Factor by which to shrink cloth

        Type:
        :   float in [-inf, 1], default 0.0

    target\_volume[#](#bpy.types.ClothSettings.target_volume "Link to this definition")
    :   The mesh volume where the inner/outer pressure will be the same. If set to zero the change in volume will not affect pressure

        Type:
        :   float in [0, 10000], default 0.0

    tension\_damping[#](#bpy.types.ClothSettings.tension_damping "Link to this definition")
    :   Amount of damping in stretching behavior

        Type:
        :   float in [0, 50], default 5.0

    tension\_stiffness[#](#bpy.types.ClothSettings.tension_stiffness "Link to this definition")
    :   How much the material resists stretching

        Type:
        :   float in [0, 10000], default 15.0

    tension\_stiffness\_max[#](#bpy.types.ClothSettings.tension_stiffness_max "Link to this definition")
    :   Maximum tension stiffness value

        Type:
        :   float in [0, 10000], default 15.0

    time\_scale[#](#bpy.types.ClothSettings.time_scale "Link to this definition")
    :   Cloth speed is multiplied by this value

        Type:
        :   float in [0, inf], default 1.0

    uniform\_pressure\_force[#](#bpy.types.ClothSettings.uniform_pressure_force "Link to this definition")
    :   The uniform pressure that is constantly applied to the mesh, in units of Pressure Scale. Can be negative

        Type:
        :   float in [-10000, 10000], default 0.0

    use\_dynamic\_mesh[#](#bpy.types.ClothSettings.use_dynamic_mesh "Link to this definition")
    :   Make simulation respect deformations in the base mesh

        Type:
        :   boolean, default False

    use\_internal\_springs[#](#bpy.types.ClothSettings.use_internal_springs "Link to this definition")
    :   Simulate an internal volume structure by creating springs connecting the opposite sides of the mesh

        Type:
        :   boolean, default False

    use\_pressure[#](#bpy.types.ClothSettings.use_pressure "Link to this definition")
    :   Simulate pressure inside a closed cloth mesh

        Type:
        :   boolean, default False

    use\_pressure\_volume[#](#bpy.types.ClothSettings.use_pressure_volume "Link to this definition")
    :   Use the Target Volume parameter as the initial volume, instead of calculating it from the mesh itself

        Type:
        :   boolean, default False

    use\_sewing\_springs[#](#bpy.types.ClothSettings.use_sewing_springs "Link to this definition")
    :   Pulls loose edges together

        Type:
        :   boolean, default False

    vertex\_group\_bending[#](#bpy.types.ClothSettings.vertex_group_bending "Link to this definition")
    :   Vertex group for fine control over bending stiffness

        Type:
        :   string, default “”, (never None)

    vertex\_group\_intern[#](#bpy.types.ClothSettings.vertex_group_intern "Link to this definition")
    :   Vertex group for fine control over the internal spring stiffness

        Type:
        :   string, default “”, (never None)

    vertex\_group\_mass[#](#bpy.types.ClothSettings.vertex_group_mass "Link to this definition")
    :   Vertex Group for pinning of vertices

        Type:
        :   string, default “”, (never None)

    vertex\_group\_pressure[#](#bpy.types.ClothSettings.vertex_group_pressure "Link to this definition")
    :   Vertex Group for where to apply pressure. Zero weight means no pressure while a weight of one means full pressure. Faces with a vertex that has zero weight will be excluded from the volume calculation

        Type:
        :   string, default “”, (never None)

    vertex\_group\_shear\_stiffness[#](#bpy.types.ClothSettings.vertex_group_shear_stiffness "Link to this definition")
    :   Vertex group for fine control over shear stiffness

        Type:
        :   string, default “”, (never None)

    vertex\_group\_shrink[#](#bpy.types.ClothSettings.vertex_group_shrink "Link to this definition")
    :   Vertex Group for shrinking cloth

        Type:
        :   string, default “”, (never None)

    vertex\_group\_structural\_stiffness[#](#bpy.types.ClothSettings.vertex_group_structural_stiffness "Link to this definition")
    :   Vertex group for fine control over structural stiffness

        Type:
        :   string, default “”, (never None)

    voxel\_cell\_size[#](#bpy.types.ClothSettings.voxel_cell_size "Link to this definition")
    :   Size of the voxel grid cells for interaction effects

        Type:
        :   float in [0.0001, 10000], default 0.1
