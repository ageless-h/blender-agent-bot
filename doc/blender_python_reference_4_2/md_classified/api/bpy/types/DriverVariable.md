# DriverVariable(bpy\_struct)[#](#drivervariable-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.DriverVariable(*bpy\_struct*)[#](#bpy.types.DriverVariable "Link to this definition")
:   Variable from some source/target for driver relationship

    is\_name\_valid[#](#bpy.types.DriverVariable.is_name_valid "Link to this definition")
    :   Is this a valid name for a driver variable

        Type:
        :   boolean, default False, (readonly)

    name[#](#bpy.types.DriverVariable.name "Link to this definition")
    :   Name to use in scripted expressions/functions (no spaces or dots are allowed, and must start with a letter)

        Type:
        :   string, default “”, (never None)

    targets[#](#bpy.types.DriverVariable.targets "Link to this definition")
    :   Sources of input data for evaluating this variable

        Type:
        :   [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`DriverTarget`](DriverTarget.md#bpy.types.DriverTarget "bpy.types.DriverTarget"), (readonly)

    type[#](#bpy.types.DriverVariable.type "Link to this definition")
    :   Driver variable type

        * `SINGLE_PROP`
          Single Property – Use the value from some RNA property.
        * `TRANSFORMS`
          Transform Channel – Final transformation value of object or bone.
        * `ROTATION_DIFF`
          Rotational Difference – Use the angle between two bones.
        * `LOC_DIFF`
          Distance – Distance between two bones or objects.
        * `CONTEXT_PROP`
          Context Property – Use the value from some RNA property within the current evaluation context.

        Type:
        :   enum in [‘SINGLE\_PROP’, ‘TRANSFORMS’, ‘ROTATION\_DIFF’, ‘LOC\_DIFF’, ‘CONTEXT\_PROP’], default ‘SINGLE\_PROP’

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*)[#](#bpy.types.DriverVariable.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*string*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass

    *classmethod* bl\_rna\_get\_subclass\_py(*id*, *default=None*)[#](#bpy.types.DriverVariable.bl_rna_get_subclass_py "Link to this definition")

        Returns:
