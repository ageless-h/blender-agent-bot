*class* bpy.types.ShaderNodeTexPointDensity(*ShaderNode*)[#](#bpy.types.ShaderNodeTexPointDensity "Link to this definition")
:   Generate a volumetric point for each particle or vertex of another object

    interpolation[#](#bpy.types.ShaderNodeTexPointDensity.interpolation "Link to this definition")
    :   Texture interpolation

        * `Closest`
          Closest – No interpolation (sample closest texel).
        * `Linear`
          Linear – Linear interpolation.
        * `Cubic`
          Cubic – Cubic interpolation.

        Type:
        :   enum in [‘Closest’, ‘Linear’, ‘Cubic’], default ‘Linear’

    object[#](#bpy.types.ShaderNodeTexPointDensity.object "Link to this definition")

        Type:
        :   [`Object`](Object.md#bpy.types.Object "bpy.types.Object")

    particle\_color\_source[#](#bpy.types.ShaderNodeTexPointDensity.particle_color_source "Link to this definition")
    :   Data to derive color results from

        * `PARTICLE_AGE`
          Particle Age – Lifetime mapped as 0.0 to 1.0 intensity.
        * `PARTICLE_SPEED`
          Particle Speed – Particle speed (absolute magnitude of velocity) mapped as 0.0 to 1.0 intensity.
        * `PARTICLE_VELOCITY`
          Particle Velocity – XYZ velocity mapped to RGB colors.

        Type:
        :   enum in [‘PARTICLE\_AGE’, ‘PARTICLE\_SPEED’, ‘PARTICLE\_VELOCITY’], default ‘PARTICLE\_AGE’

    :   Point data to use as renderable point density

        * `PARTICLE_SYSTEM`
          Particle System – Generate point density from a particle system.
        * `OBJECT`
          Object Vertices – Generate point density from an object’s vertices.

        Type:
        :   enum in [‘PARTICLE\_SYSTEM’, ‘OBJECT’], default ‘PARTICLE\_SYSTEM’

    radius[#](#bpy.types.ShaderNodeTexPointDensity.radius "Link to this definition")
    :   Radius from the shaded sample to look for points within

        Type:
        :   float in [0.001, inf], default 0.0

    resolution[#](#bpy.types.ShaderNodeTexPointDensity.resolution "Link to this definition")
    :   Resolution used by the texture holding the point density

        Type:
        :   int in [1, 32768], default 0

    space[#](#bpy.types.ShaderNodeTexPointDensity.space "Link to this definition")
    :   Coordinate system to calculate voxels in

        Type:
        :   enum in [‘OBJECT’, ‘WORLD’], default ‘OBJECT’

    vertex\_attribute\_name[#](#bpy.types.ShaderNodeTexPointDensity.vertex_attribute_name "Link to this definition")
    :   Vertex attribute to use for color

        Type:
        :   string, default “”, (never None)

    vertex\_color\_source[#](#bpy.types.ShaderNodeTexPointDensity.vertex_color_source "Link to this definition")
    :   Data to derive color results from

        * `VERTEX_COLOR`
          Vertex Color – Vertex color layer.
        * `VERTEX_WEIGHT`
          Vertex Weight – Vertex group weight.
        * `VERTEX_NORMAL`
          Vertex Normal – XYZ normal vector mapped to RGB colors.

        Type:
        :   enum in [‘VERTEX\_COLOR’, ‘VERTEX\_WEIGHT’, ‘VERTEX\_NORMAL’], default ‘VERTEX\_COLOR’
