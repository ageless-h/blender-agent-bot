# NormalEditModifier(Modifier)[#](#normaleditmodifier-modifier "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`Modifier`](Modifier.md#bpy.types.Modifier "bpy.types.Modifier")

*class* bpy.types.NormalEditModifier(*Modifier*)[#](#bpy.types.NormalEditModifier "Link to this definition")
:   Modifier affecting/generating custom normals

    invert\_vertex\_group[#](#bpy.types.NormalEditModifier.invert_vertex_group "Link to this definition")
    :   Invert vertex group influence

        Type:
        :   boolean, default False

    mix\_factor[#](#bpy.types.NormalEditModifier.mix_factor "Link to this definition")
    :   How much of generated normals to mix with existing ones

        Type:
        :   float in [0, 1], default 1.0

    mix\_limit[#](#bpy.types.NormalEditModifier.mix_limit "Link to this definition")
    :   Maximum angle between old and new normals

        Type:
        :   float in [0, 3.14159], default 3.14159

    mix\_mode[#](#bpy.types.NormalEditModifier.mix_mode "Link to this definition")
    :   How to mix generated normals with existing ones

        * `COPY`
          Copy – Copy new normals (overwrite existing).
        * `ADD`
          Add – Copy sum of new and old normals.
        * `SUB`
          Subtract – Copy new normals minus old normals.
        * `MUL`
          Multiply – Copy product of old and new normals (not cross product).

        Type:
        :   enum in [‘COPY’, ‘ADD’, ‘SUB’, ‘MUL’], default ‘COPY’

    mode[#](#bpy.types.NormalEditModifier.mode "Link to this definition")
    :   How to affect (generate) normals

        * `RADIAL`
          Radial – From an ellipsoid (shape defined by the boundbox’s dimensions, target is optional).
        * `DIRECTIONAL`
          Directional – Normals ‘track’ (point to) the target object.

        Type:
        :   enum in [‘RADIAL’, ‘DIRECTIONAL’], default ‘RADIAL’

    no\_polynors\_fix[#](#bpy.types.NormalEditModifier.no_polynors_fix "Link to this definition")
    :   Do not flip polygons when their normals are not consistent with their newly computed custom vertex normals

        Type:
        :   boolean, default False

    offset[#](#bpy.types.NormalEditModifier.offset "Link to this definition")
    :   Offset from object’s center

        Type:
        :   [`mathutils.Vector`](../../mathutils/index.md#mathutils.Vector "mathutils.Vector") of 3 items in [-inf, inf], default (0.0, 0.0, 0.0)

    target[#](#bpy.types.NormalEditModifier.target "Link to this definition")
    :   Target object used to affect normals

        Type:
        :   [`Object`](Object.md#bpy.types.Object "bpy.types.Object")

    use\_direction\_parallel[#](#bpy.types.NormalEditModifier.use_direction_parallel "Link to this definition")
    :   Use same direction for all normals, from origin to target’s center (Directional mode only)

        Type:
        :   boolean, default True

    vertex\_group[#](#bpy.types.NormalEditModifier.vertex_group "Link to this definition")
    :   Vertex group name for selecting/weighting the affected areas

        Type:
        :   string, default “”, (never None)

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*)[#](#bpy.types.NormalEditModifier.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*string*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass
