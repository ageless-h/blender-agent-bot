# GPencilSculptSettings(bpy\_struct)[#](#gpencilsculptsettings-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.GPencilSculptSettings(*bpy\_struct*)[#](#bpy.types.GPencilSculptSettings "Link to this definition")
:   General properties for Grease Pencil stroke sculpting tools

    guide[#](#bpy.types.GPencilSculptSettings.guide "Link to this definition")
    :   Type:
        :   [`GPencilSculptGuide`](GPencilSculptGuide.md#bpy.types.GPencilSculptGuide "bpy.types.GPencilSculptGuide"), (readonly)

    intersection\_threshold[#](#bpy.types.GPencilSculptSettings.intersection_threshold "Link to this definition")
    :   Threshold for stroke intersections

        Type:
        :   float in [0, 10], default 0.1

    lock\_axis[#](#bpy.types.GPencilSculptSettings.lock_axis "Link to this definition")
    :   * `VIEW`
          View – Align strokes to current view plane.
        * `AXIS_Y`
          Front (X-Z) – Project strokes to plane locked to Y.
        * `AXIS_X`
          Side (Y-Z) – Project strokes to plane locked to X.
        * `AXIS_Z`
          Top (X-Y) – Project strokes to plane locked to Z.
        * `CURSOR`
          Cursor – Align strokes to current 3D cursor orientation.

        Type:
        :   enum in [‘VIEW’, ‘AXIS\_Y’, ‘AXIS\_X’, ‘AXIS\_Z’, ‘CURSOR’], default ‘VIEW’

    multiframe\_falloff\_curve[#](#bpy.types.GPencilSculptSettings.multiframe_falloff_curve "Link to this definition")
    :   Custom curve to control falloff of brush effect by Grease Pencil frames

        Type:
        :   [`CurveMapping`](CurveMapping.md#bpy.types.CurveMapping "bpy.types.CurveMapping"), (readonly)

    thickness\_primitive\_curve[#](#bpy.types.GPencilSculptSettings.thickness_primitive_curve "Link to this definition")
    :   Custom curve to control primitive thickness

        Type:
        :   [`CurveMapping`](CurveMapping.md#bpy.types.CurveMapping "bpy.types.CurveMapping"), (readonly)

    use\_automasking\_layer\_active[#](#bpy.types.GPencilSculptSettings.use_automasking_layer_active "Link to this definition")
    :   Affect only the Active Layer

        Type:
        :   boolean, default False

    use\_automasking\_layer\_stroke[#](#bpy.types.GPencilSculptSettings.use_automasking_layer_stroke "Link to this definition")
    :   Affect only strokes below the cursor

        Type:
        :   boolean, default False

    use\_automasking\_material\_active[#](#bpy.types.GPencilSculptSettings.use_automasking_material_active "Link to this definition")
    :   Affect only the Active Material

        Type:
        :   boolean, default False

    use\_automasking\_material\_stroke[#](#bpy.types.GPencilSculptSettings.use_automasking_material_stroke "Link to this definition")
    :   Affect only strokes below the cursor

        Type:
        :   boolean, default False

    use\_automasking\_stroke[#](#bpy.types.GPencilSculptSettings.use_automasking_stroke "Link to this definition")
    :   Affect only strokes below the cursor

        Type:
        :   boolean, default False

    use\_multiframe\_falloff[#](#bpy.types.GPencilSculptSettings.use_multiframe_falloff "Link to this definition")
    :   Use falloff effect when edit in multiframe mode to compute brush effect by frame

        Type:
        :   boolean, default False

    use\_scale\_thickness[#](#bpy.types.GPencilSculptSettings.use_scale_thickness "Link to this definition")
    :   Scale the stroke thickness when transforming strokes

        Type:
        :   boolean, default False

    use\_thickness\_curve[#](#bpy.types.GPencilSculptSettings.use_thickness_curve "Link to this definition")
    :   Use curve to define primitive stroke thickness

        Type:
        :   boolean, default False
