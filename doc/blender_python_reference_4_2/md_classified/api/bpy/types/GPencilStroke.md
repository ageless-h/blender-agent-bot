*class* bpy.types.GPencilStroke(*bpy\_struct*)[#](#bpy.types.GPencilStroke "Link to this definition")
:   Freehand curve defining part of a sketch

    aspect[#](#bpy.types.GPencilStroke.aspect "Link to this definition")
    :   Type:
        :   [`mathutils.Vector`](../../mathutils/index.md#mathutils.Vector "mathutils.Vector") of 2 items in [0.01, 1], default (1.0, 1.0)

    bound\_box\_max[#](#bpy.types.GPencilStroke.bound_box_max "Link to this definition")
    :   Type:
        :   [`mathutils.Vector`](../../mathutils/index.md#mathutils.Vector "mathutils.Vector") of 3 items in [-inf, inf], default (0.0, 0.0, 0.0), (readonly)

    bound\_box\_min[#](#bpy.types.GPencilStroke.bound_box_min "Link to this definition")
    :   Type:
        :   [`mathutils.Vector`](../../mathutils/index.md#mathutils.Vector "mathutils.Vector") of 3 items in [-inf, inf], default (0.0, 0.0, 0.0), (readonly)

    display\_mode[#](#bpy.types.GPencilStroke.display_mode "Link to this definition")
    :   Coordinate space that stroke is in

        * `SCREEN`
          Screen – Stroke is in screen-space.
        * `3DSPACE`
          3D Space – Stroke is in 3D-space.
        * `2DSPACE`
          2D Space – Stroke is in 2D-space.
        * `2DIMAGE`
          2D Image – Stroke is in 2D-space (but with special ‘image’ scaling).

        Type:
        :   enum in [‘SCREEN’, ‘3DSPACE’, ‘2DSPACE’, ‘2DIMAGE’], default ‘SCREEN’

    edit\_curve[#](#bpy.types.GPencilStroke.edit_curve "Link to this definition")
    :   Temporary data for Edit Curve

        Type:
        :   [`GPencilEditCurve`](GPencilEditCurve.md#bpy.types.GPencilEditCurve "bpy.types.GPencilEditCurve"), (readonly)

    end\_cap\_mode[#](#bpy.types.GPencilStroke.end_cap_mode "Link to this definition")
    :   Stroke end extreme cap style

        Type:
        :   enum in [‘ROUND’, ‘FLAT’], default ‘ROUND’

    hardness[#](#bpy.types.GPencilStroke.hardness "Link to this definition")
    :   Amount of gradient along section of stroke

        Type:
        :   float in [0.001, 1], default 1.0

    has\_edit\_curve[#](#bpy.types.GPencilStroke.has_edit_curve "Link to this definition")
    :   Stroke has Curve data to edit shape

        Type:
        :   boolean, default False, (readonly)

    is\_nofill\_stroke[#](#bpy.types.GPencilStroke.is_nofill_stroke "Link to this definition")
    :   Special stroke to use as boundary for filling areas

        Type:
        :   boolean, default False

    line\_width[#](#bpy.types.GPencilStroke.line_width "Link to this definition")
    :   Thickness of stroke (in pixels)

        Type:
        :   int in [1, 1000], default 0

    material\_index[#](#bpy.types.GPencilStroke.material_index "Link to this definition")
    :   Material slot index of this stroke

        Type:
        :   int in [-inf, inf], default 0

    points[#](#bpy.types.GPencilStroke.points "Link to this definition")
    :   Stroke data points

        Type:
        :   [`GPencilStrokePoints`](GPencilStrokePoints.md#bpy.types.GPencilStrokePoints "bpy.types.GPencilStrokePoints") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`GPencilStrokePoint`](GPencilStrokePoint.md#bpy.types.GPencilStrokePoint "bpy.types.GPencilStrokePoint"), (readonly)

    select[#](#bpy.types.GPencilStroke.select "Link to this definition")
    :   Stroke is selected for viewport editing

        Type:
        :   boolean, default False

    select\_index[#](#bpy.types.GPencilStroke.select_index "Link to this definition")
    :   Index of selection used for interpolation

        Type:
        :   int in [-inf, inf], default 0

    start\_cap\_mode[#](#bpy.types.GPencilStroke.start_cap_mode "Link to this definition")
    :   Stroke start extreme cap style

        Type:
        :   enum in [‘ROUND’, ‘FLAT’], default ‘ROUND’

    time\_start[#](#bpy.types.GPencilStroke.time_start "Link to this definition")
    :   Initial time of the stroke

        Type:
        :   float in [-inf, inf], default 0.0, (readonly)

    triangles[#](#bpy.types.GPencilStroke.triangles "Link to this definition")
    :   Triangulation data for HQ fill

        Type:
        :   [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`GPencilTriangle`](GPencilTriangle.md#bpy.types.GPencilTriangle "bpy.types.GPencilTriangle"), (readonly)

    use\_cyclic[#](#bpy.types.GPencilStroke.use_cyclic "Link to this definition")
    :   Enable cyclic drawing, closing the stroke

        Type:
        :   boolean, default False

    uv\_rotation[#](#bpy.types.GPencilStroke.uv_rotation "Link to this definition")
    :   Rotation of the UV

        Type:
        :   float in [-inf, inf], default 0.0

    uv\_scale[#](#bpy.types.GPencilStroke.uv_scale "Link to this definition")
    :   Scale of the UV

        Type:
        :   float in [0.01, 100], default 1.0

    uv\_translation[#](#bpy.types.GPencilStroke.uv_translation "Link to this definition")
    :   Translation of default UV position

        Type:
        :   [`mathutils.Vector`](../../mathutils/index.md#mathutils.Vector "mathutils.Vector") of 2 items in [-inf, inf], default (0.0, 0.0)

    vertex\_color\_fill[#](#bpy.types.GPencilStroke.vertex_color_fill "Link to this definition")
    :   Color used to mix with fill color to get final color

        Type:
        :   float array of 4 items in [0, 1], default (0.0, 0.0, 0.0, 0.0)
