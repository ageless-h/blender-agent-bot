*class* bpy.types.GreasePencilDashModifierSegment(*bpy\_struct*)[#](#bpy.types.GreasePencilDashModifierSegment "Link to this definition")
:   Configuration for a single dash segment

    dash[#](#bpy.types.GreasePencilDashModifierSegment.dash "Link to this definition")
    :   The number of consecutive points from the original stroke to include in this segment

        Type:
        :   int in [1, 32767], default 2

    gap[#](#bpy.types.GreasePencilDashModifierSegment.gap "Link to this definition")
    :   The number of points skipped after this segment

        Type:
        :   int in [0, 32767], default 1

    material\_index[#](#bpy.types.GreasePencilDashModifierSegment.material_index "Link to this definition")
    :   Use this index on generated segment. -1 means using the existing material

        Type:
        :   int in [-1, 32767], default -1

    name[#](#bpy.types.GreasePencilDashModifierSegment.name "Link to this definition")
    :   Name of the dash segment

        Type:
        :   string, default “”, (never None)

    opacity[#](#bpy.types.GreasePencilDashModifierSegment.opacity "Link to this definition")
    :   The factor to apply to the original point’s opacity for the new points

        Type:
        :   float in [0, 1], default 1.0

    radius[#](#bpy.types.GreasePencilDashModifierSegment.radius "Link to this definition")
    :   The factor to apply to the original point’s radius for the new points

        Type:
        :   float in [0, 1], default 1.0

    use\_cyclic[#](#bpy.types.GreasePencilDashModifierSegment.use_cyclic "Link to this definition")
    :   Enable cyclic on individual stroke dashes

        Type:
        :   boolean, default False
