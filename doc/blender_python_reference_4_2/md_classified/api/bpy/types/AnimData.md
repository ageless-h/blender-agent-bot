# AnimData(bpy\_struct)[#](#animdata-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.AnimData(*bpy\_struct*)[#](#bpy.types.AnimData "Link to this definition")
:   Animation data for data-block

    action[#](#bpy.types.AnimData.action "Link to this definition")
    :   Active Action for this data-block

        Type:
        :   [`Action`](Action.md#bpy.types.Action "bpy.types.Action")

    action\_blend\_type[#](#bpy.types.AnimData.action_blend_type "Link to this definition")
    :   Method used for combining Active Action’s result with result of NLA stack

        * `REPLACE`
          Replace – The strip values replace the accumulated results by amount specified by influence.
        * `COMBINE`
          Combine – The strip values are combined with accumulated results by appropriately using addition, multiplication, or quaternion math, based on channel type.
        * `ADD`
          Add – Weighted result of strip is added to the accumulated results.
        * `SUBTRACT`
          Subtract – Weighted result of strip is removed from the accumulated results.
        * `MULTIPLY`
          Multiply – Weighted result of strip is multiplied with the accumulated results.

        Type:
        :   enum in [‘REPLACE’, ‘COMBINE’, ‘ADD’, ‘SUBTRACT’, ‘MULTIPLY’], default ‘REPLACE’

    action\_extrapolation[#](#bpy.types.AnimData.action_extrapolation "Link to this definition")
    :   Action to take for gaps past the Active Action’s range (when evaluating with NLA)

        * `NOTHING`
          Nothing – Strip has no influence past its extents.
        * `HOLD`
          Hold – Hold the first frame if no previous strips in track, and always hold last frame.
        * `HOLD_FORWARD`
          Hold Forward – Only hold last frame.

        Type:
        :   enum in [‘NOTHING’, ‘HOLD’, ‘HOLD\_FORWARD’], default ‘HOLD’

    action\_influence[#](#bpy.types.AnimData.action_influence "Link to this definition")
    :   Amount the Active Action contributes to the result of the NLA stack

        Type:
        :   float in [0, 1], default 1.0

    action\_tweak\_storage[#](#bpy.types.AnimData.action_tweak_storage "Link to this definition")
    :   Slot to temporarily hold the main action while in tweak mode

        Type:
        :   [`Action`](Action.md#bpy.types.Action "bpy.types.Action")

    drivers[#](#bpy.types.AnimData.drivers "Link to this definition")
    :   The Drivers/Expressions for this data-block

        Type:
        :   [`AnimDataDrivers`](AnimDataDrivers.md#bpy.types.AnimDataDrivers "bpy.types.AnimDataDrivers") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`FCurve`](FCurve.md#bpy.types.FCurve "bpy.types.FCurve"), (readonly)

    nla\_tracks[#](#bpy.types.AnimData.nla_tracks "Link to this definition")
    :   NLA Tracks (i.e. Animation Layers)

        Type:
        :   [`NlaTracks`](NlaTracks.md#bpy.types.NlaTracks "bpy.types.NlaTracks") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`NlaTrack`](NlaTrack.md#bpy.types.NlaTrack "bpy.types.NlaTrack"), (readonly)

    use\_nla[#](#bpy.types.AnimData.use_nla "Link to this definition")
    :   NLA stack is evaluated when evaluating this block

        Type:
        :   boolean, default False

    use\_pin[#](#bpy.types.AnimData.use_pin "Link to this definition")
    :   Type:
        :   boolean, default False

    use\_tweak\_mode[#](#bpy.types.AnimData.use_tweak_mode "Link to this definition")
    :   Whether to enable or disable tweak mode in NLA

        Type:
        :   boolean, default False

    nla\_tweak\_strip\_time\_to\_scene(*frame*, *invert=False*)[#](#bpy.types.AnimData.nla_tweak_strip_time_to_scene "Link to this definition")
    :   Convert a time value from the local time of the tweaked strip to scene time, exactly as done by built-in key editing tools. Returns the input time unchanged if not tweaking.

        Parameters:
        :   * **frame** (*float in* *[**-1.04857e+06**,* *1.04857e+06**]*) – Input time
            * **invert** (*boolean**,* *(**optional**)*) – Invert, Convert scene time to action time

        Returns:
        :   Converted time

        Return type:
        :   float in [-1.04857e+06, 1.04857e+06]
