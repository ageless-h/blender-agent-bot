# VertexWeightEditModifier(Modifier)[#](#vertexweighteditmodifier-modifier "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`Modifier`](Modifier.md#bpy.types.Modifier "bpy.types.Modifier")

*class* bpy.types.VertexWeightEditModifier(*Modifier*)[#](#bpy.types.VertexWeightEditModifier "Link to this definition")
:   Edit the weights of vertices in a group

    add\_threshold[#](#bpy.types.VertexWeightEditModifier.add_threshold "Link to this definition")
    :   Lower (inclusive) bound for a vertex’s weight to be added to the vgroup

        Type:
        :   float in [-1000, 1000], default 0.01

    default\_weight[#](#bpy.types.VertexWeightEditModifier.default_weight "Link to this definition")
    :   Default weight a vertex will have if it is not in the vgroup

        Type:
        :   float in [0, 1], default 0.0

    falloff\_type[#](#bpy.types.VertexWeightEditModifier.falloff_type "Link to this definition")
    :   How weights are mapped to their new values

        * `LINEAR`
          Linear – Null action.
        * `CURVE`
          Custom Curve.
        * `SHARP`
          Sharp.
        * `SMOOTH`
          Smooth.
        * `ROOT`
          Root.
        * `ICON_SPHERECURVE`
          Sphere.
        * `RANDOM`
          Random.
        * `STEP`
          Median Step – Map all values below 0.5 to 0.0, and all others to 1.0.

        Type:
        :   enum in [‘LINEAR’, ‘CURVE’, ‘SHARP’, ‘SMOOTH’, ‘ROOT’, ‘ICON\_SPHERECURVE’, ‘RANDOM’, ‘STEP’], default ‘LINEAR’

    invert\_falloff[#](#bpy.types.VertexWeightEditModifier.invert_falloff "Link to this definition")
    :   Invert the resulting falloff weight

        Type:
        :   boolean, default False

    invert\_mask\_vertex\_group[#](#bpy.types.VertexWeightEditModifier.invert_mask_vertex_group "Link to this definition")
    :   Invert vertex group mask influence

        Type:
        :   boolean, default False

    map\_curve[#](#bpy.types.VertexWeightEditModifier.map_curve "Link to this definition")
    :   Custom mapping curve

        Type:
        :   [`CurveMapping`](CurveMapping.md#bpy.types.CurveMapping "bpy.types.CurveMapping"), (readonly)

    mask\_constant[#](#bpy.types.VertexWeightEditModifier.mask_constant "Link to this definition")
    :   Global influence of current modifications on vgroup

        Type:
        :   float in [-inf, inf], default 1.0

    mask\_tex\_map\_bone[#](#bpy.types.VertexWeightEditModifier.mask_tex_map_bone "Link to this definition")
    :   Which bone to take texture coordinates from

        Type:
        :   string, default “”, (never None)

    mask\_tex\_map\_object[#](#bpy.types.VertexWeightEditModifier.mask_tex_map_object "Link to this definition")
    :   Which object to take texture coordinates from

        Type:
        :   [`Object`](Object.md#bpy.types.Object "bpy.types.Object")

    mask\_tex\_mapping[#](#bpy.types.VertexWeightEditModifier.mask_tex_mapping "Link to this definition")
    :   Which texture coordinates to use for mapping

        * `LOCAL`
          Local – Use local generated coordinates.
        * `GLOBAL`
          Global – Use global coordinates.
        * `OBJECT`
          Object – Use local generated coordinates of another object.
        * `UV`
          UV – Use coordinates from a UV layer.

        Type:
        :   enum in [‘LOCAL’, ‘GLOBAL’, ‘OBJECT’, ‘UV’], default ‘LOCAL’

    mask\_tex\_use\_channel[#](#bpy.types.VertexWeightEditModifier.mask_tex_use_channel "Link to this definition")
    :   Which texture channel to use for masking

        Type:
        :   enum in [‘INT’, ‘RED’, ‘GREEN’, ‘BLUE’, ‘HUE’, ‘SAT’, ‘VAL’, ‘ALPHA’], default ‘INT’

    mask\_tex\_uv\_layer[#](#bpy.types.VertexWeightEditModifier.mask_tex_uv_layer "Link to this definition")
    :   UV map name

        Type:
        :   string, default “”, (never None)

    mask\_texture[#](#bpy.types.VertexWeightEditModifier.mask_texture "Link to this definition")
    :   Masking texture

        Type:
        :   [`Texture`](Texture.md#bpy.types.Texture "bpy.types.Texture")

    mask\_vertex\_group[#](#bpy.types.VertexWeightEditModifier.mask_vertex_group "Link to this definition")
    :   Masking vertex group name

        Type:
        :   string, default “”, (never None)

    normalize[#](#bpy.types.VertexWeightEditModifier.normalize "Link to this definition")
    :   Normalize the resulting weights (otherwise they are only clamped within 0.0 to 1.0 range)

        Type:
        :   boolean, default False

    remove\_threshold[#](#bpy.types.VertexWeightEditModifier.remove_threshold "Link to this definition")
    :   Upper (inclusive) bound for a vertex’s weight to be removed from the vgroup

        Type:
        :   float in [-1000, 1000], default 0.01

    use\_add[#](#bpy.types.VertexWeightEditModifier.use_add "Link to this definition")
    :   Add vertices with weight over threshold to vgroup

        Type:
        :   boolean, default False

    use\_remove[#](#bpy.types.VertexWeightEditModifier.use_remove "Link to this definition")
    :   Remove vertices with weight below threshold from vgroup

        Type:
        :   boolean, default False

    vertex\_group[#](#bpy.types.VertexWeightEditModifier.vertex_group "Link to this definition")
    :   Vertex group name

        Type:
        :   string, default “”, (never None)

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*)[#](#bpy.types.VertexWeightEditModifier.bl_rna_get_subclass "Link to this definition")

        :   **id** (*string*) – The RNA type identifier.
