# DynamicPaintBrushSettings(bpy\_struct)[#](#dynamicpaintbrushsettings-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.DynamicPaintBrushSettings(*bpy\_struct*)[#](#bpy.types.DynamicPaintBrushSettings "Link to this definition")
:   Brush settings

    invert\_proximity[#](#bpy.types.DynamicPaintBrushSettings.invert_proximity "Link to this definition")
    :   Proximity falloff is applied inside the volume

        Type:
        :   boolean, default False

    paint\_alpha[#](#bpy.types.DynamicPaintBrushSettings.paint_alpha "Link to this definition")
    :   Paint alpha

        Type:
        :   float in [0, 1], default 0.0

    paint\_color[#](#bpy.types.DynamicPaintBrushSettings.paint_color "Link to this definition")
    :   Color of the paint

        Type:
        :   [`mathutils.Color`](../../mathutils/index.md#mathutils.Color "mathutils.Color") of 3 items in [0, inf], default (0.0, 0.0, 0.0)

    paint\_distance[#](#bpy.types.DynamicPaintBrushSettings.paint_distance "Link to this definition")
    :   Maximum distance from brush to mesh surface to affect paint

        Type:
        :   float in [0, 500], default 0.0

    paint\_ramp[#](#bpy.types.DynamicPaintBrushSettings.paint_ramp "Link to this definition")
    :   Color ramp used to define proximity falloff

        Type:
        :   [`ColorRamp`](ColorRamp.md#bpy.types.ColorRamp "bpy.types.ColorRamp"), (readonly)

    paint\_source[#](#bpy.types.DynamicPaintBrushSettings.paint_source "Link to this definition")
    :   Type:
        :   enum in [‘PARTICLE\_SYSTEM’, ‘POINT’, ‘DISTANCE’, ‘VOLUME\_DISTANCE’, ‘VOLUME’], default ‘VOLUME’

    paint\_wetness[#](#bpy.types.DynamicPaintBrushSettings.paint_wetness "Link to this definition")
    :   Paint wetness, visible in wetmap (some effects only affect wet paint)

        Type:
        :   float in [0, 1], default 0.0

    particle\_system[#](#bpy.types.DynamicPaintBrushSettings.particle_system "Link to this definition")
    :   The particle system to paint with

        Type:
        :   [`ParticleSystem`](ParticleSystem.md#bpy.types.ParticleSystem "bpy.types.ParticleSystem")

    proximity\_falloff[#](#bpy.types.DynamicPaintBrushSettings.proximity_falloff "Link to this definition")
    :   Proximity falloff type

        Type:
        :   enum in [‘SMOOTH’, ‘CONSTANT’, ‘RAMP’], default ‘CONSTANT’

    ray\_direction[#](#bpy.types.DynamicPaintBrushSettings.ray_direction "Link to this definition")
    :   Ray direction to use for projection (if brush object is located in that direction it’s painted)

        Type:
        :   enum in [‘CANVAS’, ‘BRUSH’, ‘Z\_AXIS’], default ‘CANVAS’

    smooth\_radius[#](#bpy.types.DynamicPaintBrushSettings.smooth_radius "Link to this definition")
    :   Smooth falloff added after solid radius

        Type:
        :   float in [0, 10], default 0.0

    smudge\_strength[#](#bpy.types.DynamicPaintBrushSettings.smudge_strength "Link to this definition")
    :   Smudge effect strength

        Type:
        :   float in [0, 1], default 0.0

    solid\_radius[#](#bpy.types.DynamicPaintBrushSettings.solid_radius "Link to this definition")
    :   Radius that will be painted solid

        Type:
        :   float in [0.01, 10], default 0.0

    use\_absolute\_alpha[#](#bpy.types.DynamicPaintBrushSettings.use_absolute_alpha "Link to this definition")
    :   Only increase alpha value if paint alpha is higher than existing

        Type:
        :   boolean, default False

    use\_negative\_volume[#](#bpy.types.DynamicPaintBrushSettings.use_negative_volume "Link to this definition")
    :   Negate influence inside the volume

        Type:
        :   boolean, default False

    use\_paint\_erase[#](#bpy.types.DynamicPaintBrushSettings.use_paint_erase "Link to this definition")
    :   Erase / remove paint instead of adding it

        Type:
        :   boolean, default False

    use\_particle\_radius[#](#bpy.types.DynamicPaintBrushSettings.use_particle_radius "Link to this definition")
    :   Use radius from particle settings

        Type:
        :   boolean, default False

    use\_proximity\_project[#](#bpy.types.DynamicPaintBrushSettings.use_proximity_project "Link to this definition")
    :   Brush is projected to canvas from defined direction within brush proximity

        Type:
        :   boolean, default False

    use\_proximity\_ramp\_alpha[#](#bpy.types.DynamicPaintBrushSettings.use_proximity_ramp_alpha "Link to this definition")
    :   Only read color ramp alpha

        Type:
        :   boolean, default False

    use\_smudge[#](#bpy.types.DynamicPaintBrushSettings.use_smudge "Link to this definition")
    :   Make this brush to smudge existing paint as it moves

        Type:
        :   boolean, default False

    use\_velocity\_alpha[#](#bpy.types.DynamicPaintBrushSettings.use_velocity_alpha "Link to this definition")
    :   Multiply brush influence by velocity color ramp alpha

        Type:
        :   boolean, default False

    use\_velocity\_color[#](#bpy.types.DynamicPaintBrushSettings.use_velocity_color "Link to this definition")
    :   Replace brush color by velocity color ramp

        Type:
        :   boolean, default False

    use\_velocity\_depth[#](#bpy.types.DynamicPaintBrushSettings.use_velocity_depth "Link to this definition")
    :   Multiply brush intersection depth (displace, waves) by velocity ramp alpha

        Type:
        :   boolean, default False

    velocity\_max[#](#bpy.types.DynamicPaintBrushSettings.velocity_max "Link to this definition")
    :   Velocity considered as maximum influence (Blender units per frame)

        Type:
        :   float in [0.0001, 10], default 0.0

    velocity\_ramp[#](#bpy.types.DynamicPaintBrushSettings.velocity_ramp "Link to this definition")
    :   Color ramp used to define brush velocity effect

        Type:
        :   [`ColorRamp`](ColorRamp.md#bpy.types.ColorRamp "bpy.types.ColorRamp"), (readonly)

    wave\_clamp[#](#bpy.types.DynamicPaintBrushSettings.wave_clamp "Link to this definition")
    :   Maximum level of surface intersection used to influence waves (use 0.0 to disable)

        Type:
        :   float in [0, 50], default 0.0

    wave\_factor[#](#bpy.types.DynamicPaintBrushSettings.wave_factor "Link to this definition")
    :   Multiplier for wave influence of this brush

        Type:
        :   float in [-2, 2], default 0.0

    wave\_type[#](#bpy.types.DynamicPaintBrushSettings.wave_type "Link to this definition")
    :   Type:
        :   enum in [‘CHANGE’, ‘DEPTH’, ‘FORCE’, ‘REFLECT’], default ‘DEPTH’

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*)[#](#bpy.types.DynamicPaintBrushSettings.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*string*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

    *classmethod* bl\_rna\_get\_subclass\_py(*id*, *default=None*)[#](#bpy.types.DynamicPaintBrushSettings.bl_rna_get_subclass_py "Link to this definition")
    :   Parameters:

        :   The class or default when not found.
