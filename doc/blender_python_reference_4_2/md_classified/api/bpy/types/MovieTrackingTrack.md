# MovieTrackingTrack(bpy\_struct)[#](#movietrackingtrack-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.MovieTrackingTrack(*bpy\_struct*)[#](#bpy.types.MovieTrackingTrack "Link to this definition")
:   Match-moving track data for tracking

    average\_error[#](#bpy.types.MovieTrackingTrack.average_error "Link to this definition")
    :   Average error of re-projection

        Type:
        :   float in [-inf, inf], default 0.0, (readonly)

    bundle[#](#bpy.types.MovieTrackingTrack.bundle "Link to this definition")
    :   Position of bundle reconstructed from this track

        Type:
        :   [`mathutils.Vector`](../../mathutils/index.md#mathutils.Vector "mathutils.Vector") of 3 items in [-inf, inf], default (0.0, 0.0, 0.0), (readonly)

    color[#](#bpy.types.MovieTrackingTrack.color "Link to this definition")
    :   Color of the track in the Movie Clip Editor and the 3D viewport after a solve

        Type:
        :   [`mathutils.Color`](../../mathutils/index.md#mathutils.Color "mathutils.Color") of 3 items in [0, 1], default (0.0, 0.0, 0.0)

    correlation\_min[#](#bpy.types.MovieTrackingTrack.correlation_min "Link to this definition")
    :   Minimal value of correlation between matched pattern and reference that is still treated as successful tracking

        Type:
        :   float in [0, 1], default 0.0

    frames\_limit[#](#bpy.types.MovieTrackingTrack.frames_limit "Link to this definition")
    :   Every tracking cycle, this number of frames are tracked

        Type:
        :   int in [0, 32767], default 0

    grease\_pencil[#](#bpy.types.MovieTrackingTrack.grease_pencil "Link to this definition")
    :   Grease pencil data for this track

        Type:
        :   [`GreasePencil`](GreasePencil.md#bpy.types.GreasePencil "bpy.types.GreasePencil")

    has\_bundle[#](#bpy.types.MovieTrackingTrack.has_bundle "Link to this definition")
    :   True if track has a valid bundle

        Type:
        :   boolean, default False, (readonly)

    hide[#](#bpy.types.MovieTrackingTrack.hide "Link to this definition")
    :   Track is hidden

        Type:
        :   boolean, default False

    lock[#](#bpy.types.MovieTrackingTrack.lock "Link to this definition")
    :   Track is locked and all changes to it are disabled

        Type:
        :   boolean, default False

    margin[#](#bpy.types.MovieTrackingTrack.margin "Link to this definition")
    :   Distance from image boundary at which marker stops tracking

        Type:
        :   int in [0, 300], default 0

    markers[#](#bpy.types.MovieTrackingTrack.markers "Link to this definition")
    :   Collection of markers in track

        Type:
        :   [`MovieTrackingMarkers`](MovieTrackingMarkers.md#bpy.types.MovieTrackingMarkers "bpy.types.MovieTrackingMarkers") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`MovieTrackingMarker`](MovieTrackingMarker.md#bpy.types.MovieTrackingMarker "bpy.types.MovieTrackingMarker"), (readonly)

    motion\_model[#](#bpy.types.MovieTrackingTrack.motion_model "Link to this definition")
    :   Default motion model to use for tracking

        * `Perspective`
          Perspective – Search for markers that are perspectively deformed (homography) between frames.
        * `Affine`
          Affine – Search for markers that are affine-deformed (t, r, k, and skew) between frames.
        * `LocRotScale`
          Location, Rotation & Scale – Search for markers that are translated, rotated, and scaled between frames.
        * `LocScale`
          Location & Scale – Search for markers that are translated and scaled between frames.
        * `LocRot`
          Location & Rotation – Search for markers that are translated and rotated between frames.
        * `Loc`
          Location – Search for markers that are translated between frames.

        Type:
        :   enum in [‘Perspective’, ‘Affine’, ‘LocRotScale’, ‘LocScale’, ‘LocRot’, ‘Loc’], default ‘Loc’

    name[#](#bpy.types.MovieTrackingTrack.name "Link to this definition")
    :   Unique name of track

        Type:
        :   string, default “”, (never None)

    offset[#](#bpy.types.MovieTrackingTrack.offset "Link to this definition")
    :   Offset of track from the parenting point

        Type:
        :   [`mathutils.Vector`](../../mathutils/index.md#mathutils.Vector "mathutils.Vector") of 2 items in [-inf, inf], default (0.0, 0.0)

    pattern\_match[#](#bpy.types.MovieTrackingTrack.pattern_match "Link to this definition")
    :   Track pattern from given frame when tracking marker to next frame

        * `KEYFRAME`
          Keyframe – Track pattern from keyframe to next frame.
        * `PREV_FRAME`
          Previous frame – Track pattern from current frame to next frame.

        Type:
        :   enum in [‘KEYFRAME’, ‘PREV\_FRAME’], default ‘KEYFRAME’

    select[#](#bpy.types.MovieTrackingTrack.select "Link to this definition")
    :   Track is selected

        Type:
        :   boolean, default False

    select\_anchor[#](#bpy.types.MovieTrackingTrack.select_anchor "Link to this definition")
    :   Track’s anchor point is selected

        Type:
        :   boolean, default False

    select\_pattern[#](#bpy.types.MovieTrackingTrack.select_pattern "Link to this definition")
    :   Track’s pattern area is selected

        Type:
        :   boolean, default False

    select\_search[#](#bpy.types.MovieTrackingTrack.select_search "Link to this definition")
    :   Track’s search area is selected

        Type:
        :   boolean, default False

    use\_alpha\_preview[#](#bpy.types.MovieTrackingTrack.use_alpha_preview "Link to this definition")
    :   Apply track’s mask on displaying preview

        Type:
        :   boolean, default False

    use\_blue\_channel[#](#bpy.types.MovieTrackingTrack.use_blue_channel "Link to this definition")
    :   Use blue channel from footage for tracking

        Type:
        :   boolean, default False

    use\_brute[#](#bpy.types.MovieTrackingTrack.use_brute "Link to this definition")
    :   Use a brute-force translation only pre-track before refinement

        Type:
        :   boolean, default False

    use\_custom\_color[#](#bpy.types.MovieTrackingTrack.use_custom_color "Link to this definition")
    :   Use custom color instead of theme-defined

        Type:
        :   boolean, default False

    use\_grayscale\_preview[#](#bpy.types.MovieTrackingTrack.use_grayscale_preview "Link to this definition")
    :   Display what the tracking algorithm sees in the preview

        Type:
        :   boolean, default False

    use\_green\_channel[#](#bpy.types.MovieTrackingTrack.use_green_channel "Link to this definition")
    :   Use green channel from footage for tracking

        Type:
        :   boolean, default False

    use\_mask[#](#bpy.types.MovieTrackingTrack.use_mask "Link to this definition")
    :   Use a grease pencil data-block as a mask to use only specified areas of pattern when tracking

        Type:
        :   boolean, default False

    use\_normalization[#](#bpy.types.MovieTrackingTrack.use_normalization "Link to this definition")
    :   Normalize light intensities while tracking. Slower

        Type:
        :   boolean, default False

    use\_red\_channel[#](#bpy.types.MovieTrackingTrack.use_red_channel "Link to this definition")
    :   Use red channel from footage for tracking

        Type:
        :   boolean, default False

    weight[#](#bpy.types.MovieTrackingTrack.weight "Link to this definition")
    :   Influence of this track on a final solution

        Type:
        :   float in [0, 1], default 0.0

    weight\_stab[#](#bpy.types.MovieTrackingTrack.weight_stab "Link to this definition")
    :   Influence of this track on 2D stabilization

        Type:
        :   float in [0, 1], default 0.0
