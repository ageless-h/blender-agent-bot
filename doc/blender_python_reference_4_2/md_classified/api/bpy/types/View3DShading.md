# View3DShading(bpy\_struct)[#](#view3dshading-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.View3DShading(*bpy\_struct*)[#](#bpy.types.View3DShading "Link to this definition")
:   Settings for shading in the 3D viewport

    aov\_name[#](#bpy.types.View3DShading.aov_name "Link to this definition")
    :   Name of the active Shader AOV

        Type:
        :   string, default “”, (never None)

    background\_color[#](#bpy.types.View3DShading.background_color "Link to this definition")
    :   Color for custom background color

        Type:
        :   [`mathutils.Color`](../../mathutils/index.md#mathutils.Color "mathutils.Color") of 3 items in [0, 1], default (0.05, 0.05, 0.05)

    background\_type[#](#bpy.types.View3DShading.background_type "Link to this definition")
    :   Way to display the background

        * `THEME`
          Theme – Use the theme for background color.
        * `WORLD`
          World – Use the world for background color.
        * `VIEWPORT`
          Viewport – Use a custom color limited to this viewport only.

        Type:
        :   enum in [‘THEME’, ‘WORLD’, ‘VIEWPORT’], default ‘THEME’

    cavity\_ridge\_factor[#](#bpy.types.View3DShading.cavity_ridge_factor "Link to this definition")
    :   Factor for the cavity ridges

        Type:
        :   float in [0, 250], default 1.0

    cavity\_type[#](#bpy.types.View3DShading.cavity_type "Link to this definition")
    :   Way to display the cavity shading

        * `WORLD`
          World – Cavity shading computed in world space, useful for larger-scale occlusion.
        * `SCREEN`
          Screen – Curvature-based shading, useful for making fine details more visible.
        * `BOTH`
          Both – Use both effects simultaneously.

        Type:
        :   enum in [‘WORLD’, ‘SCREEN’, ‘BOTH’], default ‘SCREEN’

    cavity\_valley\_factor[#](#bpy.types.View3DShading.cavity_valley_factor "Link to this definition")
    :   Factor for the cavity valleys

        Type:
        :   float in [0, 250], default 1.0

    color\_type[#](#bpy.types.View3DShading.color_type "Link to this definition")
    :   Color Type

        * `MATERIAL`
          Material – Show material color.
        * `SINGLE`
          Single – Show scene in a single color.
        * `OBJECT`
          Object – Show object color.
        * `RANDOM`
          Random – Show random object color.
        * `VERTEX`
          Attribute – Show active color attribute.
        * `TEXTURE`
          Texture – Show the texture from the active image texture node using the active UV map coordinates.

        Type:
        :   enum in [‘MATERIAL’, ‘SINGLE’, ‘OBJECT’, ‘RANDOM’, ‘VERTEX’, ‘TEXTURE’], default ‘MATERIAL’

    curvature\_ridge\_factor[#](#bpy.types.View3DShading.curvature_ridge_factor "Link to this definition")
    :   Factor for the curvature ridges

        Type:
        :   float in [0, 2], default 1.0

    curvature\_valley\_factor[#](#bpy.types.View3DShading.curvature_valley_factor "Link to this definition")
    :   Factor for the curvature valleys

        Type:
        :   float in [0, 2], default 1.0

    cycles[#](#bpy.types.View3DShading.cycles "Link to this definition")
    :   Type:
        :   `CyclesView3DShadingSettings`, (readonly)

    light[#](#bpy.types.View3DShading.light "Link to this definition")
    :   Lighting Method for Solid/Texture Viewport Shading

        * `STUDIO`
          Studio – Display using studio lighting.
        * `MATCAP`
          MatCap – Display using matcap material and lighting.
        * `FLAT`
          Flat – Display using flat lighting.

        Type:
        :   enum in [‘STUDIO’, ‘MATCAP’, ‘FLAT’], default ‘STUDIO’

    object\_outline\_color[#](#bpy.types.View3DShading.object_outline_color "Link to this definition")
    :   Color for object outline

        Type:
        :   [`mathutils.Color`](../../mathutils/index.md#mathutils.Color "mathutils.Color") of 3 items in [0, 1], default (0.0, 0.0, 0.0)

    render\_pass[#](#bpy.types.View3DShading.render_pass "Link to this definition")
    :   Render Pass to show in the viewport

        Type:
        :   enum in [‘COMBINED’, ‘EMISSION’, ‘ENVIRONMENT’, ‘AO’, ‘SHADOW’, ‘TRANSPARENT’, ‘DIFFUSE\_LIGHT’, ‘DIFFUSE\_COLOR’, ‘SPECULAR\_LIGHT’, ‘SPECULAR\_COLOR’, ‘VOLUME\_LIGHT’, ‘NORMAL’, ‘MIST’, ‘CryptoObject’, ‘CryptoAsset’, ‘CryptoMaterial’, ‘AOV’], default ‘COMBINED’

    selected\_studio\_light[#](#bpy.types.View3DShading.selected_studio_light "Link to this definition")
    :   Selected StudioLight

        Type:
        :   [`StudioLight`](StudioLight.md#bpy.types.StudioLight "bpy.types.StudioLight"), (readonly)

    shadow\_intensity[#](#bpy.types.View3DShading.shadow_intensity "Link to this definition")
    :   Darkness of shadows

        Type:
        :   float in [0, 1], default 0.5

    show\_backface\_culling[#](#bpy.types.View3DShading.show_backface_culling "Link to this definition")
    :   Use back face culling to hide the back side of faces

        Type:
        :   boolean, default False

    show\_cavity[#](#bpy.types.View3DShading.show_cavity "Link to this definition")
    :   Show Cavity

        Type:
        :   boolean, default False

    show\_object\_outline[#](#bpy.types.View3DShading.show_object_outline "Link to this definition")
    :   Show Object Outline

        Type:
        :   boolean, default False

    show\_shadows[#](#bpy.types.View3DShading.show_shadows "Link to this definition")
    :   Show Shadow

        Type:
        :   boolean, default False

    show\_specular\_highlight[#](#bpy.types.View3DShading.show_specular_highlight "Link to this definition")
    :   Render specular highlights

        Type:
        :   boolean, default True

    show\_xray[#](#bpy.types.View3DShading.show_xray "Link to this definition")
    :   Show whole scene transparent

        Type:
        :   boolean, default False

    show\_xray\_wireframe[#](#bpy.types.View3DShading.show_xray_wireframe "Link to this definition")
    :   Show whole scene transparent

        Type:
        :   boolean, default True

    single\_color[#](#bpy.types.View3DShading.single_color "Link to this definition")
    :   Color for single color mode

        Type:
        :   [`mathutils.Color`](../../mathutils/index.md#mathutils.Color "mathutils.Color") of 3 items in [0, 1], default (0.8, 0.8, 0.8)

    studio\_light[#](#bpy.types.View3DShading.studio_light "Link to this definition")
    :   Studio lighting setup

        Type:
        :   enum in [‘DEFAULT’], default ‘DEFAULT’

    studiolight\_background\_alpha[#](#bpy.types.View3DShading.studiolight_background_alpha "Link to this definition")
    :   Show the studiolight in the background

        Type:
        :   float in [0, 1], default 0.0

    studiolight\_background\_blur[#](#bpy.types.View3DShading.studiolight_background_blur "Link to this definition")
    :   Blur the studiolight in the background

        Type:
        :   float in [0, 1], default 0.5

    studiolight\_intensity[#](#bpy.types.View3DShading.studiolight_intensity "Link to this definition")
    :   Strength of the studiolight

        Type:
        :   float in [0, inf], default 1.0

    studiolight\_rotate\_z[#](#bpy.types.View3DShading.studiolight_rotate_z "Link to this definition")
    :   Rotation of the studiolight around the Z-Axis

        Type:
        :   float in [-3.14159, 3.14159], default 0.0

    type[#](#bpy.types.View3DShading.type "Link to this definition")
    :   Method to display/shade objects in the 3D View

        Type:
        :   enum in [Shading Type Items](../../../appendix/bpy_types_enum_items/shading_type_items.md#rna-enum-shading-type-items), default ‘SOLID’

    use\_compositor[#](#bpy.types.View3DShading.use_compositor "Link to this definition")
    :   When to preview the compositor output inside the viewport

        * `DISABLED`
          Disabled – The compositor is disabled.
        * `CAMERA`
          Camera – The compositor is enabled only in camera view.
        * `ALWAYS`
          Always – The compositor is always enabled regardless of the view.

        Type:
        :   enum in [‘DISABLED’, ‘CAMERA’, ‘ALWAYS’], default ‘DISABLED’

    use\_dof[#](#bpy.types.View3DShading.use_dof "Link to this definition")
    :   Use depth of field on viewport using the values from the active camera

        Type:
        :   boolean, default False

    use\_scene\_lights[#](#bpy.types.View3DShading.use_scene_lights "Link to this definition")
    :   Render lights and light probes of the scene

        Type:
        :   boolean, default False

    use\_scene\_lights\_render[#](#bpy.types.View3DShading.use_scene_lights_render "Link to this definition")
    :   Render lights and light probes of the scene

        Type:
        :   boolean, default True

    use\_scene\_world[#](#bpy.types.View3DShading.use_scene_world "Link to this definition")
    :   Use scene world for lighting

        Type:
        :   boolean, default False

    use\_scene\_world\_render[#](#bpy.types.View3DShading.use_scene_world_render "Link to this definition")
    :   Use scene world for lighting

        Type:
        :   boolean, default True

    use\_studiolight\_view\_rotation[#](#bpy.types.View3DShading.use_studiolight_view_rotation "Link to this definition")
    :   Make the HDR rotation fixed and not follow the camera

        Type:
        :   boolean, default False

    use\_world\_space\_lighting[#](#bpy.types.View3DShading.use_world_space_lighting "Link to this definition")
    :   Make the lighting fixed and not follow the camera

        Type:
        :   boolean, default False

    wireframe\_color\_type[#](#bpy.types.View3DShading.wireframe_color_type "Link to this definition")
    :   Wire Color Type

        * `THEME`
          Theme – Show scene wireframes with the theme’s wire color.
        * `OBJECT`
          Object – Show object color on wireframe.
        * `RANDOM`
          Random – Show random object color on wireframe.

        Type:
        :   enum in [‘THEME’, ‘OBJECT’, ‘RANDOM’], default ‘THEME’

    xray\_alpha[#](#bpy.types.View3DShading.xray_alpha "Link to this definition")
    :   Amount of alpha to use

        Type:
        :   float in [0, 1], default 0.5

    xray\_alpha\_wireframe[#](#bpy.types.View3DShading.xray_alpha_wireframe "Link to this definition")
    :   Amount of alpha to use

        Type:
        :   float in [0, 1], default 0.5
