# PreferencesSystem(bpy\_struct)[#](#preferencessystem-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.PreferencesSystem(*bpy\_struct*)[#](#bpy.types.PreferencesSystem "Link to this definition")
:   Graphics driver and operating system settings

    anisotropic\_filter[#](#bpy.types.PreferencesSystem.anisotropic_filter "Link to this definition")
    :   Quality of anisotropic filtering

        Type:
        :   enum in [‘FILTER\_0’, ‘FILTER\_2’, ‘FILTER\_4’, ‘FILTER\_8’, ‘FILTER\_16’], default ‘FILTER\_0’

    audio\_channels[#](#bpy.types.PreferencesSystem.audio_channels "Link to this definition")
    :   Audio channel count

        * `MONO`
          Mono – Set audio channels to mono.
        * `STEREO`
          Stereo – Set audio channels to stereo.
        * `SURROUND4`
          4 Channels – Set audio channels to 4 channels.
        * `SURROUND51`
          5.1 Surround – Set audio channels to 5.1 surround sound.
        * `SURROUND71`
          7.1 Surround – Set audio channels to 7.1 surround sound.

        Type:
        :   enum in [‘MONO’, ‘STEREO’, ‘SURROUND4’, ‘SURROUND51’, ‘SURROUND71’], default ‘STEREO’

    audio\_device[#](#bpy.types.PreferencesSystem.audio_device "Link to this definition")
    :   Audio output device

        * `None`
          None – No device - there will be no audio output.

        Type:
        :   enum in [‘None’], default ‘None’

    audio\_mixing\_buffer[#](#bpy.types.PreferencesSystem.audio_mixing_buffer "Link to this definition")
    :   Number of samples used by the audio mixing buffer

        * `SAMPLES_256`
          256 Samples – Set audio mixing buffer size to 256 samples.
        * `SAMPLES_512`
          512 Samples – Set audio mixing buffer size to 512 samples.
        * `SAMPLES_1024`
          1024 Samples – Set audio mixing buffer size to 1024 samples.
        * `SAMPLES_2048`
          2048 Samples – Set audio mixing buffer size to 2048 samples.
        * `SAMPLES_4096`
          4096 Samples – Set audio mixing buffer size to 4096 samples.
        * `SAMPLES_8192`
          8192 Samples – Set audio mixing buffer size to 8192 samples.
        * `SAMPLES_16384`
          16384 Samples – Set audio mixing buffer size to 16384 samples.
        * `SAMPLES_32768`
          32768 Samples – Set audio mixing buffer size to 32768 samples.

        Type:
        :   enum in [‘SAMPLES\_256’, ‘SAMPLES\_512’, ‘SAMPLES\_1024’, ‘SAMPLES\_2048’, ‘SAMPLES\_4096’, ‘SAMPLES\_8192’, ‘SAMPLES\_16384’, ‘SAMPLES\_32768’], default ‘SAMPLES\_2048’

    audio\_sample\_format[#](#bpy.types.PreferencesSystem.audio_sample_format "Link to this definition")
    :   Audio sample format

        * `U8`
          8-bit Unsigned – Set audio sample format to 8-bit unsigned integer.
        * `S16`
          16-bit Signed – Set audio sample format to 16-bit signed integer.
        * `S24`
          24-bit Signed – Set audio sample format to 24-bit signed integer.
        * `S32`
          32-bit Signed – Set audio sample format to 32-bit signed integer.
        * `FLOAT`
          32-bit Float – Set audio sample format to 32-bit float.
        * `DOUBLE`
          64-bit Float – Set audio sample format to 64-bit float.

        Type:
        :   enum in [‘U8’, ‘S16’, ‘S24’, ‘S32’, ‘FLOAT’, ‘DOUBLE’], default ‘FLOAT’

    audio\_sample\_rate[#](#bpy.types.PreferencesSystem.audio_sample_rate "Link to this definition")
    :   Audio sample rate

        * `RATE_44100`
          44.1 kHz – Set audio sampling rate to 44100 samples per second.
        * `RATE_48000`
          48 kHz – Set audio sampling rate to 48000 samples per second.
        * `RATE_96000`
          96 kHz – Set audio sampling rate to 96000 samples per second.
        * `RATE_192000`
          192 kHz – Set audio sampling rate to 192000 samples per second.

        Type:
        :   enum in [‘RATE\_44100’, ‘RATE\_48000’, ‘RATE\_96000’, ‘RATE\_192000’], default ‘RATE\_48000’

    dpi[#](#bpy.types.PreferencesSystem.dpi "Link to this definition")
    :   Type:
        :   int in [-inf, inf], default 0, (readonly)

    gl\_clip\_alpha[#](#bpy.types.PreferencesSystem.gl_clip_alpha "Link to this definition")
    :   Clip alpha below this threshold in the 3D textured view

        Type:
        :   float in [0, 1], default 0.004

    gl\_texture\_limit[#](#bpy.types.PreferencesSystem.gl_texture_limit "Link to this definition")
    :   Limit the texture size to save graphics memory

        Type:
        :   enum in [‘CLAMP\_OFF’, ‘CLAMP\_8192’, ‘CLAMP\_4096’, ‘CLAMP\_2048’, ‘CLAMP\_1024’, ‘CLAMP\_512’, ‘CLAMP\_256’, ‘CLAMP\_128’], default ‘CLAMP\_OFF’

    gpu\_backend[#](#bpy.types.PreferencesSystem.gpu_backend "Link to this definition")
    :   GPU backend to use (requires restarting Blender for changes to take effect)

        * `OPENGL`
          OpenGL – Use OpenGL backend.
        * `METAL`
          Metal – Use Metal backend.
        * `VULKAN`
          Vulkan – Use Vulkan backend.

        Type:
        :   enum in [‘OPENGL’, ‘METAL’, ‘VULKAN’], default ‘OPENGL’

    image\_draw\_method[#](#bpy.types.PreferencesSystem.image_draw_method "Link to this definition")
    :   Method used for displaying images on the screen

        * `AUTO`
          Automatic – Automatically choose method based on GPU and image.
        * `2DTEXTURE`
          2D Texture – Use CPU for display transform and display image with 2D texture.
        * `GLSL`
          GLSL – Use GLSL shaders for display transform and display image with 2D texture.

        Type:
        :   enum in [‘AUTO’, ‘2DTEXTURE’, ‘GLSL’], default ‘AUTO’

    is\_microsoft\_store\_install[#](#bpy.types.PreferencesSystem.is_microsoft_store_install "Link to this definition")
    :   Whether this blender installation is a sandboxed Microsoft Store version

        Type:
        :   boolean, default False, (readonly)

    legacy\_compute\_device\_type[#](#bpy.types.PreferencesSystem.legacy_compute_device_type "Link to this definition")
    :   For backwards compatibility only

        Type:
        :   int in [-inf, inf], default 0, (readonly)

    light\_ambient[#](#bpy.types.PreferencesSystem.light_ambient "Link to this definition")
    :   Color of the ambient light that uniformly lit the scene

        Type:
        :   [`mathutils.Color`](../../mathutils/index.md#mathutils.Color "mathutils.Color") of 3 items in [0, inf], default (0.0, 0.0, 0.0)

    max\_shader\_compilation\_subprocesses[#](#bpy.types.PreferencesSystem.max_shader_compilation_subprocesses "Link to this definition")
    :   Max number of parallel shader compilation subprocesses, clamped at the max threads supported by the CPU (requires restarting Blender for changes to take effect). Setting it to 0 disables subprocess shader compilation

        Type:
        :   int in [0, 32767], default 0

    memory\_cache\_limit[#](#bpy.types.PreferencesSystem.memory_cache_limit "Link to this definition")
    :   Memory cache limit (in megabytes)

        Type:
        :   int in [0, inf], default 4096

    network\_connection\_limit[#](#bpy.types.PreferencesSystem.network_connection_limit "Link to this definition")
    :   Limit the number of simultaneous internet connections online operations may make at once. Zero disables the limit

        Type:
        :   int in [0, 255], default 0

    network\_timeout[#](#bpy.types.PreferencesSystem.network_timeout "Link to this definition")
    :   The time in seconds to wait for online operations before a connection may fail with a time-out error. Zero uses the systems default

        Type:
        :   int in [0, 255], default 0

    pixel\_size[#](#bpy.types.PreferencesSystem.pixel_size "Link to this definition")
    :   Type:
        :   float in [-inf, inf], default 1.0, (readonly)

    register\_all\_users[#](#bpy.types.PreferencesSystem.register_all_users "Link to this definition")
    :   Make this Blender version open blend files for all users. Requires elevated privileges

        Type:
        :   boolean, default False

    scrollback[#](#bpy.types.PreferencesSystem.scrollback "Link to this definition")
    :   Maximum number of lines to store for the console buffer

        Type:
        :   int in [32, 32768], default 256

    sequencer\_disk\_cache\_compression[#](#bpy.types.PreferencesSystem.sequencer_disk_cache_compression "Link to this definition")
    :   Smaller compression will result in larger files, but less decoding overhead

        * `NONE`
          None – Requires fast storage, but uses minimum CPU resources.
        * `LOW`
          Low – Doesn’t require fast storage and uses less CPU resources.
        * `HIGH`
          High – Works on slower storage devices and uses most CPU resources.

        Type:
        :   enum in [‘NONE’, ‘LOW’, ‘HIGH’], default ‘NONE’

    sequencer\_disk\_cache\_dir[#](#bpy.types.PreferencesSystem.sequencer_disk_cache_dir "Link to this definition")
    :   Override default directory

        Type:
        :   string, default “”, (never None)

    sequencer\_disk\_cache\_size\_limit[#](#bpy.types.PreferencesSystem.sequencer_disk_cache_size_limit "Link to this definition")
    :   Disk cache limit (in gigabytes)

        Type:
        :   int in [0, inf], default 100

    sequencer\_proxy\_setup[#](#bpy.types.PreferencesSystem.sequencer_proxy_setup "Link to this definition")
    :   When and how proxies are created

        * `MANUAL`
          Manual – Set up proxies manually.
        * `AUTOMATIC`
          Automatic – Build proxies for added movie and image strips in each preview size.

        Type:
        :   enum in [‘MANUAL’, ‘AUTOMATIC’], default ‘AUTOMATIC’

    solid\_lights[#](#bpy.types.PreferencesSystem.solid_lights "Link to this definition")
    :   Lights used to display objects in solid shading mode

        Type:
        :   [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`UserSolidLight`](UserSolidLight.md#bpy.types.UserSolidLight "bpy.types.UserSolidLight"), (readonly)

    texture\_collection\_rate[#](#bpy.types.PreferencesSystem.texture_collection_rate "Link to this definition")
    :   Number of seconds between each run of the GL texture garbage collector

        Type:
        :   int in [1, 3600], default 60

    texture\_time\_out[#](#bpy.types.PreferencesSystem.texture_time_out "Link to this definition")
    :   Time since last access of a GL texture in seconds after which it is freed (set to 0 to keep textures allocated)

        Type:
        :   int in [0, 3600], default 120

    ui\_line\_width[#](#bpy.types.PreferencesSystem.ui_line_width "Link to this definition")
    :   Suggested line thickness and point size in pixels, for add-ons displaying custom user interface elements, based on operating system settings and Blender UI scale

        Type:
        :   float in [-inf, inf], default 1.0, (readonly)

    ui\_scale[#](#bpy.types.PreferencesSystem.ui_scale "Link to this definition")
    :   Size multiplier to use when displaying custom user interface elements, so that they are scaled correctly on screens with different DPI. This value is based on operating system DPI settings and Blender display scale

        Type:
        :   float in [-inf, inf], default 0.0, (readonly)

    use\_edit\_mode\_smooth\_wire[#](#bpy.types.PreferencesSystem.use_edit_mode_smooth_wire "Link to this definition")
    :   Enable edit mode edge smoothing, reducing aliasing (requires restart)

        Type:
        :   boolean, default False

    use\_gpu\_subdivision[#](#bpy.types.PreferencesSystem.use_gpu_subdivision "Link to this definition")
    :   Enable GPU acceleration for evaluating the last subdivision surface modifiers in the stack

        Type:
        :   boolean, default True

    use\_online\_access[#](#bpy.types.PreferencesSystem.use_online_access "Link to this definition")
    :   Allow Blender to access the internet. Add-ons that follow this setting will only connect to the internet if enabled. However, Blender cannot prevent third-party add-ons from violating this rule

        Type:
        :   boolean, default False

    use\_overlay\_smooth\_wire[#](#bpy.types.PreferencesSystem.use_overlay_smooth_wire "Link to this definition")
    :   Enable overlay smooth wires, reducing aliasing

        Type:
        :   boolean, default True

    use\_region\_overlap[#](#bpy.types.PreferencesSystem.use_region_overlap "Link to this definition")
    :   Display tool/property regions over the main region

        Type:
        :   boolean, default True

    use\_select\_pick\_depth[#](#bpy.types.PreferencesSystem.use_select_pick_depth "Link to this definition")
    :   When making a selection in 3D View, use the GPU depth buffer to ensure the frontmost object is selected first

        Type:
        :   boolean, default False

    use\_sequencer\_disk\_cache[#](#bpy.types.PreferencesSystem.use_sequencer_disk_cache "Link to this definition")
    :   Store cached images to disk

        Type:
        :   boolean, default False

    use\_studio\_light\_edit[#](#bpy.types.PreferencesSystem.use_studio_light_edit "Link to this definition")
    :   View the result of the studio light editor in the viewport

        Type:
        :   boolean, default False

    vbo\_collection\_rate[#](#bpy.types.PreferencesSystem.vbo_collection_rate "Link to this definition")
    :   Number of seconds between each run of the GL vertex buffer object garbage collector

        Type:
        :   int in [1, 3600], default 60

    vbo\_time\_out[#](#bpy.types.PreferencesSystem.vbo_time_out "Link to this definition")
    :   Time since last access of a GL vertex buffer object in seconds after which it is freed (set to 0 to keep VBO allocated)

        Type:
        :   int in [0, 3600], default 120

    viewport\_aa[#](#bpy.types.PreferencesSystem.viewport_aa "Link to this definition")
    :   Method of anti-aliasing in 3d viewport

        * `OFF`
          No Anti-Aliasing – Scene will be rendering without any anti-aliasing.
        * `FXAA`
          Single Pass Anti-Aliasing – Scene will be rendered using a single pass anti-aliasing method (FXAA).
        * `5`
          5 Samples – Scene will be rendered using 5 anti-aliasing samples.
        * `8`
          8 Samples – Scene will be rendered using 8 anti-aliasing samples.
        * `11`
          11 Samples – Scene will be rendered using 11 anti-aliasing samples.
        * `16`
          16 Samples – Scene will be rendered using 16 anti-aliasing samples.
        * `32`
          32 Samples – Scene will be rendered using 32 anti-aliasing samples.

        Type:
        :   enum in [‘OFF’, ‘FXAA’, ‘5’, ‘8’, ‘11’, ‘16’, ‘32’], default ‘8’
