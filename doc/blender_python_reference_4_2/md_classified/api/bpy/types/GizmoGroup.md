# GizmoGroup(bpy\_struct)[#](#gizmogroup-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.GizmoGroup(*bpy\_struct*)[#](#bpy.types.GizmoGroup "Link to this definition")
:   Storage of an operator being executed, or registered after execution

    bl\_idname[#](#bpy.types.GizmoGroup.bl_idname "Link to this definition")
    :   Type:
        :   string, default “”, (never None)

    bl\_label[#](#bpy.types.GizmoGroup.bl_label "Link to this definition")
    :   Type:
        :   string, default “”, (never None)

    bl\_options[#](#bpy.types.GizmoGroup.bl_options "Link to this definition")
    :   Options for this operator type

        * `3D`
          3D – Use in 3D viewport.
        * `SCALE`
          Scale – Scale to respect zoom (otherwise zoom independent display size).
        * `DEPTH_3D`
          Depth 3D – Supports culled depth by other objects in the view.
        * `SELECT`
          Select – Supports selection.
        * `PERSISTENT`
          Persistent.
        * `SHOW_MODAL_ALL`
          Show Modal All – Show all while interacting, as well as this group when another is being interacted with.
        * `EXCLUDE_MODAL`
          Exclude Modal – Show all except this group while interacting.
        * `TOOL_INIT`
          Tool Init – Postpone running until tool operator run (when used with a tool).
        * `TOOL_FALLBACK_KEYMAP`
          Use fallback tools keymap – Add fallback tools keymap to this gizmo type.
        * `VR_REDRAWS`
          VR Redraws – The gizmos are made for use with virtual reality sessions and require special redraw management.

        Type:
        :   enum set in {‘3D’, ‘SCALE’, ‘DEPTH\_3D’, ‘SELECT’, ‘PERSISTENT’, ‘SHOW\_MODAL\_ALL’, ‘EXCLUDE\_MODAL’, ‘TOOL\_INIT’, ‘TOOL\_FALLBACK\_KEYMAP’, ‘VR\_REDRAWS’}, default {‘3D’}

    bl\_owner\_id[#](#bpy.types.GizmoGroup.bl_owner_id "Link to this definition")
    :   Type:
        :   string, default “”, (never None)

    bl\_region\_type[#](#bpy.types.GizmoGroup.bl_region_type "Link to this definition")
    :   The region where the panel is going to be used in

        Type:
        :   enum in [Region Type Items](../../../appendix/bpy_types_enum_items/region_type_items.md#rna-enum-region-type-items), default ‘WINDOW’

    bl\_space\_type[#](#bpy.types.GizmoGroup.bl_space_type "Link to this definition")
    :   The space where the panel is going to be used in

        Type:
        :   enum in [Space Type Items](../../../appendix/bpy_types_enum_items/space_type_items.md#rna-enum-space-type-items), default ‘EMPTY’

    gizmos[#](#bpy.types.GizmoGroup.gizmos "Link to this definition")
    :   List of gizmos in the Gizmo Map

        Type:
        :   [`Gizmos`](Gizmos.md#bpy.types.Gizmos "bpy.types.Gizmos") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`Gizmo`](Gizmo.md#bpy.types.Gizmo "bpy.types.Gizmo"), (readonly)

    name[#](#bpy.types.GizmoGroup.name "Link to this definition")
    :   Type:
        :   string, default “”, (readonly, never None)

    *classmethod* poll(*context*)[#](#bpy.types.GizmoGroup.poll "Link to this definition")

    *classmethod* setup\_keymap(*keyconfig*)[#](#bpy.types.GizmoGroup.setup_keymap "Link to this definition")
