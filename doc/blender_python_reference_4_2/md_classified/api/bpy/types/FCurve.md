# FCurve(bpy\_struct)[#](#fcurve-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.FCurve(*bpy\_struct*)[#](#bpy.types.FCurve "Link to this definition")
:   F-Curve defining values of a period of time

    array\_index[#](#bpy.types.FCurve.array_index "Link to this definition")
    :   Index to the specific property affected by F-Curve if applicable

        Type:
        :   int in [0, inf], default 0

    auto\_smoothing[#](#bpy.types.FCurve.auto_smoothing "Link to this definition")
    :   Algorithm used to compute automatic handles

        Type:
        :   enum in [Fcurve Auto Smoothing Items](../../../appendix/bpy_types_enum_items/fcurve_auto_smoothing_items.md#rna-enum-fcurve-auto-smoothing-items), default ‘NONE’

    color[#](#bpy.types.FCurve.color "Link to this definition")
    :   Color of the F-Curve in the Graph Editor

        Type:
        :   [`mathutils.Color`](../../mathutils/index.md#mathutils.Color "mathutils.Color") of 3 items in [0, 1], default (0.0, 0.0, 0.0)

    color\_mode[#](#bpy.types.FCurve.color_mode "Link to this definition")
    :   Method used to determine color of F-Curve in Graph Editor

        * `AUTO_RAINBOW`
          Auto Rainbow – Cycle through the rainbow, trying to give each curve a unique color.
        * `AUTO_RGB`
          Auto XYZ to RGB – Use axis colors for transform and color properties, and auto-rainbow for the rest.
        * `AUTO_YRGB`
          Auto WXYZ to YRGB – Use axis colors for XYZ parts of transform, and yellow for the ‘W’ channel.
        * `CUSTOM`
          User Defined – Use custom hand-picked color for F-Curve.

        Type:
        :   enum in [‘AUTO\_RAINBOW’, ‘AUTO\_RGB’, ‘AUTO\_YRGB’, ‘CUSTOM’], default ‘AUTO\_RAINBOW’

    data\_path[#](#bpy.types.FCurve.data_path "Link to this definition")
    :   RNA Path to property affected by F-Curve

        Type:
        :   string, default “”, (never None)

    driver[#](#bpy.types.FCurve.driver "Link to this definition")
    :   Channel Driver (only set for Driver F-Curves)

        Type:
        :   [`Driver`](Driver.md#bpy.types.Driver "bpy.types.Driver"), (readonly)

    extrapolation[#](#bpy.types.FCurve.extrapolation "Link to this definition")
    :   Method used for evaluating value of F-Curve outside first and last keyframes

        * `CONSTANT`
          Constant – Hold values of endpoint keyframes.
        * `LINEAR`
          Linear – Use slope of curve leading in/out of endpoint keyframes.

        Type:
        :   enum in [‘CONSTANT’, ‘LINEAR’], default ‘CONSTANT’

    group[#](#bpy.types.FCurve.group "Link to this definition")
    :   Action Group that this F-Curve belongs to

        Type:
        :   [`ActionGroup`](ActionGroup.md#bpy.types.ActionGroup "bpy.types.ActionGroup")

    hide[#](#bpy.types.FCurve.hide "Link to this definition")
    :   F-Curve and its keyframes are hidden in the Graph Editor graphs

        Type:
        :   boolean, default False

    is\_empty[#](#bpy.types.FCurve.is_empty "Link to this definition")
    :   True if the curve contributes no animation due to lack of keyframes or useful modifiers, and should be deleted

        Type:
        :   boolean, default False, (readonly)

    is\_valid[#](#bpy.types.FCurve.is_valid "Link to this definition")
    :   False when F-Curve could not be evaluated in past, so should be skipped when evaluating

        Type:
        :   boolean, default False

    keyframe\_points[#](#bpy.types.FCurve.keyframe_points "Link to this definition")
    :   User-editable keyframes

        Type:
        :   [`FCurveKeyframePoints`](FCurveKeyframePoints.md#bpy.types.FCurveKeyframePoints "bpy.types.FCurveKeyframePoints") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`Keyframe`](Keyframe.md#bpy.types.Keyframe "bpy.types.Keyframe"), (readonly)

    lock[#](#bpy.types.FCurve.lock "Link to this definition")
    :   F-Curve’s settings cannot be edited

        Type:
        :   boolean, default False

    modifiers[#](#bpy.types.FCurve.modifiers "Link to this definition")
    :   Modifiers affecting the shape of the F-Curve

        Type:
        :   [`FCurveModifiers`](FCurveModifiers.md#bpy.types.FCurveModifiers "bpy.types.FCurveModifiers") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`FModifier`](FModifier.md#bpy.types.FModifier "bpy.types.FModifier"), (readonly)

    mute[#](#bpy.types.FCurve.mute "Link to this definition")
    :   Disable F-Curve evaluation

        Type:
        :   boolean, default False

    sampled\_points[#](#bpy.types.FCurve.sampled_points "Link to this definition")
    :   Sampled animation data

        Type:
        :   [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`FCurveSample`](FCurveSample.md#bpy.types.FCurveSample "bpy.types.FCurveSample"), (readonly)

    select[#](#bpy.types.FCurve.select "Link to this definition")
    :   F-Curve is selected for editing

        Type:
        :   boolean, default False

    evaluate(*frame*)[#](#bpy.types.FCurve.evaluate "Link to this definition")
    :   Evaluate F-Curve

        Parameters:
        :   **frame** (*float in* *[**-inf**,* *inf**]*) – Frame, Evaluate F-Curve at given frame

        Returns:
        :   Value, Value of F-Curve specific frame

        Return type:
        :   float in [-inf, inf]

    update()[#](#bpy.types.FCurve.update "Link to this definition")
    :   Ensure keyframes are sorted in chronological order and handles are set correctly

    range()[#](#bpy.types.FCurve.range "Link to this definition")
    :   Get the time extents for F-Curve

        Returns:
        :   Range, Min/Max values

        Return type:
        :   [`mathutils.Vector`](../../mathutils/index.md#mathutils.Vector "mathutils.Vector") of 2 items in [-inf, inf]

    update\_autoflags(*data*)[#](#bpy.types.FCurve.update_autoflags "Link to this definition")
    :   Update FCurve flags set automatically from affected property (currently, integer/discrete flags set when the property is not a float)

        Parameters:
        :   **data** ([`AnyType`](AnyType.md#bpy.types.AnyType "bpy.types.AnyType"), (never None)) – Data, Data containing the property controlled by given FCurve

    convert\_to\_samples(*start*, *end*)[#](#bpy.types.FCurve.convert_to_samples "Link to this definition")
    :   Convert current FCurve from keyframes to sample points, if necessary

        Parameters:
        :   * **start** (*int in* *[**-1048574**,* *1048574**]*) – Start Frame
            * **end** (*int in* *[**-1048574**,* *1048574**]*) – End Frame

    convert\_to\_keyframes(*start*, *end*)[#](#bpy.types.FCurve.convert_to_keyframes "Link to this definition")
    :   Convert current FCurve from sample points to keyframes (linear interpolation), if necessary

        Parameters:
        :   * **start** (*int in* *[**-1048574**,* *1048574**]*) – Start Frame
            * **end** (*int in* *[**-1048574**,* *1048574**]*) – End Frame

    bake(*start*, *end*, *step=1.0*, *remove='IN\_RANGE'*)[#](#bpy.types.FCurve.bake "Link to this definition")
    :   Place keys at even intervals on the existing curve.

        Parameters:
        :   * **start** (*int in* *[**-1048574**,* *1048574**]*) – Start Frame, Frame at which to start baking
            * **end** (*int in* *[**-1048574**,* *1048574**]*) – End Frame, Frame at which to end baking (inclusive)
            * **step** (*float in* *[**0.01**,* *inf**]**,* *(**optional**,* *optional argument**)*) – Step, At which interval to add keys
            * **remove** (*enum in* *[**'NONE'**,* *'IN\_RANGE'**,* *'OUT\_RANGE'**,* *'ALL'**]**,* *(**optional**)*) –

              Remove Options, Choose which keys should be automatically removed by the bake

              + `NONE`
                None – Keep all keys.
              + `IN_RANGE`
                In Range – Remove all keys within the defined range.
              + `OUT_RANGE`
                Outside Range – Remove all keys outside the defined range.
              + `ALL`
                All – Remove all existing keys.

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*)[#](#bpy.types.FCurve.bl_rna_get_subclass "Link to this definition")

    :   Parameters:

        Returns:
        :   The class or default when not found.

        :   type
