*class* bpy.types.MeshCacheModifier(*Modifier*)[#](#bpy.types.MeshCacheModifier "Link to this definition")
:   Cache Mesh

    cache\_format[#](#bpy.types.MeshCacheModifier.cache_format "Link to this definition")
    :   Type:
        :   enum in [‘MDD’, ‘PC2’], default ‘MDD’

    deform\_mode[#](#bpy.types.MeshCacheModifier.deform_mode "Link to this definition")
    :   * `OVERWRITE`
          Overwrite – Replace vertex coordinates with cached values.
        * `INTEGRATE`
          Integrate – Integrate deformation from this modifier’s input with the mesh-cache coordinates (useful for shape keys).

        Type:
        :   enum in [‘OVERWRITE’, ‘INTEGRATE’], default ‘OVERWRITE’

    eval\_factor[#](#bpy.types.MeshCacheModifier.eval_factor "Link to this definition")
    :   Evaluation time in seconds

        Type:
        :   float in [0, 1], default 0.0

    eval\_frame[#](#bpy.types.MeshCacheModifier.eval_frame "Link to this definition")
    :   The frame to evaluate (starting at 0)

        Type:
        :   float in [0, 1.04857e+06], default 0.0

    eval\_time[#](#bpy.types.MeshCacheModifier.eval_time "Link to this definition")
    :   Evaluation time in seconds

        Type:
        :   float in [0, inf], default 0.0

    factor[#](#bpy.types.MeshCacheModifier.factor "Link to this definition")
    :   Influence of the deformation

        Type:
        :   float in [0, 1], default 1.0

    filepath[#](#bpy.types.MeshCacheModifier.filepath "Link to this definition")
    :   Path to external displacements file

        Type:
        :   string, default “”, (never None)

    flip\_axis[#](#bpy.types.MeshCacheModifier.flip_axis "Link to this definition")
    :   Type:
        :   enum set in [Axis Flag Xyz Items](../../../appendix/bpy_types_enum_items/axis_flag_xyz_items.md#rna-enum-axis-flag-xyz-items), default {‘X’}

    forward\_axis[#](#bpy.types.MeshCacheModifier.forward_axis "Link to this definition")
    :   Type:
        :   enum in [Object Axis Items](../../../appendix/bpy_types_enum_items/object_axis_items.md#rna-enum-object-axis-items), default ‘POS\_Y’

    frame\_scale[#](#bpy.types.MeshCacheModifier.frame_scale "Link to this definition")
    :   Evaluation time in seconds

        Type:
        :   float in [0, 100], default 1.0

    frame\_start[#](#bpy.types.MeshCacheModifier.frame_start "Link to this definition")
    :   Add this to the start frame

        Type:
        :   float in [-1.04857e+06, 1.04857e+06], default 0.0

    interpolation[#](#bpy.types.MeshCacheModifier.interpolation "Link to this definition")
    :   Type:
        :   enum in [‘NONE’, ‘LINEAR’], default ‘LINEAR’

    invert\_vertex\_group[#](#bpy.types.MeshCacheModifier.invert_vertex_group "Link to this definition")
    :   Invert vertex group influence

        Type:
        :   boolean, default False

    play\_mode[#](#bpy.types.MeshCacheModifier.play_mode "Link to this definition")
    :   * `SCENE`
          Scene – Use the time from the scene.
        * `CUSTOM`
          Custom – Use the modifier’s own time evaluation.

        Type:
        :   enum in [‘SCENE’, ‘CUSTOM’], default ‘SCENE’

    time\_mode[#](#bpy.types.MeshCacheModifier.time_mode "Link to this definition")
    :   Method to control playback time

        * `FRAME`
          Frame – Control playback using a frame-number (ignoring time FPS and start frame from the file).
        * `TIME`
          Time – Control playback using time in seconds.
        * `FACTOR`
          Factor – Control playback using a value between 0 and 1.

        Type:
        :   enum in [‘FRAME’, ‘TIME’, ‘FACTOR’], default ‘FRAME’

    up\_axis[#](#bpy.types.MeshCacheModifier.up_axis "Link to this definition")
    :   Type:
        :   enum in [Object Axis Items](../../../appendix/bpy_types_enum_items/object_axis_items.md#rna-enum-object-axis-items), default ‘POS\_Z’

    vertex\_group[#](#bpy.types.MeshCacheModifier.vertex_group "Link to this definition")
    :   Name of the Vertex Group which determines the influence of the modifier per point

        Type:
        :   string, default “”, (never None)
