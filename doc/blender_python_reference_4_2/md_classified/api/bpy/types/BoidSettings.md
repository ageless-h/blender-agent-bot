:   Settings for boid physics

    accuracy[#](#bpy.types.BoidSettings.accuracy "Link to this definition")
    :   Accuracy of attack

        Type:
        :   float in [0, 1], default 0.0

    active\_boid\_state[#](#bpy.types.BoidSettings.active_boid_state "Link to this definition")
    :   Type:

        :   int in [0, inf], default 0

    :   Boid will fight this times stronger enemy

        Type:
        :   float in [0, 100], default 0.0

    air\_acc\_max[#](#bpy.types.BoidSettings.air_acc_max "Link to this definition")

        Type:

    :   Maximum speed in air

        Type:
        :   float in [0, 100], default 0.0

    :   Minimum speed in air (relative to maximum speed)

        Type:
        :   float in [0, 1], default 0.0

    bank[#](#bpy.types.BoidSettings.bank "Link to this definition")
    :   Amount of rotation around velocity vector on turns

        Type:
        :   float in [0, 2], default 0.0

    health[#](#bpy.types.BoidSettings.health "Link to this definition")
    :   Initial boid health when born

        Type:
        :   float in [0, 100], default 0.0

    height[#](#bpy.types.BoidSettings.height "Link to this definition")
    :   Boid height relative to particle size

        Type:
        :   float in [0, 2], default 0.0

    land\_acc\_max[#](#bpy.types.BoidSettings.land_acc_max "Link to this definition")
    :   Maximum acceleration on land (relative to maximum speed)

        Type:
        :   float in [0, 1], default 0.0

    land\_ave\_max[#](#bpy.types.BoidSettings.land_ave_max "Link to this definition")
    :   Maximum angular velocity on land (relative to 180 degrees)

        Type:
        :   float in [0, 1], default 0.0

    land\_jump\_speed[#](#bpy.types.BoidSettings.land_jump_speed "Link to this definition")
    :   Maximum speed for jumping

        Type:
        :   float in [0, 100], default 0.0

    land\_personal\_space[#](#bpy.types.BoidSettings.land_personal_space "Link to this definition")
    :   Radius of boids personal space on land (% of particle size)

        Type:
        :   float in [0, 10], default 0.0

    land\_smooth[#](#bpy.types.BoidSettings.land_smooth "Link to this definition")
    :   How smoothly the boids land

        Type:
        :   float in [0, 10], default 0.0

    land\_speed\_max[#](#bpy.types.BoidSettings.land_speed_max "Link to this definition")
    :   Maximum speed on land

        Type:
        :   float in [0, 100], default 0.0

    land\_stick\_force[#](#bpy.types.BoidSettings.land_stick_force "Link to this definition")
    :   How strong a force must be to start effecting a boid on land

        Type:
        :   float in [0, 1000], default 0.0

    pitch[#](#bpy.types.BoidSettings.pitch "Link to this definition")
    :   Amount of rotation around side vector

        Type:
        :   float in [0, 2], default 0.0

    range[#](#bpy.types.BoidSettings.range "Link to this definition")
    :   Maximum distance from which a boid can attack

        Type:
        :   float in [0, 100], default 0.0

    states[#](#bpy.types.BoidSettings.states "Link to this definition")
    :   Type:
        :   [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`BoidState`](BoidState.md#bpy.types.BoidState "bpy.types.BoidState"), (readonly)

    strength[#](#bpy.types.BoidSettings.strength "Link to this definition")
    :   Maximum caused damage on attack per second

        Type:
        :   float in [0, 100], default 0.0

    use\_climb[#](#bpy.types.BoidSettings.use_climb "Link to this definition")
    :   Allow boids to climb goal objects

        Type:
        :   boolean, default False

    use\_flight[#](#bpy.types.BoidSettings.use_flight "Link to this definition")
    :   Allow boids to move in air

        Type:
        :   boolean, default False

    use\_land[#](#bpy.types.BoidSettings.use_land "Link to this definition")
    :   Allow boids to move on land

        Type:
        :   boolean, default False
