    bake[#](#bpy.types.RenderSettings.bake "Link to this definition")

        Type:
        :   float in [0, 1000], default 0.001

    bake\_margin[#](#bpy.types.RenderSettings.bake_margin "Link to this definition")

        * `EXTEND`
          Extend – Extend border pixels outwards.

        Type:

        :   int in [64, 1024], default 256

    :   Choose shading information to bake into the image

        * `NORMALS`
          Normals – Bake normals.
        * `DISPLACEMENT`
          Displacement – Bake displacement.

        Type:
        :   enum in [‘NORMALS’, ‘DISPLACEMENT’], default ‘NORMALS’

    bake\_user\_scale[#](#bpy.types.RenderSettings.bake_user_scale "Link to this definition")

        :   float in [0, 1000], default 0.0

        Type:
        :   float in [0, 1], default 1.0

    border\_max\_y[#](#bpy.types.RenderSettings.border_max_y "Link to this definition")
    :   Maximum Y value for the render region

        Type:
        :   float in [0, 1], default 1.0

    border\_min\_x[#](#bpy.types.RenderSettings.border_min_x "Link to this definition")
    :   Minimum X value for the render region

        Type:
        :   float in [0, 1], default 0.0

    border\_min\_y[#](#bpy.types.RenderSettings.border_min_y "Link to this definition")
    :   Minimum Y value for the render region

        Type:
        :   float in [0, 1], default 0.0

    compositor\_device[#](#bpy.types.RenderSettings.compositor_device "Link to this definition")
    :   Set how compositing is executed

        Type:
        :   enum in [‘CPU’, ‘GPU’], default ‘CPU’

    compositor\_precision[#](#bpy.types.RenderSettings.compositor_precision "Link to this definition")
    :   The precision of compositor intermediate result

        * `AUTO`
          Auto – Full precision for final renders, half precision otherwise.
        * `FULL`
          Full – Full precision.

        Type:
        :   enum in [‘AUTO’, ‘FULL’], default ‘AUTO’

        :   float in [0, inf], default 1.0

        Type:
        :   enum in [‘BLENDER\_EEVEE\_NEXT’], default ‘BLENDER\_EEVEE\_NEXT’
