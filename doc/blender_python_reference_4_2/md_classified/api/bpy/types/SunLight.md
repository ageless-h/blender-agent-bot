# SunLight(Light)[#](#sunlight-light "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`ID`](ID.md#bpy.types.ID "bpy.types.ID"), [`Light`](Light.md#bpy.types.Light "bpy.types.Light")

*class* bpy.types.SunLight(*Light*)[#](#bpy.types.SunLight "Link to this definition")
:   Constant direction parallel ray Light

    angle[#](#bpy.types.SunLight.angle "Link to this definition")
    :   Angular diameter of the Sun as seen from the Earth

        Type:
        :   float in [0, 3.14159], default 0.00918043

    contact\_shadow\_bias[#](#bpy.types.SunLight.contact_shadow_bias "Link to this definition")
    :   Bias to avoid self shadowing

        Type:
        :   float in [0.001, 9999], default 0.03

    contact\_shadow\_distance[#](#bpy.types.SunLight.contact_shadow_distance "Link to this definition")
    :   World space distance in which to search for screen space occluder

        Type:
        :   float in [0, 9999], default 0.2

    contact\_shadow\_thickness[#](#bpy.types.SunLight.contact_shadow_thickness "Link to this definition")
    :   Pixel thickness used to detect occlusion

        Type:
        :   float in [0, 9999], default 0.2

    energy[#](#bpy.types.SunLight.energy "Link to this definition")
    :   Sunlight strength in watts per meter squared (W/m²)

        Type:
        :   float in [-inf, inf], default 10.0

    shadow\_buffer\_bias[#](#bpy.types.SunLight.shadow_buffer_bias "Link to this definition")
    :   Bias for reducing self shadowing

        Type:
        :   float in [0, inf], default 1.0

    shadow\_buffer\_clip\_start[#](#bpy.types.SunLight.shadow_buffer_clip_start "Link to this definition")
    :   Shadow map clip start, below which objects will not generate shadows

        Type:
        :   float in [1e-06, inf], default 0.05

    shadow\_cascade\_count[#](#bpy.types.SunLight.shadow_cascade_count "Link to this definition")
    :   Number of texture used by the cascaded shadow map

        Type:
        :   int in [1, 4], default 4

    shadow\_cascade\_exponent[#](#bpy.types.SunLight.shadow_cascade_exponent "Link to this definition")
    :   Higher value increase resolution towards the viewpoint

        Type:
        :   float in [0, 1], default 0.8

    shadow\_cascade\_fade[#](#bpy.types.SunLight.shadow_cascade_fade "Link to this definition")
    :   How smooth is the transition between each cascade

        Type:
        :   float in [0, 1], default 0.1

    shadow\_cascade\_max\_distance[#](#bpy.types.SunLight.shadow_cascade_max_distance "Link to this definition")
    :   End distance of the cascaded shadow map (only in perspective view)

        Type:
        :   float in [0, inf], default 200.0

    shadow\_color[#](#bpy.types.SunLight.shadow_color "Link to this definition")
    :   Color of shadows cast by the light

        Type:
        :   [`mathutils.Color`](../../mathutils/index.md#mathutils.Color "mathutils.Color") of 3 items in [0, inf], default (0.0, 0.0, 0.0)

    shadow\_filter\_radius[#](#bpy.types.SunLight.shadow_filter_radius "Link to this definition")
    :   Blur shadow aliasing using Percentage Closer Filtering

        Type:
        :   float in [0, inf], default 1.0

    shadow\_jitter\_overblur[#](#bpy.types.SunLight.shadow_jitter_overblur "Link to this definition")
    :   Apply shadow tracing to each jittered sample to reduce under-sampling artifacts

        Type:
        :   float in [0, 100], default 10.0

    shadow\_maximum\_resolution[#](#bpy.types.SunLight.shadow_maximum_resolution "Link to this definition")
    :   Minimum size of a shadow map pixel. Higher values use less memory at the cost of shadow quality

        Type:
        :   float in [0, inf], default 0.001

    shadow\_soft\_size[#](#bpy.types.SunLight.shadow_soft_size "Link to this definition")
    :   Light size for ray shadow sampling (Raytraced shadows)

        Type:
        :   float in [0, inf], default 0.0

    use\_contact\_shadow[#](#bpy.types.SunLight.use_contact_shadow "Link to this definition")
    :   Use screen space ray-tracing to have correct shadowing near occluder, or for small features that does not appear in shadow maps

        Type:
        :   boolean, default False

    use\_shadow\_jitter[#](#bpy.types.SunLight.use_shadow_jitter "Link to this definition")
    :   Enable jittered soft shadows to increase shadow precision (disabled in viewport unless enabled in the render settings). Has a high performance impact

        Type:
        :   boolean, default False
