:   Denoise rendered animation sequence using current scene and view layer settings. Requires denoising data passes and output to OpenEXR multilayer files

    Parameters:
    :   * **input\_filepath** (*string**,* *(**optional**,* *never None**)*) – Input Filepath, File path for image to denoise. If not specified, uses the render file path and frame range from the scene
        * **output\_filepath** (*string**,* *(**optional**,* *never None**)*) – Output Filepath, If not specified, renders will be denoised in-place
