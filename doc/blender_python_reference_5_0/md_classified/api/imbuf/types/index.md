# Image Buffer Types (imbuf.types)[¶](#module-imbuf.types "Link to this heading")

This module provides access to image buffer types.

Note

Image buffer is also the structure used by [`bpy.types.Image`](../../bpy/types/Image.md#bpy.types.Image "bpy.types.Image")
ID type to store and manipulate image data at runtime.

*class* imbuf.types.ImBuf[¶](#imbuf.types.imbuf.types.ImBuf "Link to this definition")
:   copy()[¶](#imbuf.types.imbuf.types.ImBuf.copy "Link to this definition")
    :   Returns:
        :   A copy of the image.

        Return type:
        :   `ImBuf`

    crop(*min*, *max*)[¶](#imbuf.types.imbuf.types.ImBuf.crop "Link to this definition")
    :   Crop the image.

        Parameters:
        :   * **min** (*tuple**[**int**,* *int**]*) – X, Y minimum.
            * **max** (*tuple**[**int**,* *int**]*) – X, Y maximum.

    free()[¶](#imbuf.types.imbuf.types.ImBuf.free "Link to this definition")
    :   Clear image data immediately (causing an error on re-use).

    resize(*size*, *\**, *method='FAST'*)[¶](#imbuf.types.imbuf.types.ImBuf.resize "Link to this definition")
    :   Resize the image.

        Parameters:
        :   * **size** (*tuple**[**int**,* *int**]*) – New size.
            * **method** (*str*) – Method of resizing (‘FAST’, ‘BILINEAR’)

    channels[¶](#imbuf.types.imbuf.types.ImBuf.channels "Link to this definition")
    :   Number of bit-planes.

        Type:
        :   int

    filepath[¶](#imbuf.types.imbuf.types.ImBuf.filepath "Link to this definition")
    :   filepath associated with this image.

        Type:
        :   str

    planes[¶](#imbuf.types.imbuf.types.ImBuf.planes "Link to this definition")
    :   Number of bits associated with this image.

        Type:
        :   int

    ppm[¶](#imbuf.types.imbuf.types.ImBuf.ppm "Link to this definition")
    :   pixels per meter.

        Type:
        :   tuple[float, float]

    size[¶](#imbuf.types.imbuf.types.ImBuf.size "Link to this definition")
    :   size of the image in pixels.

        Type:
        :   tuple[int, int]
