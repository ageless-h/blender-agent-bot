:   Copy particle systems from the active object to selected objects

    Parameters:
    :   * **space** (enum in [`'OBJECT'`, `'WORLD'`], (optional)) –

          Space, Space transform for copying from one object to another

          + `OBJECT`
            Object – Copy inside each object’s local space.
          + `WORLD`
            World – Copy in world space.
        * **remove\_target\_particles** (*boolean**,* *(**optional**)*) – Remove Target Particles, Remove particle systems on the target objects
        * **use\_active** (*boolean**,* *(**optional**)*) – Use Active, Use the active particle system from the context
