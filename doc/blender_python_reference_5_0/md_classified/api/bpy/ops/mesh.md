:   Cut into selected items at an angle to create bevel or chamfer

    Parameters:
    :   * **offset\_type** (enum in [`'OFFSET'`, `'WIDTH'`, `'DEPTH'`, `'PERCENT'`, `'ABSOLUTE'`], (optional)) ‚Äì

          Width Type, The method for determining the size of the bevel

          + `OFFSET`
            Offset ‚Äì Amount is offset of new edges from original.
          + `WIDTH`
            Width ‚Äì Amount is width of new face.
          + `DEPTH`
            Depth ‚Äì Amount is perpendicular distance from original edge to bevel face.
          + `PERCENT`
            Percent ‚Äì Amount is percent of adjacent edge length.
          + `ABSOLUTE`
            Absolute ‚Äì Amount is absolute distance along adjacent edge.
        * **offset** (*float in* *[**0**,* *1e+06**]**,* *(**optional**)*) ‚Äì Width, Bevel amount

          Profile Type, The type of shape used to rebuild a beveled section

          + `SUPERELLIPSE`
            Superellipse ‚Äì The profile can be a concave or convex curve.
          + `CUSTOM`
            Custom ‚Äì The profile can be any arbitrary path between its endpoints.
        * **offset\_pct** (*float in* *[**0**,* *100**]**,* *(**optional**)*) ‚Äì Width Percent, Bevel amount for percentage method
        * **segments** (*int in* *[**1**,* *1000**]**,* *(**optional**)*) ‚Äì Segments, Segments for curved edge
        * **profile** (*float in* *[**0**,* *1**]**,* *(**optional**)*) ‚Äì Profile, Controls profile shape (0.5 = round)
        * **affect** (enum in [`'VERTICES'`, `'EDGES'`], (optional)) ‚Äì

          Affect, Affect edges or vertices

          + `VERTICES`
            Vertices ‚Äì Affect only vertices.
          + `EDGES`
            Edges ‚Äì Affect only edges.
        * **clamp\_overlap** (*boolean**,* *(**optional**)*) ‚Äì Clamp Overlap, Do not allow beveled edges/vertices to overlap each other
        * **loop\_slide** (*boolean**,* *(**optional**)*) ‚Äì Loop Slide, Prefer sliding along edges to even widths
        * **mark\_seam** (*boolean**,* *(**optional**)*) ‚Äì Mark Seams, Preserve seams along beveled edges

        * **material** (*int in* *[**-1**,* *inf**]**,* *(**optional**)*) ‚Äì Material Index, Material for bevel faces (-1 means use adjacent faces)
        * **harden\_normals** (*boolean**,* *(**optional**)*) ‚Äì Harden Normals, Match normals of new faces to adjacent faces
        * **face\_strength\_mode** (enum in [`'NONE'`, `'NEW'`, `'AFFECTED'`, `'ALL'`], (optional)) ‚Äì

          Face Strength Mode, Whether to set face strength, and which faces to set face strength on

          + `NONE`
            None ‚Äì Do not set face strength.
          + `NEW`
            New ‚Äì Set face strength on new faces only.
          + `AFFECTED`
            Affected ‚Äì Set face strength on new and modified faces only.
          + `ALL`
            All ‚Äì Set face strength on all faces.
        * **miter\_outer** (enum in [`'SHARP'`, `'PATCH'`, `'ARC'`], (optional)) ‚Äì

          Outer Miter, Pattern to use for outside of miters

          + `SHARP`
            Sharp ‚Äì Outside of miter is sharp.
          + `PATCH`
            Patch ‚Äì Outside of miter is squared-off patch.
          + `ARC`
            Arc ‚Äì Outside of miter is arc.
        * **miter\_inner** (enum in [`'SHARP'`, `'ARC'`], (optional)) ‚Äì

          Inner Miter, Pattern to use for inside of miters

          + `SHARP`
            Sharp ‚Äì Inside of miter is sharp.
          + `ARC`
            Arc ‚Äì Inside of miter is arc.

            Cutoff ‚Äì A cutoff at each profile‚Äôs end before the intersection.
