# KeyingSetPath(bpy\_struct)[¶](#keyingsetpath-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.KeyingSetPath(*bpy\_struct*)[¶](#bpy.types.KeyingSetPath "Link to this definition")
:   Path to a setting for use in a Keying Set

    array\_index[¶](#bpy.types.KeyingSetPath.array_index "Link to this definition")
    :   Index to the specific setting if applicable

        Type:
        :   int in [-inf, inf], default 0

    data\_path[¶](#bpy.types.KeyingSetPath.data_path "Link to this definition")
    :   Path to property setting

        Type:
        :   string, default “”, (never None)

    group[¶](#bpy.types.KeyingSetPath.group "Link to this definition")
    :   Name of Action Group to assign setting(s) for this path to

        Type:
        :   string, default “”, (never None)

    group\_method[¶](#bpy.types.KeyingSetPath.group_method "Link to this definition")
    :   Method used to define which Group-name to use

        Type:
        :   enum in [Keyingset Path Grouping Items](../../../appendix/bpy_types_enum_items/keyingset_path_grouping_items.md#rna-enum-keyingset-path-grouping-items), default `'NAMED'`

    id[¶](#bpy.types.KeyingSetPath.id "Link to this definition")
    :   ID-Block that keyframes for Keying Set should be added to (for Absolute Keying Sets only)

        Type:
        :   [`ID`](ID.md#bpy.types.ID "bpy.types.ID")

    id\_type[¶](#bpy.types.KeyingSetPath.id_type "Link to this definition")
    :   Type of ID-block that can be used

        Type:
        :   enum in [Id Type Items](../../../appendix/bpy_types_enum_items/id_type_items.md#rna-enum-id-type-items), default `'OBJECT'`

    use\_entire\_array[¶](#bpy.types.KeyingSetPath.use_entire_array "Link to this definition")
    :   When an ‘array/vector’ type is chosen (Location, Rotation, Color, etc.), entire array is to be used

        Type:
        :   boolean, default False

    use\_insertkey\_needed[¶](#bpy.types.KeyingSetPath.use_insertkey_needed "Link to this definition")
    :   Only insert keyframes where they’re needed in the relevant F-Curves

        Type:
        :   boolean, default False

    use\_insertkey\_override\_needed[¶](#bpy.types.KeyingSetPath.use_insertkey_override_needed "Link to this definition")
    :   Override default setting to only insert keyframes where they’re needed in the relevant F-Curves

        Type:
        :   boolean, default False

    use\_insertkey\_override\_visual[¶](#bpy.types.KeyingSetPath.use_insertkey_override_visual "Link to this definition")
    :   Override default setting to insert keyframes based on ‘visual transforms’

        Type:
        :   boolean, default False

    use\_insertkey\_visual[¶](#bpy.types.KeyingSetPath.use_insertkey_visual "Link to this definition")
    :   Insert keyframes based on ‘visual transforms’

        Type:
        :   boolean, default False

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.KeyingSetPath.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass

    *classmethod* bl\_rna\_get\_subclass\_py(*id*, *default=None*, */*)[¶](#bpy.types.KeyingSetPath.bl_rna_get_subclass_py "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The class or default when not found.

        Return type:
        :   type
