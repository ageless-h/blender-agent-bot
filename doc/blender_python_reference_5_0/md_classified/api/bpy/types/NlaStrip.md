# NlaStrip(bpy\_struct)[¶](#nlastrip-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.NlaStrip(*bpy\_struct*)[¶](#bpy.types.NlaStrip "Link to this definition")
:   A container referencing an existing Action

    action[¶](#bpy.types.NlaStrip.action "Link to this definition")
    :   Action referenced by this strip

        Type:
        :   [`Action`](Action.md#bpy.types.Action "bpy.types.Action")

    action\_frame\_end[¶](#bpy.types.NlaStrip.action_frame_end "Link to this definition")
    :   Last frame from action to use

        Type:
        :   float in [-inf, inf], default 0.0

    action\_frame\_start[¶](#bpy.types.NlaStrip.action_frame_start "Link to this definition")
    :   First frame from action to use

        Type:
        :   float in [-inf, inf], default 0.0

    action\_slot[¶](#bpy.types.NlaStrip.action_slot "Link to this definition")
    :   The slot identifies which sub-set of the Action is considered to be for this strip, and its name is used to find the right slot when assigning another Action

        Type:
        :   [`ActionSlot`](ActionSlot.md#bpy.types.ActionSlot "bpy.types.ActionSlot")

    action\_slot\_handle[¶](#bpy.types.NlaStrip.action_slot_handle "Link to this definition")
    :   A number that identifies which sub-set of the Action is considered to be for this NLA strip

        Type:
        :   int in [-inf, inf], default 0

    action\_suitable\_slots[¶](#bpy.types.NlaStrip.action_suitable_slots "Link to this definition")
    :   The list of action slots suitable for this NLA strip

        Type:
        :   [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`ActionSlot`](ActionSlot.md#bpy.types.ActionSlot "bpy.types.ActionSlot"), (readonly)

    active[¶](#bpy.types.NlaStrip.active "Link to this definition")
    :   NLA Strip is active

        Type:
        :   boolean, default False, (readonly)

    blend\_in[¶](#bpy.types.NlaStrip.blend_in "Link to this definition")
    :   Number of frames at start of strip to fade in influence

        Type:
        :   float in [-inf, inf], default 0.0

    blend\_out[¶](#bpy.types.NlaStrip.blend_out "Link to this definition")
    :   Type:
        :   float in [-inf, inf], default 0.0

    blend\_type[¶](#bpy.types.NlaStrip.blend_type "Link to this definition")
    :   Method used for combining strip’s result with accumulated result

        * `REPLACE`
          Replace – The strip values replace the accumulated results by amount specified by influence.
        * `COMBINE`
          Combine – The strip values are combined with accumulated results by appropriately using addition, multiplication, or quaternion math, based on channel type.
        * `ADD`
          Add – Weighted result of strip is added to the accumulated results.
        * `SUBTRACT`
          Subtract – Weighted result of strip is removed from the accumulated results.
        * `MULTIPLY`
          Multiply – Weighted result of strip is multiplied with the accumulated results.

        Type:
        :   enum in [`'REPLACE'`, `'COMBINE'`, `'ADD'`, `'SUBTRACT'`, `'MULTIPLY'`], default `'REPLACE'`

    extrapolation[¶](#bpy.types.NlaStrip.extrapolation "Link to this definition")
    :   Action to take for gaps past the strip extents

        * `NOTHING`
          Nothing – Strip has no influence past its extents.
        * `HOLD`
          Hold – Hold the first frame if no previous strips in track, and always hold last frame.
        * `HOLD_FORWARD`
          Hold Forward – Only hold last frame.

        Type:
        :   enum in [`'NOTHING'`, `'HOLD'`, `'HOLD_FORWARD'`], default `'HOLD'`

    fcurves[¶](#bpy.types.NlaStrip.fcurves "Link to this definition")
    :   F-Curves for controlling the strip’s influence and timing

        Type:
        :   [`NlaStripFCurves`](NlaStripFCurves.md#bpy.types.NlaStripFCurves "bpy.types.NlaStripFCurves") [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`FCurve`](FCurve.md#bpy.types.FCurve "bpy.types.FCurve"), (readonly)

    frame\_end[¶](#bpy.types.NlaStrip.frame_end "Link to this definition")
    :   Type:
        :   float in [-inf, inf], default 0.0

    frame\_end\_raw[¶](#bpy.types.NlaStrip.frame_end_raw "Link to this definition")
    :   Same as frame\_end, except that any value can be set, including ones that create an invalid state

        Type:
        :   float in [-inf, inf], default 0.0

    frame\_end\_ui[¶](#bpy.types.NlaStrip.frame_end_ui "Link to this definition")
    :   End frame of the NLA strip. Note: changing this value also updates the value of the strip’s repeats or its action’s end frame. If only the end frame should be changed, see the “frame\_end” property instead.

        Type:
        :   float in [-inf, inf], default 0.0

    frame\_start[¶](#bpy.types.NlaStrip.frame_start "Link to this definition")
    :   Type:
        :   float in [-inf, inf], default 0.0

    frame\_start\_raw[¶](#bpy.types.NlaStrip.frame_start_raw "Link to this definition")
    :   Same as frame\_start, except that any value can be set, including ones that create an invalid state

        Type:
        :   float in [-inf, inf], default 0.0

    frame\_start\_ui[¶](#bpy.types.NlaStrip.frame_start_ui "Link to this definition")
    :   Start frame of the NLA strip. Note: changing this value also updates the value of the strip’s end frame. If only the start frame should be changed, see the “frame\_start” property instead.

        Type:
        :   float in [-inf, inf], default 0.0

    influence[¶](#bpy.types.NlaStrip.influence "Link to this definition")
    :   Amount the strip contributes to the current result

        Type:
        :   float in [0, 1], default 0.0

    last\_slot\_identifier[¶](#bpy.types.NlaStrip.last_slot_identifier "Link to this definition")
    :   The identifier of the most recently assigned action slot. The slot identifies which sub-set of the Action is considered to be for this strip, and its identifier is used to find the right slot when assigning an Action.

        Type:
        :   string, default “”, (never None)

    modifiers[¶](#bpy.types.NlaStrip.modifiers "Link to this definition")
    :   Modifiers affecting all the F-Curves in the referenced Action

        Type:
        :   [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`FModifier`](FModifier.md#bpy.types.FModifier "bpy.types.FModifier"), (readonly)

    mute[¶](#bpy.types.NlaStrip.mute "Link to this definition")
    :   Disable NLA Strip evaluation

        Type:
        :   boolean, default False

    name[¶](#bpy.types.NlaStrip.name "Link to this definition")
    :   Type:
        :   string, default “”, (never None)

    repeat[¶](#bpy.types.NlaStrip.repeat "Link to this definition")
    :   Number of times to repeat the action range

        Type:
        :   float in [0.1, 1000], default 1.0

    scale[¶](#bpy.types.NlaStrip.scale "Link to this definition")
    :   Scaling factor for action

        Type:
        :   float in [0.0001, 1000], default 1.0

    select[¶](#bpy.types.NlaStrip.select "Link to this definition")
    :   NLA Strip is selected

        Type:
        :   boolean, default False

    strip\_time[¶](#bpy.types.NlaStrip.strip_time "Link to this definition")
    :   Frame of referenced Action to evaluate

        Type:
        :   float in [-inf, inf], default 0.0

    strips[¶](#bpy.types.NlaStrip.strips "Link to this definition")
    :   NLA Strips that this strip acts as a container for (if it is of type Meta)

        Type:
        :   [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`NlaStrip`](#bpy.types.NlaStrip "bpy.types.NlaStrip"), (readonly)

    type[¶](#bpy.types.NlaStrip.type "Link to this definition")
    :   Type of NLA Strip

        * `CLIP`
          Action Clip – NLA Strip references some Action.
        * `TRANSITION`
          Transition – NLA Strip ‘transitions’ between adjacent strips.
        * `META`
          Meta – NLA Strip acts as a container for adjacent strips.
        * `SOUND`
          Sound Clip – NLA Strip representing a sound event for speakers.

        Type:
        :   enum in [`'CLIP'`, `'TRANSITION'`, `'META'`, `'SOUND'`], default `'CLIP'`, (readonly)

    use\_animated\_influence[¶](#bpy.types.NlaStrip.use_animated_influence "Link to this definition")
    :   Influence setting is controlled by an F-Curve rather than automatically determined

        Type:
        :   boolean, default False

    use\_animated\_time[¶](#bpy.types.NlaStrip.use_animated_time "Link to this definition")
    :   Strip time is controlled by an F-Curve rather than automatically determined

        Type:
        :   boolean, default False

    use\_animated\_time\_cyclic[¶](#bpy.types.NlaStrip.use_animated_time_cyclic "Link to this definition")
    :   Cycle the animated time within the action start and end

        Type:
        :   boolean, default False

    use\_auto\_blend[¶](#bpy.types.NlaStrip.use_auto_blend "Link to this definition")
    :   Number of frames for Blending In/Out is automatically determined from overlapping strips

        Type:
        :   boolean, default False

    use\_reverse[¶](#bpy.types.NlaStrip.use_reverse "Link to this definition")
    :   NLA Strip is played back in reverse order (only when timing is automatically determined)

        Type:
        :   boolean, default False

    use\_sync\_length[¶](#bpy.types.NlaStrip.use_sync_length "Link to this definition")
    :   Update range of frames referenced from action after tweaking strip and its keyframes

        Type:
        :   boolean, default False
