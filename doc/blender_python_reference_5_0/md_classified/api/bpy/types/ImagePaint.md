# ImagePaint(Paint)[¶](#imagepaint-paint "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`Paint`](Paint.md#bpy.types.Paint "bpy.types.Paint")

*class* bpy.types.ImagePaint(*Paint*)[¶](#bpy.types.ImagePaint "Link to this definition")
:   Properties of image and texture painting mode

    canvas[¶](#bpy.types.ImagePaint.canvas "Link to this definition")
    :   Image used as canvas

        Type:
        :   [`Image`](Image.md#bpy.types.Image "bpy.types.Image")

    clone\_alpha[¶](#bpy.types.ImagePaint.clone_alpha "Link to this definition")
    :   Opacity of clone image display

        Type:
        :   float in [0, 1], default 0.5

    clone\_image[¶](#bpy.types.ImagePaint.clone_image "Link to this definition")
    :   Image used as clone source

        Type:
        :   [`Image`](Image.md#bpy.types.Image "bpy.types.Image")

    clone\_offset[¶](#bpy.types.ImagePaint.clone_offset "Link to this definition")
    :   Type:
        :   [`mathutils.Vector`](../../mathutils/index.md#mathutils.Vector "mathutils.Vector") of 2 items in [-inf, inf], default (0.0, 0.0)

    dither[¶](#bpy.types.ImagePaint.dither "Link to this definition")
    :   Amount of dithering when painting on byte images

        Type:
        :   float in [0, 2], default 0.0

    interpolation[¶](#bpy.types.ImagePaint.interpolation "Link to this definition")
    :   Texture filtering type

        * `LINEAR`
          Linear – Linear interpolation.
        * `CLOSEST`
          Closest – No interpolation (sample closest texel).

        Type:
        :   enum in [`'LINEAR'`, `'CLOSEST'`], default `'LINEAR'`

    invert\_stencil[¶](#bpy.types.ImagePaint.invert_stencil "Link to this definition")
    :   Invert the stencil layer

        Type:
        :   boolean, default False

    missing\_materials[¶](#bpy.types.ImagePaint.missing_materials "Link to this definition")
    :   The mesh is missing materials

        Type:
        :   boolean, default False, (readonly)

    missing\_stencil[¶](#bpy.types.ImagePaint.missing_stencil "Link to this definition")
    :   Image Painting does not have a stencil

        Type:
        :   boolean, default False, (readonly)

    missing\_texture[¶](#bpy.types.ImagePaint.missing_texture "Link to this definition")
    :   Image Painting does not have a texture to paint on

        Type:
        :   boolean, default False, (readonly)

    missing\_uvs[¶](#bpy.types.ImagePaint.missing_uvs "Link to this definition")
    :   A UV layer is missing on the mesh

        Type:
        :   boolean, default False, (readonly)

    mode[¶](#bpy.types.ImagePaint.mode "Link to this definition")
    :   Mode of operation for projection painting

        * `MATERIAL`
          Material – Detect image slots from the material.
        * `IMAGE`
          Single Image – Set image for texture painting directly.

        Type:
        :   enum in [`'MATERIAL'`, `'IMAGE'`], default `'MATERIAL'`

    normal\_angle[¶](#bpy.types.ImagePaint.normal_angle "Link to this definition")
    :   Paint most on faces pointing towards the view according to this angle

        Type:
        :   int in [0, 90], default 80

    screen\_grab\_size[¶](#bpy.types.ImagePaint.screen_grab_size "Link to this definition")
    :   Size to capture the image for re-projecting

        Type:
        :   int array of 2 items in [512, 16384], default (0, 0)

    seam\_bleed[¶](#bpy.types.ImagePaint.seam_bleed "Link to this definition")
    :   Extend paint beyond the faces’ UVs to reduce seams (in pixels, slower)

        Type:
        :   int in [-32768, 32767], default 2

    stencil\_color[¶](#bpy.types.ImagePaint.stencil_color "Link to this definition")
    :   Stencil color in the viewport

        Type:
        :   [`mathutils.Color`](../../mathutils/index.md#mathutils.Color "mathutils.Color") of 3 items in [0, 1], default (0.0, 0.0, 0.0)

    stencil\_image[¶](#bpy.types.ImagePaint.stencil_image "Link to this definition")
    :   Image used as stencil

        Type:
        :   [`Image`](Image.md#bpy.types.Image "bpy.types.Image")

    use\_backface\_culling[¶](#bpy.types.ImagePaint.use_backface_culling "Link to this definition")
    :   Ignore faces pointing away from the view (faster)

        Type:
        :   boolean, default True

    use\_clone\_layer[¶](#bpy.types.ImagePaint.use_clone_layer "Link to this definition")
    :   Use another UV map as clone source, otherwise use the 3D cursor as the source

        Type:
        :   boolean, default False

    use\_normal\_falloff[¶](#bpy.types.ImagePaint.use_normal_falloff "Link to this definition")
    :   Paint most on faces pointing towards the view

        Type:
        :   boolean, default True

    use\_occlude[¶](#bpy.types.ImagePaint.use_occlude "Link to this definition")
    :   Only paint onto the faces directly under the brush (slower)

        Type:
        :   boolean, default True

    use\_stencil\_layer[¶](#bpy.types.ImagePaint.use_stencil_layer "Link to this definition")
    :   Set the mask layer from the UV map buttons

        Type:
        :   boolean, default False

    detect\_data()[¶](#bpy.types.ImagePaint.detect_data "Link to this definition")
    :   Check if required texpaint data exist

        Return type:
        :   boolean
