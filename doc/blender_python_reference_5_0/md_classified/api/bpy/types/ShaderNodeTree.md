*class* bpy.types.ShaderNodeTree(*NodeTree*)[¶](#bpy.types.ShaderNodeTree "Link to this definition")
:   Node tree consisting of linked nodes used for materials (and other shading data-blocks)

    get\_output\_node(*target*)[¶](#bpy.types.ShaderNodeTree.get_output_node "Link to this definition")
    :   Return active shader output node for the specified target

        Parameters:
        :   **target** (enum in [`'ALL'`, `'EEVEE'`, `'CYCLES'`]) –

            Target

            * `ALL`
              All – Use shaders for all renderers and viewports, unless there exists a more specific output.
            * `EEVEE`
              EEVEE – Use shaders for EEVEE renderer.
            * `CYCLES`
              Cycles – Use shaders for Cycles renderer.

        Returns:
        :   Node

        Return type:
        :   [`ShaderNode`](ShaderNode.md#bpy.types.ShaderNode "bpy.types.ShaderNode")
