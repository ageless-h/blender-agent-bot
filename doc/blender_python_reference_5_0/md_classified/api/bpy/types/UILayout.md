*class* bpy.types.UILayout(*bpy\_struct*)[¶](#bpy.types.UILayout "Link to this definition")
:   User interface layout in a panel or header

    activate\_init[¶](#bpy.types.UILayout.activate_init "Link to this definition")
    :   When true, buttons defined in popups will be activated on first display (use so you can type into a field without having to click on it first)

        Type:
        :   boolean, default False

    active[¶](#bpy.types.UILayout.active "Link to this definition")
    :   Type:
        :   boolean, default False

    active\_default[¶](#bpy.types.UILayout.active_default "Link to this definition")
    :   When true, an operator button defined after this will be activated when pressing return(use with popup dialogs)

        Type:
        :   boolean, default False

    alert[¶](#bpy.types.UILayout.alert "Link to this definition")
    :   Type:
        :   boolean, default False

    alignment[¶](#bpy.types.UILayout.alignment "Link to this definition")
    :   Type:
        :   enum in [`'EXPAND'`, `'LEFT'`, `'CENTER'`, `'RIGHT'`], default `'EXPAND'`

    direction[¶](#bpy.types.UILayout.direction "Link to this definition")
    :   Type:
        :   enum in [`'HORIZONTAL'`, `'VERTICAL'`], default `'HORIZONTAL'`, (readonly)

    emboss[¶](#bpy.types.UILayout.emboss "Link to this definition")
    :   * `NORMAL`
          Regular – Draw standard button emboss style.
        * `NONE`
          None – Draw only text and icons.
        * `PULLDOWN_MENU`
          Pull-down Menu – Draw pull-down menu style.
        * `PIE_MENU`
          Pie Menu – Draw radial menu style.
        * `NONE_OR_STATUS`
          None or Status – Draw with no emboss unless the button has a coloring status like an animation state.

        Type:
        :   enum in [`'NORMAL'`, `'NONE'`, `'PULLDOWN_MENU'`, `'PIE_MENU'`, `'NONE_OR_STATUS'`], default `'NORMAL'`

    enabled[¶](#bpy.types.UILayout.enabled "Link to this definition")
    :   When false, this (sub)layout is grayed out

        Type:
        :   boolean, default False

    operator\_context[¶](#bpy.types.UILayout.operator_context "Link to this definition")
    :   Typically set to ‘INVOKE\_REGION\_WIN’, except some cases in [`bpy.types.Menu`](Menu.md#bpy.types.Menu "bpy.types.Menu") when it’s set to ‘EXEC\_REGION\_WIN’.

        Type:
        :   enum in [Operator Context Items](../../../appendix/bpy_types_enum_items/operator_context_items.md#rna-enum-operator-context-items), default `'INVOKE_DEFAULT'`

    scale\_x[¶](#bpy.types.UILayout.scale_x "Link to this definition")
    :   Scale factor along the X for items in this (sub)layout

        Type:
        :   float in [0, inf], default 0.0

    scale\_y[¶](#bpy.types.UILayout.scale_y "Link to this definition")
    :   Scale factor along the Y for items in this (sub)layout

        Type:
        :   float in [0, inf], default 0.0

        Type:
        :   float in [0, inf], default 0.0

    ui\_units\_y[¶](#bpy.types.UILayout.ui_units_y "Link to this definition")
    :   Fixed size along the Y for items in this (sub)layout

        Type:
        :   float in [0, inf], default 0.0

    :   Creates a collapsible panel. Whether it is open or closed is stored in the region using the given idname. This can only be used when the panel has the full width of the panel region available to it. So it can’t be used in e.g. in a box or columns.

        Parameters:

        :   ([`UILayout`](#bpy.types.UILayout "bpy.types.UILayout"), [`UILayout`](#bpy.types.UILayout "bpy.types.UILayout"))

    panel\_prop(*data*, *property*)[¶](#bpy.types.UILayout.panel_prop "Link to this definition")
    :   Similar to `.panel(...)` but instead of storing whether it is open or closed in the region, it is stored in the provided boolean property. This should be used when multiple instances of the same panel can exist. For example one for every item in a collection property or list. This can only be used when the panel has the full width of the panel region available to it. So it can’t be used in e.g. in a box or columns.

        Parameters:
        :   * **data** ([`AnyType`](AnyType.md#bpy.types.AnyType "bpy.types.AnyType"), (never None)) – Data from which to take the open-state property
            * **property** (*string**,* *(**never None**)*) – Identifier of the boolean property that determines whether the panel is open or closed

        Returns:
        :   `layout_header`, Sub-layout to put items in, [`UILayout`](#bpy.types.UILayout "bpy.types.UILayout")

            `layout_body`, Sub-layout to put items in. Will be none if the panel is collapsed., [`UILayout`](#bpy.types.UILayout "bpy.types.UILayout")
