# FFmpegSettings(bpy\_struct)[¶](#ffmpegsettings-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.FFmpegSettings(*bpy\_struct*)[¶](#bpy.types.FFmpegSettings "Link to this definition")
:   FFmpeg related settings for the scene

    audio\_bitrate[¶](#bpy.types.FFmpegSettings.audio_bitrate "Link to this definition")
    :   Audio bitrate (kb/s)

        Type:
        :   int in [32, 384], default 192

    audio\_channels[¶](#bpy.types.FFmpegSettings.audio_channels "Link to this definition")
    :   Audio channel count

        * `MONO`
          Mono – Set audio channels to mono.
        * `STEREO`
          Stereo – Set audio channels to stereo.
        * `SURROUND4`
          4 Channels – Set audio channels to 4 channels.
        * `SURROUND51`
          5.1 Surround – Set audio channels to 5.1 surround sound.
        * `SURROUND71`
          7.1 Surround – Set audio channels to 7.1 surround sound.

        Type:
        :   enum in [`'MONO'`, `'STEREO'`, `'SURROUND4'`, `'SURROUND51'`, `'SURROUND71'`], default `'STEREO'`

    audio\_codec[¶](#bpy.types.FFmpegSettings.audio_codec "Link to this definition")
    :   FFmpeg audio codec to use

        * `NONE`
          No Audio – Disables audio output, for video-only renders.
        * `AAC`
          AAC.
        * `AC3`
          AC3.
        * `FLAC`
          FLAC.
        * `MP2`
          MP2.
        * `MP3`
          MP3.
        * `OPUS`
          Opus.
        * `PCM`
          PCM.
        * `VORBIS`
          Vorbis.

        Type:
        :   enum in [`'NONE'`, `'AAC'`, `'AC3'`, `'FLAC'`, `'MP2'`, `'MP3'`, `'OPUS'`, `'PCM'`, `'VORBIS'`], default `'NONE'`

    audio\_mixrate[¶](#bpy.types.FFmpegSettings.audio_mixrate "Link to this definition")
    :   Audio sample rate (samples/s)

        Type:
        :   int in [8000, 192000], default 48000

    audio\_volume[¶](#bpy.types.FFmpegSettings.audio_volume "Link to this definition")
    :   Audio volume

        Type:
        :   float in [0, 1], default 1.0

    buffersize[¶](#bpy.types.FFmpegSettings.buffersize "Link to this definition")
    :   Rate control: buffer size (kb)

        Type:
        :   int in [0, 2000], default 0

    codec[¶](#bpy.types.FFmpegSettings.codec "Link to this definition")
    :   FFmpeg codec to use for video output

        * `NONE`
          No Video – Disables video output, for audio-only renders.
        * `AV1`
          AV1.
        * `H264`
          H.264.
        * `H265`
          H.265 / HEVC.
        * `WEBM`
          WebM / VP9.
        * `DNXHD`
          DNxHD.
        * `DV`
          DV.
        * `FFV1`
          FFmpeg video codec #1.
        * `FLASH`
          Flash Video.
        * `HUFFYUV`
          HuffYUV.
        * `MPEG1`
          MPEG-1.
        * `MPEG2`
          MPEG-2.
        * `MPEG4`
          MPEG-4 (divx).
        * `PNG`
          PNG.
        * `PRORES`
          ProRes.
        * `QTRLE`
          QuickTime Animation.
        * `THEORA`
          Theora.

        Type:
        :   enum in [`'NONE'`, `'AV1'`, `'H264'`, `'H265'`, `'WEBM'`, `'DNXHD'`, `'DV'`, `'FFV1'`, `'FLASH'`, `'HUFFYUV'`, `'MPEG1'`, `'MPEG2'`, `'MPEG4'`, `'PNG'`, `'PRORES'`, `'QTRLE'`, `'THEORA'`], default `'H264'`

    constant\_rate\_factor[¶](#bpy.types.FFmpegSettings.constant_rate_factor "Link to this definition")
    :   Constant Rate Factor (CRF); tradeoff between video quality and file size

        * `NONE`
          Constant Bitrate – Configure constant bit rate, rather than constant output quality.
        * `LOSSLESS`
          Lossless.
        * `PERC_LOSSLESS`
          Perceptually Lossless.
        * `HIGH`
          High Quality.
        * `MEDIUM`
          Medium Quality.
        * `LOW`
          Low Quality.
        * `VERYLOW`
          Very Low Quality.
        * `LOWEST`
          Lowest Quality.

        Type:
        :   enum in [`'NONE'`, `'LOSSLESS'`, `'PERC_LOSSLESS'`, `'HIGH'`, `'MEDIUM'`, `'LOW'`, `'VERYLOW'`, `'LOWEST'`], default `'MEDIUM'`

    ffmpeg\_preset[¶](#bpy.types.FFmpegSettings.ffmpeg_preset "Link to this definition")
    :   Tradeoff between encoding speed and compression ratio

        * `BEST`
          Slowest – Recommended if you have lots of time and want the best compression efficiency.
        * `GOOD`
          Good – The default and recommended for most applications.
        * `REALTIME`
          Realtime – Recommended for fast encoding.

        Type:
        :   enum in [`'BEST'`, `'GOOD'`, `'REALTIME'`], default `'GOOD'`

    ffmpeg\_prores\_profile[¶](#bpy.types.FFmpegSettings.ffmpeg_prores_profile "Link to this definition")
    :   ProRes Profile

        Type:
        :   enum in [`'422_PROXY'`, `'422_LT'`, `'422_STD'`, `'422_HQ'`, `'4444'`, `'4444_XQ'`], default `'422_STD'`

    format[¶](#bpy.types.FFmpegSettings.format "Link to this definition")
    :   Output file container

        Type:
        :   enum in [`'MPEG4'`, `'MKV'`, `'WEBM'`, `'AVI'`, `'DV'`, `'FLASH'`, `'MPEG1'`, `'MPEG2'`, `'OGG'`, `'QUICKTIME'`], default `'MKV'`

    gopsize[¶](#bpy.types.FFmpegSettings.gopsize "Link to this definition")
    :   Distance between key frames, also known as GOP size; influences file size and seekability

        Type:
        :   int in [0, 500], default 25

    max\_b\_frames[¶](#bpy.types.FFmpegSettings.max_b_frames "Link to this definition")
    :   Maximum number of B-frames between non-B-frames; influences file size and seekability

        Type:
        :   int in [0, 16], default 0

    maxrate[¶](#bpy.types.FFmpegSettings.maxrate "Link to this definition")
    :   Rate control: max rate (kbit/s)

        Type:
        :   int in [-inf, inf], default 0

    minrate[¶](#bpy.types.FFmpegSettings.minrate "Link to this definition")
    :   Rate control: min rate (kbit/s)

        Type:
        :   int in [-inf, inf], default 0

    muxrate[¶](#bpy.types.FFmpegSettings.muxrate "Link to this definition")
    :   Mux rate (bits/second)

        Type:
        :   int in [0, inf], default 0

    packetsize[¶](#bpy.types.FFmpegSettings.packetsize "Link to this definition")
    :   Mux packet size (byte)

        Type:
        :   int in [0, 16384], default 0

    use\_autosplit[¶](#bpy.types.FFmpegSettings.use_autosplit "Link to this definition")
    :   Autosplit output at 2GB boundary

        Type:
        :   boolean, default False

    use\_lossless\_output[¶](#bpy.types.FFmpegSettings.use_lossless_output "Link to this definition")
    :   Use lossless output for video streams

        Type:
        :   boolean, default False

    use\_max\_b\_frames[¶](#bpy.types.FFmpegSettings.use_max_b_frames "Link to this definition")
    :   Set a maximum number of B-frames

        Type:
        :   boolean, default False

    video\_bitrate[¶](#bpy.types.FFmpegSettings.video_bitrate "Link to this definition")
    :   Video bitrate (kbit/s)

        Type:
        :   int in [-inf, inf], default 0
