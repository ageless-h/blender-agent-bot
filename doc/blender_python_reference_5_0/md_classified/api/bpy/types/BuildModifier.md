    frame\_start[¶](#bpy.types.BuildModifier.frame_start "Link to this definition")
    :   Start frame of the effect

        Type:
        :   float in [-1.04857e+06, 1.04857e+06], default 1.0

    seed[¶](#bpy.types.BuildModifier.seed "Link to this definition")
    :   Seed for random if used

        Type:
        :   int in [1, 1048574], default 0

    use\_random\_order[¶](#bpy.types.BuildModifier.use_random_order "Link to this definition")
    :   Randomize the faces or edges during build

        Type:
        :   boolean, default False

    use\_reverse[¶](#bpy.types.BuildModifier.use_reverse "Link to this definition")
    :   Deconstruct the mesh instead of building it

        Type:
        :   boolean, default False
