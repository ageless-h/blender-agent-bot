*class* bpy.types.ActionConstraint(*Constraint*)[¶](#bpy.types.ActionConstraint "Link to this definition")
:   Map an action to the transform axes of a bone

    action[¶](#bpy.types.ActionConstraint.action "Link to this definition")
    :   The constraining action

        Type:
        :   [`Action`](Action.md#bpy.types.Action "bpy.types.Action")

    action\_slot[¶](#bpy.types.ActionConstraint.action_slot "Link to this definition")
    :   The slot identifies which sub-set of the Action is considered to be for this strip, and its name is used to find the right slot when assigning another Action

        Type:
        :   [`ActionSlot`](ActionSlot.md#bpy.types.ActionSlot "bpy.types.ActionSlot")

    action\_slot\_handle[¶](#bpy.types.ActionConstraint.action_slot_handle "Link to this definition")
    :   A number that identifies which sub-set of the Action is considered to be for this Action Constraint

        Type:
        :   int in [-inf, inf], default 0

    action\_suitable\_slots[¶](#bpy.types.ActionConstraint.action_suitable_slots "Link to this definition")
    :   The list of action slots suitable for this NLA strip

        Type:
        :   [`bpy_prop_collection`](bpy_prop_collection.md#bpy.types.bpy_prop_collection "bpy.types.bpy_prop_collection") of [`ActionSlot`](ActionSlot.md#bpy.types.ActionSlot "bpy.types.ActionSlot"), (readonly)

    eval\_time[¶](#bpy.types.ActionConstraint.eval_time "Link to this definition")
    :   Interpolates between Action Start and End frames

        Type:
        :   float in [0, 1], default 0.0

    frame\_end[¶](#bpy.types.ActionConstraint.frame_end "Link to this definition")
    :   Last frame of the Action to use

        Type:
        :   int in [-1048574, 1048574], default 0

    frame\_start[¶](#bpy.types.ActionConstraint.frame_start "Link to this definition")
    :   First frame of the Action to use

        Type:
        :   int in [-1048574, 1048574], default 0

    last\_slot\_identifier[¶](#bpy.types.ActionConstraint.last_slot_identifier "Link to this definition")
    :   The identifier of the most recently assigned action slot. The slot identifies which sub-set of the Action is considered to be for this constraint, and its identifier is used to find the right slot when assigning an Action.

        Type:
        :   string, default “”, (never None)

    max[¶](#bpy.types.ActionConstraint.max "Link to this definition")
    :   Maximum value for target channel range

        Type:
        :   float in [-1000, 1000], default 0.0

    min[¶](#bpy.types.ActionConstraint.min "Link to this definition")
    :   Minimum value for target channel range

        Type:
        :   float in [-1000, 1000], default 0.0

    mix\_mode[¶](#bpy.types.ActionConstraint.mix_mode "Link to this definition")
    :   Specify how existing transformations and the action channels are combined

        * `REPLACE`
          Replace – Replace the original transformation with the action channels.
        * `BEFORE_FULL`
          Before Original (Full) – Apply the action channels before the original transformation, as if applied to an imaginary parent in Full Inherit Scale mode. Will create shear when combining rotation and non-uniform scale..
        * `BEFORE`
          Before Original (Aligned) – Apply the action channels before the original transformation, as if applied to an imaginary parent in Aligned Inherit Scale mode. This effectively uses Full for location and Split Channels for rotation and scale..
        * `BEFORE_SPLIT`
          Before Original (Split Channels) – Apply the action channels before the original transformation, handling location, rotation and scale separately.
        * `AFTER_FULL`
          After Original (Full) – Apply the action channels after the original transformation, as if applied to an imaginary child in Full Inherit Scale mode. Will create shear when combining rotation and non-uniform scale..
        * `AFTER`
          After Original (Aligned) – Apply the action channels after the original transformation, as if applied to an imaginary child in Aligned Inherit Scale mode. This effectively uses Full for location and Split Channels for rotation and scale..
        * `AFTER_SPLIT`
          After Original (Split Channels) – Apply the action channels after the original transformation, handling location, rotation and scale separately.

        Type:
        :   enum in [`'REPLACE'`, `'BEFORE_FULL'`, `'BEFORE'`, `'BEFORE_SPLIT'`, `'AFTER_FULL'`, `'AFTER'`, `'AFTER_SPLIT'`], default `'AFTER_FULL'`

    subtarget[¶](#bpy.types.ActionConstraint.subtarget "Link to this definition")
    :   Armature bone, mesh or lattice vertex group, …

        Type:
        :   string, default “”, (never None)

    target[¶](#bpy.types.ActionConstraint.target "Link to this definition")
    :   Target object

        Type:
        :   [`Object`](Object.md#bpy.types.Object "bpy.types.Object")

    transform\_channel[¶](#bpy.types.ActionConstraint.transform_channel "Link to this definition")
    :   Transformation channel from the target that is used to key the Action

        Type:
        :   enum in [`'LOCATION_X'`, `'LOCATION_Y'`, `'LOCATION_Z'`, `'ROTATION_X'`, `'ROTATION_Y'`, `'ROTATION_Z'`, `'SCALE_X'`, `'SCALE_Y'`, `'SCALE_Z'`], default `'ROTATION_X'`

    use\_bone\_object\_action[¶](#bpy.types.ActionConstraint.use_bone_object_action "Link to this definition")
    :   Bones only: apply the object’s transformation channels of the action to the constrained bone, instead of bone’s channels

        Type:
        :   boolean, default False

    use\_eval\_time[¶](#bpy.types.ActionConstraint.use_eval_time "Link to this definition")
    :   Interpolate between Action Start and End frames, with the Evaluation Time slider instead of the Target object/bone

        Type:
        :   boolean, default False
