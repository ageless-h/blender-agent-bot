# LightProbeVolume(LightProbe)[¶](#lightprobevolume-lightprobe "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`ID`](ID.md#bpy.types.ID "bpy.types.ID"), [`LightProbe`](LightProbe.md#bpy.types.LightProbe "bpy.types.LightProbe")

*class* bpy.types.LightProbeVolume(*LightProbe*)[¶](#bpy.types.LightProbeVolume "Link to this definition")
:   Light probe that captures low frequency lighting inside a volume

    bake\_samples[¶](#bpy.types.LightProbeVolume.bake_samples "Link to this definition")
    :   Number of ray directions to evaluate when baking

        Type:
        :   int in [1, inf], default 2048

    capture\_distance[¶](#bpy.types.LightProbeVolume.capture_distance "Link to this definition")
    :   Distance around the probe volume that will be considered during the bake

        Type:
        :   float in [1e-06, inf], default 20.0

    capture\_emission[¶](#bpy.types.LightProbeVolume.capture_emission "Link to this definition")
    :   Bake emissive surfaces for more accurate lighting

        Type:
        :   boolean, default True

    capture\_indirect[¶](#bpy.types.LightProbeVolume.capture_indirect "Link to this definition")
    :   Bake light bounces from light sources for more accurate lighting

        Type:
        :   boolean, default True

    capture\_world[¶](#bpy.types.LightProbeVolume.capture_world "Link to this definition")
    :   Bake incoming light from the world instead of just the visibility for more accurate lighting, but lose correct blending to surrounding irradiance volumes

        Type:
        :   boolean, default False

    clamp\_direct[¶](#bpy.types.LightProbeVolume.clamp_direct "Link to this definition")
    :   Clamp the direct lighting intensity to reduce noise (0 to disable)

        Type:
        :   float in [0, inf], default 0.0

    clamp\_indirect[¶](#bpy.types.LightProbeVolume.clamp_indirect "Link to this definition")
    :   Clamp the indirect lighting intensity to reduce noise (0 to disable)

        Type:
        :   float in [0, inf], default 10.0

    dilation\_radius[¶](#bpy.types.LightProbeVolume.dilation_radius "Link to this definition")
    :   Radius in grid sample to search valid grid samples to copy into invalid grid samples

        Type:
        :   float in [1, 5], default 1.0

    dilation\_threshold[¶](#bpy.types.LightProbeVolume.dilation_threshold "Link to this definition")
    :   Ratio of front-facing surface hits under which a grid sample will reuse neighbors grid sample lighting

        Type:
        :   float in [0, 1], default 0.5

    escape\_bias[¶](#bpy.types.LightProbeVolume.escape_bias "Link to this definition")
    :   Distance to search for valid capture positions to prevent lighting artifacts

        Type:
        :   float in [0, 1], default 0.1

    facing\_bias[¶](#bpy.types.LightProbeVolume.facing_bias "Link to this definition")
    :   Smoother irradiance interpolation but introduce light bleeding

        Type:
        :   float in [0, inf], default 0.5

    intensity[¶](#bpy.types.LightProbeVolume.intensity "Link to this definition")
    :   Modify the intensity of the lighting captured by this probe

        Type:
        :   float in [0, inf], default 1.0

    normal\_bias[¶](#bpy.types.LightProbeVolume.normal_bias "Link to this definition")
    :   Offset sampling of the irradiance grid in the surface normal direction to reduce light bleeding

        Type:
        :   float in [0, inf], default 0.3

    resolution\_x[¶](#bpy.types.LightProbeVolume.resolution_x "Link to this definition")
    :   Number of samples along the x axis of the volume

        Type:
        :   int in [1, 256], default 4

    resolution\_y[¶](#bpy.types.LightProbeVolume.resolution_y "Link to this definition")
    :   Number of samples along the y axis of the volume

        Type:
        :   int in [1, 256], default 4

    resolution\_z[¶](#bpy.types.LightProbeVolume.resolution_z "Link to this definition")
    :   Number of samples along the z axis of the volume

        Type:
        :   int in [1, 256], default 4

    surface\_bias[¶](#bpy.types.LightProbeVolume.surface_bias "Link to this definition")
    :   Moves capture points away from surfaces to prevent artifacts

        Type:
        :   float in [0, 1], default 0.05

    surfel\_density[¶](#bpy.types.LightProbeVolume.surfel_density "Link to this definition")
    :   Number of surfels to spawn in one local unit distance (higher values improve quality)

        Type:
        :   int in [1, inf], default 20

    validity\_threshold[¶](#bpy.types.LightProbeVolume.validity_threshold "Link to this definition")
    :   Ratio of front-facing surface hits under which a grid sample will not be considered for lighting

        Type:
        :   float in [0, 1], default 0.4

    view\_bias[¶](#bpy.types.LightProbeVolume.view_bias "Link to this definition")
    :   Offset sampling of the irradiance grid in the viewing direction to reduce light bleeding

        Type:
        :   float in [0, inf], default 0.0
