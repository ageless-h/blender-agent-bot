# World(ID)[¶](#world-id "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`ID`](ID.md#bpy.types.ID "bpy.types.ID")

*class* bpy.types.World(*ID*)[¶](#bpy.types.World "Link to this definition")
:   World data-block describing the environment and ambient lighting of a scene

    animation\_data[¶](#bpy.types.World.animation_data "Link to this definition")
    :   Animation data for this data-block

        Type:
        :   [`AnimData`](AnimData.md#bpy.types.AnimData "bpy.types.AnimData"), (readonly)

    color[¶](#bpy.types.World.color "Link to this definition")
    :   Color of the background

        Type:
        :   [`mathutils.Color`](../../mathutils/index.md#mathutils.Color "mathutils.Color") of 3 items in [0, inf], default (0.05, 0.05, 0.05)

    cycles[¶](#bpy.types.World.cycles "Link to this definition")
    :   Cycles world settings

        Type:
        :   `CyclesWorldSettings`, (readonly)

    cycles\_visibility[¶](#bpy.types.World.cycles_visibility "Link to this definition")
    :   Cycles visibility settings

        Type:
        :   `CyclesVisibilitySettings`, (readonly)

    light\_settings[¶](#bpy.types.World.light_settings "Link to this definition")
    :   World lighting settings

        Type:
        :   [`WorldLighting`](WorldLighting.md#bpy.types.WorldLighting "bpy.types.WorldLighting"), (readonly, never None)

    lightgroup[¶](#bpy.types.World.lightgroup "Link to this definition")
    :   Lightgroup that the world belongs to

        Type:
        :   string, default “”, (never None)

    mist\_settings[¶](#bpy.types.World.mist_settings "Link to this definition")
    :   World mist settings

        Type:
        :   [`WorldMistSettings`](WorldMistSettings.md#bpy.types.WorldMistSettings "bpy.types.WorldMistSettings"), (readonly, never None)

    node\_tree[¶](#bpy.types.World.node_tree "Link to this definition")
    :   Node tree for node based worlds

        Type:
        :   [`NodeTree`](NodeTree.md#bpy.types.NodeTree "bpy.types.NodeTree"), (readonly)

    probe\_resolution[¶](#bpy.types.World.probe_resolution "Link to this definition")
    :   Resolution when baked to a texture

        Type:
        :   enum in [`'128'`, `'256'`, `'512'`, `'1024'`, `'2048'`, `'4096'`], default `'1024'`

    sun\_angle[¶](#bpy.types.World.sun_angle "Link to this definition")
    :   Angular diameter of the Sun as seen from the Earth

        Type:
        :   float in [0, 3.14159], default 0.00918043

    sun\_shadow\_filter\_radius[¶](#bpy.types.World.sun_shadow_filter_radius "Link to this definition")
    :   Blur shadow aliasing using Percentage Closer Filtering

        Type:
        :   float in [0, inf], default 1.0

    sun\_shadow\_jitter\_overblur[¶](#bpy.types.World.sun_shadow_jitter_overblur "Link to this definition")
    :   Apply shadow tracing to each jittered sample to reduce under-sampling artifacts

        Type:
        :   float in [0, 100], default 10.0

    sun\_shadow\_maximum\_resolution[¶](#bpy.types.World.sun_shadow_maximum_resolution "Link to this definition")
    :   Maximum size of a shadow map pixel. Higher values use less memory at the cost of shadow quality.

        Type:
        :   float in [0, inf], default 0.001

    sun\_threshold[¶](#bpy.types.World.sun_threshold "Link to this definition")
    :   If non-zero, the maximum value for world contribution that will be recorded inside the world light probe. The excess contribution is converted to a sun light. This reduces the light bleeding caused by very bright light sources.

        Type:
        :   float in [0, inf], default 10.0

    use\_eevee\_finite\_volume[¶](#bpy.types.World.use_eevee_finite_volume "Link to this definition")
    :   The world’s volume used to be rendered by EEVEE Legacy. Conversion is needed for it to render properly.

        Type:
        :   boolean, default False

    use\_nodes[¶](#bpy.types.World.use_nodes "Link to this definition")
    :   Use shader nodes to render the world

        Deprecated since version 5.0: removal planned in version 6.0

        Unused but kept for compatibility reasons. Setting the property has no effect, and getting it always returns True.

        Type:
        :   boolean, default False

    use\_sun\_shadow[¶](#bpy.types.World.use_sun_shadow "Link to this definition")
    :   Enable sun shadow casting

        Type:
        :   boolean, default True

    use\_sun\_shadow\_jitter[¶](#bpy.types.World.use_sun_shadow_jitter "Link to this definition")
    :   Enable jittered soft shadows to increase shadow precision (disabled in viewport unless enabled in the render settings). Has a high performance impact.

        Type:
        :   boolean, default False

    inline\_shader\_nodes()[¶](#bpy.types.World.inline_shader_nodes "Link to this definition")
    :   Get the inlined shader nodes of this world. This preprocesses the node tree
        to remove nested groups, repeat zones and more.

        Returns:
        :   The inlined shader nodes.

        Return type:
        :   [`bpy.types.InlineShaderNodes`](InlineShaderNodes.md#bpy.types.InlineShaderNodes "bpy.types.InlineShaderNodes")

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.World.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass
