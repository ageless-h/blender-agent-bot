# PreferencesInput(bpy\_struct)[¶](#preferencesinput-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.PreferencesInput(*bpy\_struct*)[¶](#bpy.types.PreferencesInput "Link to this definition")
:   Settings for input devices

    drag\_threshold[¶](#bpy.types.PreferencesInput.drag_threshold "Link to this definition")
    :   Number of pixels to drag before a drag event is triggered for keyboard and other non mouse/tablet input (otherwise click events are detected)

        Type:
        :   int in [1, 255], default 30

    drag\_threshold\_mouse[¶](#bpy.types.PreferencesInput.drag_threshold_mouse "Link to this definition")
    :   Number of pixels to drag before a drag event is triggered for mouse/trackpad input (otherwise click events are detected)

        Type:
        :   int in [1, 255], default 3

    drag\_threshold\_tablet[¶](#bpy.types.PreferencesInput.drag_threshold_tablet "Link to this definition")
    :   Number of pixels to drag before a drag event is triggered for tablet input (otherwise click events are detected)

        Type:
        :   int in [1, 255], default 10

    invert\_mouse\_zoom[¶](#bpy.types.PreferencesInput.invert_mouse_zoom "Link to this definition")
    :   Invert the axis of mouse movement for zooming

        Type:
        :   boolean, default False

    invert\_zoom\_wheel[¶](#bpy.types.PreferencesInput.invert_zoom_wheel "Link to this definition")
    :   Swap the Mouse Wheel zoom direction

        Type:
        :   boolean, default False

    mouse\_double\_click\_time[¶](#bpy.types.PreferencesInput.mouse_double_click_time "Link to this definition")
    :   Time/delay (in ms) for a double click

        Type:
        :   int in [1, 1000], default 350

    mouse\_emulate\_3\_button\_modifier[¶](#bpy.types.PreferencesInput.mouse_emulate_3_button_modifier "Link to this definition")
    :   Hold this modifier to emulate the middle mouse button

        Type:
        :   enum in [`'ALT'`, `'OSKEY'`], default `'ALT'`

    move\_threshold[¶](#bpy.types.PreferencesInput.move_threshold "Link to this definition")
    :   Number of pixels to before the cursor is considered to have moved (used for cycling selected items on successive clicks)

        Type:
        :   int in [0, 255], default 2

    navigation\_mode[¶](#bpy.types.PreferencesInput.navigation_mode "Link to this definition")
    :   Which method to use for viewport navigation

        Type:
        :   enum in [Navigation Mode Items](../../../appendix/bpy_types_enum_items/navigation_mode_items.md#rna-enum-navigation-mode-items), default `'WALK'`

    ndof\_deadzone[¶](#bpy.types.PreferencesInput.ndof_deadzone "Link to this definition")
    :   Threshold of initial movement needed from the device’s rest position

        Type:
        :   float in [0, 1], default 0.0

    ndof\_fly\_helicopter[¶](#bpy.types.PreferencesInput.ndof_fly_helicopter "Link to this definition")
    :   Device up/down directly controls the Z position of the 3D viewport

        Type:
        :   boolean, default False

    ndof\_fly\_speed\_auto[¶](#bpy.types.PreferencesInput.ndof_fly_speed_auto "Link to this definition")
    :   Automatically adjusts fly navigation speed based on the distance of objects near the center of the viewport, making it easier to navigate complex scenes. Speed is recalculated each time movement starts.

        Type:
        :   boolean, default False

    ndof\_lock\_camera\_pan\_zoom[¶](#bpy.types.PreferencesInput.ndof_lock_camera_pan_zoom "Link to this definition")
    :   Pan/zoom the camera view instead of leaving the camera view when orbiting

        Type:
        :   boolean, default True

    ndof\_lock\_horizon[¶](#bpy.types.PreferencesInput.ndof_lock_horizon "Link to this definition")
    :   Lock Horizon forces the horizon to be kept leveled as it currently is

        Type:
        :   boolean, default True

    ndof\_navigation\_mode[¶](#bpy.types.PreferencesInput.ndof_navigation_mode "Link to this definition")
    :   3D Mouse Navigation Mode

        * `OBJECT`
          Object – This mode is like reaching into the screen and holding the model in your hand. Push the 3D Mouse cap left, and the model moves left. Push right and the model moves right.
        * `FLY`
          Fly – Enables using the 3D Mouse as if it is a camera. Push into the scene and the camera moves forward into the scene. You are entering the scene as if flying around in it. This also inverts pan & zoom for 2D views.

        Type:
        :   enum in [`'OBJECT'`, `'FLY'`], default `'OBJECT'`

    ndof\_orbit\_center\_auto[¶](#bpy.types.PreferencesInput.ndof_orbit_center_auto "Link to this definition")
    :   Auto sets the orbit center dynamically. When the complete model is in view, the center of volume of the whole model is used as the rotation point. When you move closer, the orbit center will be set on an object close to your center of the view.

        Type:
        :   boolean, default True

    ndof\_orbit\_center\_selected[¶](#bpy.types.PreferencesInput.ndof_orbit_center_selected "Link to this definition")
    :   Selected Item forces the orbit center to only take the currently selected objects into account.

        Type:
        :   boolean, default False

    ndof\_panx\_invert\_axis[¶](#bpy.types.PreferencesInput.ndof_panx_invert_axis "Link to this definition")
    :   Type:
        :   boolean, default False

    ndof\_pany\_invert\_axis[¶](#bpy.types.PreferencesInput.ndof_pany_invert_axis "Link to this definition")
    :   Type:
        :   boolean, default False

    ndof\_panz\_invert\_axis[¶](#bpy.types.PreferencesInput.ndof_panz_invert_axis "Link to this definition")
    :   Type:
        :   boolean, default False

    ndof\_rotation\_sensitivity[¶](#bpy.types.PreferencesInput.ndof_rotation_sensitivity "Link to this definition")
    :   Overall sensitivity of the 3D Mouse for rotation

        Type:
        :   float in [0.01, 40], default 4.0

    ndof\_rotx\_invert\_axis[¶](#bpy.types.PreferencesInput.ndof_rotx_invert_axis "Link to this definition")
    :   Type:
        :   boolean, default False

    ndof\_roty\_invert\_axis[¶](#bpy.types.PreferencesInput.ndof_roty_invert_axis "Link to this definition")
    :   Type:
        :   boolean, default False

    ndof\_rotz\_invert\_axis[¶](#bpy.types.PreferencesInput.ndof_rotz_invert_axis "Link to this definition")
    :   Type:
        :   boolean, default False

    ndof\_show\_guide\_orbit\_axis[¶](#bpy.types.PreferencesInput.ndof_show_guide_orbit_axis "Link to this definition")
    :   Display the center and axis during rotation

        Type:
        :   boolean, default False

    ndof\_show\_guide\_orbit\_center[¶](#bpy.types.PreferencesInput.ndof_show_guide_orbit_center "Link to this definition")
    :   Display the orbit center during rotation

        Type:
        :   boolean, default True

    ndof\_translation\_sensitivity[¶](#bpy.types.PreferencesInput.ndof_translation_sensitivity "Link to this definition")
    :   Overall sensitivity of the 3D Mouse for translation

        Type:
        :   float in [0.01, 40], default 4.0

    ndof\_zoom\_direction[¶](#bpy.types.PreferencesInput.ndof_zoom_direction "Link to this definition")
    :   Which axis of the 3D Mouse cap zooms the view

        * `NDOF_ZOOM_FORWARD`
          Forward/Backward – Zoom by pulling the 3D Mouse cap upwards or pushing the cap downwards.
        * `NDOF_ZOOM_UP`
          Up/Down – Zoom by pulling the 3D Mouse cap upwards or pushing the cap downwards.

        Type:
        :   enum in [`'NDOF_ZOOM_FORWARD'`, `'NDOF_ZOOM_UP'`], default `'NDOF_ZOOM_FORWARD'`

    pressure\_softness[¶](#bpy.types.PreferencesInput.pressure_softness "Link to this definition")
    :   Adjusts softness of the low pressure response onset using a gamma curve

        Type:
        :   float in [-inf, inf], default 0.0

    pressure\_threshold\_max[¶](#bpy.types.PreferencesInput.pressure_threshold_max "Link to this definition")
    :   Raw input pressure value that is interpreted as 100% by Blender

        Type:
        :   float in [0, 1], default 1.0

    tablet\_api[¶](#bpy.types.PreferencesInput.tablet_api "Link to this definition")
    :   Select the tablet API to use for pressure sensitivity (may require restarting Blender for changes to take effect)

        * `AUTOMATIC`
          Automatic – Automatically choose Wintab or Windows Ink depending on the device.
        * `WINDOWS_INK`
          Windows Ink – Use native Windows Ink API, for modern tablet and pen devices. Requires Windows 8 or newer..
        * `WINTAB`
          Wintab – Use Wintab driver for older tablets and Windows versions.

        Type:
        :   enum in [`'AUTOMATIC'`, `'WINDOWS_INK'`, `'WINTAB'`], default `'AUTOMATIC'`

    touchpad\_scroll\_direction[¶](#bpy.types.PreferencesInput.touchpad_scroll_direction "Link to this definition")
    :   Scroll direction (Wayland only)

        * `TRADITIONAL`
          Traditional – Traditional scroll direction.
        * `NATURAL`
          Natural – Natural scroll direction.

        Type:
        :   enum in [`'TRADITIONAL'`, `'NATURAL'`], default `'TRADITIONAL'`

    use\_auto\_perspective[¶](#bpy.types.PreferencesInput.use_auto_perspective "Link to this definition")
    :   Automatically switch between orthographic and perspective when changing from top/front/side views

        Type:
        :   boolean, default True

    use\_drag\_immediately[¶](#bpy.types.PreferencesInput.use_drag_immediately "Link to this definition")
    :   Moving things with a mouse drag confirms when releasing the button

        Type:
        :   boolean, default True

    use\_emulate\_numpad[¶](#bpy.types.PreferencesInput.use_emulate_numpad "Link to this definition")
    :   Main 1 to 0 keys act as the numpad ones (useful for laptops)

        Type:
        :   boolean, default False

    use\_mouse\_continuous[¶](#bpy.types.PreferencesInput.use_mouse_continuous "Link to this definition")
    :   Let the mouse wrap around the view boundaries so mouse movements are not limited by the screen size (used by transform, dragging of UI controls, etc.)

        Type:
        :   boolean, default True

    use\_mouse\_depth\_navigate[¶](#bpy.types.PreferencesInput.use_mouse_depth_navigate "Link to this definition")
    :   Use the depth under the mouse to improve view pan/rotate/zoom functionality

        Type:
        :   boolean, default False

    use\_mouse\_emulate\_3\_button[¶](#bpy.types.PreferencesInput.use_mouse_emulate_3_button "Link to this definition")
    :   Emulate Middle Mouse with Alt+Left Mouse

        Type:
        :   boolean, default False

    use\_multitouch\_gestures[¶](#bpy.types.PreferencesInput.use_multitouch_gestures "Link to this definition")
    :   Use multi-touch gestures for navigation with touchpad, instead of scroll wheel emulation

        Type:
        :   boolean, default True

    use\_numeric\_input\_advanced[¶](#bpy.types.PreferencesInput.use_numeric_input_advanced "Link to this definition")
    :   When entering numbers while transforming, default to advanced mode for full math expression evaluation

        Type:
        :   boolean, default False

    use\_rotate\_around\_active[¶](#bpy.types.PreferencesInput.use_rotate_around_active "Link to this definition")
    :   Use selection as the pivot point

        Type:
        :   boolean, default False

    use\_zoom\_to\_mouse[¶](#bpy.types.PreferencesInput.use_zoom_to_mouse "Link to this definition")
    :   Zoom in towards the mouse pointer’s position in the 3D view, rather than the 2D window center

        Type:
        :   boolean, default False

    view\_rotate\_method[¶](#bpy.types.PreferencesInput.view_rotate_method "Link to this definition")
    :   Orbit method in the viewport

        * `TURNTABLE`
          Turntable – Turntable keeps the Z-axis upright while orbiting.
        * `TRACKBALL`
          Trackball – Trackball allows you to tumble your view at any angle.

        Type:
        :   enum in [`'TURNTABLE'`, `'TRACKBALL'`], default `'TURNTABLE'`

    view\_rotate\_sensitivity\_trackball[¶](#bpy.types.PreferencesInput.view_rotate_sensitivity_trackball "Link to this definition")
    :   Scale trackball orbit sensitivity

        Type:
        :   float in [0.1, 10], default 1.0

    view\_rotate\_sensitivity\_turntable[¶](#bpy.types.PreferencesInput.view_rotate_sensitivity_turntable "Link to this definition")
    :   Rotation amount per pixel to control how fast the viewport orbits

        Type:
        :   float in [1.74533e-05, 0.261799], default 0.00698132

    view\_zoom\_axis[¶](#bpy.types.PreferencesInput.view_zoom_axis "Link to this definition")
    :   Axis of mouse movement to zoom in or out on

        * `VERTICAL`
          Vertical – Zoom in and out based on vertical mouse movement.
        * `HORIZONTAL`
          Horizontal – Zoom in and out based on horizontal mouse movement.

        Type:
        :   enum in [`'VERTICAL'`, `'HORIZONTAL'`], default `'VERTICAL'`

    view\_zoom\_method[¶](#bpy.types.PreferencesInput.view_zoom_method "Link to this definition")
    :   Which style to use for viewport scaling

        * `CONTINUE`
          Continue – Continuous zooming. The zoom direction and speed depends on how far along the set Zoom Axis the mouse has moved..
        * `DOLLY`
          Dolly – Zoom in and out based on mouse movement along the set Zoom Axis.
        * `SCALE`
          Scale – Zoom in and out as if you are scaling the view, mouse movements relative to center.

        Type:
        :   enum in [`'CONTINUE'`, `'DOLLY'`, `'SCALE'`], default `'DOLLY'`

    walk\_navigation[¶](#bpy.types.PreferencesInput.walk_navigation "Link to this definition")
    :   Settings for walk navigation mode

        Type:
        :   [`WalkNavigation`](WalkNavigation.md#bpy.types.WalkNavigation "bpy.types.WalkNavigation"), (readonly, never None)

    xr\_navigation[¶](#bpy.types.PreferencesInput.xr_navigation "Link to this definition")
    :   Settings for navigation in XR

        Type:
        :   [`XrNavigation`](XrNavigation.md#bpy.types.XrNavigation "bpy.types.XrNavigation"), (readonly, never None)
