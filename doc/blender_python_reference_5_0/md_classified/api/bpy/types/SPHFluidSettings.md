# SPHFluidSettings(bpy\_struct)[¶](#sphfluidsettings-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.SPHFluidSettings(*bpy\_struct*)[¶](#bpy.types.SPHFluidSettings "Link to this definition")
:   Settings for particle fluids physics

    buoyancy[¶](#bpy.types.SPHFluidSettings.buoyancy "Link to this definition")
    :   Artificial buoyancy force in negative gravity direction based on pressure differences inside the fluid

        Type:
        :   float in [0, 10], default 0.0

    fluid\_radius[¶](#bpy.types.SPHFluidSettings.fluid_radius "Link to this definition")
    :   Fluid interaction radius

        Type:
        :   float in [0, 20], default 0.0

    linear\_viscosity[¶](#bpy.types.SPHFluidSettings.linear_viscosity "Link to this definition")
    :   Linear viscosity

        Type:
        :   float in [0, 100], default 0.0

    plasticity[¶](#bpy.types.SPHFluidSettings.plasticity "Link to this definition")
    :   How much the spring rest length can change after the elastic limit is crossed

        Type:
        :   float in [0, 100], default 0.0

    repulsion[¶](#bpy.types.SPHFluidSettings.repulsion "Link to this definition")
    :   How strongly the fluid tries to keep from clustering (factor of stiffness)

        Type:
        :   float in [0, 100], default 0.0

    rest\_density[¶](#bpy.types.SPHFluidSettings.rest_density "Link to this definition")
    :   Fluid rest density

        Type:
        :   float in [0, 10000], default 0.0

    rest\_length[¶](#bpy.types.SPHFluidSettings.rest_length "Link to this definition")
    :   Spring rest length (factor of particle radius)

        Type:
        :   float in [0, 2], default 0.0

    solver[¶](#bpy.types.SPHFluidSettings.solver "Link to this definition")
    :   The code used to calculate internal forces on particles

        * `DDR`
          Double-Density – An artistic solver with strong surface tension effects (original).
        * `CLASSICAL`
          Classical – A more physically-accurate solver.

        Type:
        :   enum in [`'DDR'`, `'CLASSICAL'`], default `'DDR'`

    spring\_force[¶](#bpy.types.SPHFluidSettings.spring_force "Link to this definition")
    :   Spring force

        Type:
        :   float in [0, 100], default 0.0

    spring\_frames[¶](#bpy.types.SPHFluidSettings.spring_frames "Link to this definition")
    :   Create springs for this number of frames since particles birth (0 is always)

        Type:
        :   int in [0, 100], default 0

    stiff\_viscosity[¶](#bpy.types.SPHFluidSettings.stiff_viscosity "Link to this definition")
    :   Creates viscosity for expanding fluid

        Type:
        :   float in [0, 100], default 0.0

    stiffness[¶](#bpy.types.SPHFluidSettings.stiffness "Link to this definition")
    :   How incompressible the fluid is (speed of sound)

        Type:
        :   float in [0, 1000], default 0.0

    use\_factor\_density[¶](#bpy.types.SPHFluidSettings.use_factor_density "Link to this definition")
    :   Density is calculated as a factor of default density (depends on particle size)

        Type:
        :   boolean, default False

    use\_factor\_radius[¶](#bpy.types.SPHFluidSettings.use_factor_radius "Link to this definition")
    :   Interaction radius is a factor of 4 \* particle size

        Type:
        :   boolean, default False

    use\_factor\_repulsion[¶](#bpy.types.SPHFluidSettings.use_factor_repulsion "Link to this definition")
    :   Repulsion is a factor of stiffness

        Type:
        :   boolean, default False

    use\_factor\_rest\_length[¶](#bpy.types.SPHFluidSettings.use_factor_rest_length "Link to this definition")
    :   Spring rest length is a factor of 2 \* particle size

        Type:
        :   boolean, default False

    use\_factor\_stiff\_viscosity[¶](#bpy.types.SPHFluidSettings.use_factor_stiff_viscosity "Link to this definition")
    :   Stiff viscosity is a factor of normal viscosity

        Type:
        :   boolean, default False

    use\_initial\_rest\_length[¶](#bpy.types.SPHFluidSettings.use_initial_rest_length "Link to this definition")
    :   Use the initial length as spring rest length instead of 2 \* particle size

        Type:
        :   boolean, default False

    use\_viscoelastic\_springs[¶](#bpy.types.SPHFluidSettings.use_viscoelastic_springs "Link to this definition")
    :   Use viscoelastic springs instead of Hooke’s springs

        Type:
        :   boolean, default False

    yield\_ratio[¶](#bpy.types.SPHFluidSettings.yield_ratio "Link to this definition")
    :   How much the spring has to be stretched/compressed in order to change its rest length

        Type:
        :   float in [0, 1], default 0.0

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.SPHFluidSettings.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.
