# CurvePaintSettings(bpy\_struct)[¶](#curvepaintsettings-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.CurvePaintSettings(*bpy\_struct*)[¶](#bpy.types.CurvePaintSettings "Link to this definition")
:   corner\_angle[¶](#bpy.types.CurvePaintSettings.corner_angle "Link to this definition")
    :   Angles above this are considered corners

        Type:
        :   float in [0, 3.14159], default 1.22173

    curve\_type[¶](#bpy.types.CurvePaintSettings.curve_type "Link to this definition")
    :   Type of curve to use for new strokes

        Type:
        :   enum in [`'POLY'`, `'BEZIER'`], default `'BEZIER'`

    depth\_mode[¶](#bpy.types.CurvePaintSettings.depth_mode "Link to this definition")
    :   Method of projecting depth

        Type:
        :   enum in [`'CURSOR'`, `'SURFACE'`], default `'CURSOR'`

    error\_threshold[¶](#bpy.types.CurvePaintSettings.error_threshold "Link to this definition")
    :   Allow deviation for a smoother, less precise line

        Type:
        :   int in [1, 100], default 8

    fit\_method[¶](#bpy.types.CurvePaintSettings.fit_method "Link to this definition")
    :   Curve fitting method

        Type:
        :   enum in [Curve Fit Method Items](../../../appendix/bpy_types_enum_items/curve_fit_method_items.md#rna-enum-curve-fit-method-items), default `'REFIT'`

    radius\_max[¶](#bpy.types.CurvePaintSettings.radius_max "Link to this definition")
    :   Radius to use when the maximum pressure is applied (or when a tablet isn’t used)

        Type:
        :   float in [0, 100], default 1.0

    radius\_min[¶](#bpy.types.CurvePaintSettings.radius_min "Link to this definition")
    :   Minimum radius when the minimum pressure is applied (also the minimum when tapering)

        Type:
        :   float in [0, 100], default 0.0

    radius\_taper\_end[¶](#bpy.types.CurvePaintSettings.radius_taper_end "Link to this definition")
    :   Taper factor for the radius of each point along the curve

        Type:
        :   float in [0, 10], default 0.0

    radius\_taper\_start[¶](#bpy.types.CurvePaintSettings.radius_taper_start "Link to this definition")
    :   Taper factor for the radius of each point along the curve

        Type:
        :   float in [0, 1], default 0.0

    surface\_offset[¶](#bpy.types.CurvePaintSettings.surface_offset "Link to this definition")
    :   Offset the stroke from the surface

        Type:
        :   float in [-10, 10], default 0.0

    surface\_plane[¶](#bpy.types.CurvePaintSettings.surface_plane "Link to this definition")
    :   Plane for projected stroke

        * `NORMAL_VIEW`
          Normal to Surface – Draw in a plane perpendicular to the surface.
        * `NORMAL_SURFACE`
          Tangent to Surface – Draw in the surface plane.
        * `VIEW`
          View – Draw in a plane aligned to the viewport.

        Type:
        :   enum in [`'NORMAL_VIEW'`, `'NORMAL_SURFACE'`, `'VIEW'`], default `'NORMAL_VIEW'`

    use\_corners\_detect[¶](#bpy.types.CurvePaintSettings.use_corners_detect "Link to this definition")
    :   Detect corners and use non-aligned handles

        Type:
        :   boolean, default True

    use\_offset\_absolute[¶](#bpy.types.CurvePaintSettings.use_offset_absolute "Link to this definition")
    :   Apply a fixed offset (don’t scale by the radius)

        Type:
        :   boolean, default False

    use\_pressure\_radius[¶](#bpy.types.CurvePaintSettings.use_pressure_radius "Link to this definition")
    :   Map tablet pressure to curve radius

        Type:
        :   boolean, default False

    use\_project\_only\_selected[¶](#bpy.types.CurvePaintSettings.use_project_only_selected "Link to this definition")
    :   Project the strokes only onto selected objects

        Type:
        :   boolean, default False

    use\_stroke\_endpoints[¶](#bpy.types.CurvePaintSettings.use_stroke_endpoints "Link to this definition")
    :   Use the start of the stroke for the depth

        Type:
        :   boolean, default False
