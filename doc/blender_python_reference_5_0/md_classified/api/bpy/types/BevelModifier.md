# BevelModifier(Modifier)[¶](#bevelmodifier-modifier "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`Modifier`](Modifier.md#bpy.types.Modifier "bpy.types.Modifier")

*class* bpy.types.BevelModifier(*Modifier*)[¶](#bpy.types.BevelModifier "Link to this definition")
:   Bevel modifier to make edges and vertices more rounded

    affect[¶](#bpy.types.BevelModifier.affect "Link to this definition")
    :   Affect edges or vertices

        * `VERTICES`
          Vertices – Affect only vertices.
        * `EDGES`
          Edges – Affect only edges.

        Type:
        :   enum in [`'VERTICES'`, `'EDGES'`], default `'EDGES'`

    angle\_limit[¶](#bpy.types.BevelModifier.angle_limit "Link to this definition")
    :   Angle above which to bevel edges

        Type:
        :   float in [0, 3.14159], default 0.523599

    custom\_profile[¶](#bpy.types.BevelModifier.custom_profile "Link to this definition")
    :   The path for the custom profile

        Type:
        :   [`CurveProfile`](CurveProfile.md#bpy.types.CurveProfile "bpy.types.CurveProfile"), (readonly)

    edge\_weight[¶](#bpy.types.BevelModifier.edge_weight "Link to this definition")
    :   Attribute name for edge weight

        Type:
        :   string, default “”, (never None)

    face\_strength\_mode[¶](#bpy.types.BevelModifier.face_strength_mode "Link to this definition")
    :   Whether to set face strength, and which faces to set it on

        * `FSTR_NONE`
          None – Do not set face strength.
        * `FSTR_NEW`
          New – Set face strength on new faces only.
        * `FSTR_AFFECTED`
          Affected – Set face strength on new and affected faces only.
        * `FSTR_ALL`
          All – Set face strength on all faces.

        Type:
        :   enum in [`'FSTR_NONE'`, `'FSTR_NEW'`, `'FSTR_AFFECTED'`, `'FSTR_ALL'`], default `'FSTR_NONE'`

    harden\_normals[¶](#bpy.types.BevelModifier.harden_normals "Link to this definition")
    :   Match normals of new faces to adjacent faces

        Type:
        :   boolean, default False

    invert\_vertex\_group[¶](#bpy.types.BevelModifier.invert_vertex_group "Link to this definition")
    :   Invert vertex group influence

        Type:
        :   boolean, default False

    limit\_method[¶](#bpy.types.BevelModifier.limit_method "Link to this definition")
    :   * `NONE`
          None – Bevel the entire mesh by a constant amount.
        * `ANGLE`
          Angle – Only bevel edges with sharp enough angles between faces.
        * `WEIGHT`
          Weight – Use bevel weights to determine how much bevel is applied in edge mode.
        * `VGROUP`
          Vertex Group – Use vertex group weights to select whether vertex or edge is beveled.

        Type:
        :   enum in [`'NONE'`, `'ANGLE'`, `'WEIGHT'`, `'VGROUP'`], default `'ANGLE'`

    loop\_slide[¶](#bpy.types.BevelModifier.loop_slide "Link to this definition")
    :   Prefer sliding along edges to having even widths

        Type:
        :   boolean, default True

    mark\_seam[¶](#bpy.types.BevelModifier.mark_seam "Link to this definition")
    :   Mark Seams along beveled edges

        Type:
        :   boolean, default False

    mark\_sharp[¶](#bpy.types.BevelModifier.mark_sharp "Link to this definition")
    :   Mark beveled edges as sharp

        Type:
        :   boolean, default False

    material[¶](#bpy.types.BevelModifier.material "Link to this definition")
    :   Material index of generated faces, -1 for automatic

        Type:
        :   int in [-1, 32767], default -1

    miter\_inner[¶](#bpy.types.BevelModifier.miter_inner "Link to this definition")
    :   Pattern to use for inside of miters

        * `MITER_SHARP`
          Sharp – Inside of miter is sharp.
        * `MITER_ARC`
          Arc – Inside of miter is arc.

        Type:
        :   enum in [`'MITER_SHARP'`, `'MITER_ARC'`], default `'MITER_SHARP'`

    miter\_outer[¶](#bpy.types.BevelModifier.miter_outer "Link to this definition")
    :   Pattern to use for outside of miters

        * `MITER_SHARP`
          Sharp – Outside of miter is sharp.
        * `MITER_PATCH`
          Patch – Outside of miter is squared-off patch.
        * `MITER_ARC`
          Arc – Outside of miter is arc.

        Type:
        :   enum in [`'MITER_SHARP'`, `'MITER_PATCH'`, `'MITER_ARC'`], default `'MITER_SHARP'`

    offset\_type[¶](#bpy.types.BevelModifier.offset_type "Link to this definition")
    :   What distance Width measures

        * `OFFSET`
          Offset – Amount is offset of new edges from original.
        * `WIDTH`
          Width – Amount is width of new face.
        * `DEPTH`
          Depth – Amount is perpendicular distance from original edge to bevel face.
        * `PERCENT`
          Percent – Amount is percent of adjacent edge length.
        * `ABSOLUTE`
          Absolute – Amount is absolute distance along adjacent edge.

        Type:
        :   enum in [`'OFFSET'`, `'WIDTH'`, `'DEPTH'`, `'PERCENT'`, `'ABSOLUTE'`], default `'OFFSET'`

    profile[¶](#bpy.types.BevelModifier.profile "Link to this definition")
    :   The profile shape (0.5 = round)

        Type:
        :   float in [0, 1], default 0.5

    profile\_type[¶](#bpy.types.BevelModifier.profile_type "Link to this definition")
    :   The type of shape used to rebuild a beveled section

        * `SUPERELLIPSE`
          Superellipse – The profile can be a concave or convex curve.
        * `CUSTOM`
          Custom – The profile can be any arbitrary path between its endpoints.

        Type:
        :   enum in [`'SUPERELLIPSE'`, `'CUSTOM'`], default `'SUPERELLIPSE'`

    segments[¶](#bpy.types.BevelModifier.segments "Link to this definition")
    :   Number of segments for round edges/verts

        Type:
        :   int in [1, 1000], default 1

    spread[¶](#bpy.types.BevelModifier.spread "Link to this definition")
    :   Spread distance for inner miter arcs

        Type:
        :   float in [0, inf], default 0.1

    use\_clamp\_overlap[¶](#bpy.types.BevelModifier.use_clamp_overlap "Link to this definition")
    :   Clamp the width to avoid overlap

        Type:
        :   boolean, default True

    vertex\_group[¶](#bpy.types.BevelModifier.vertex_group "Link to this definition")
    :   Vertex group name

        Type:
        :   string, default “”, (never None)

    vertex\_weight[¶](#bpy.types.BevelModifier.vertex_weight "Link to this definition")
    :   Attribute name for vertex weight

        Type:
        :   string, default “”, (never None)

    vmesh\_method[¶](#bpy.types.BevelModifier.vmesh_method "Link to this definition")
    :   The method to use to create the mesh at intersections

        * `ADJ`
          Grid Fill – Default patterned fill.
        * `CUTOFF`
          Cutoff – A cut-off at the end of each profile before the intersection.

        Type:
        :   enum in [`'ADJ'`, `'CUTOFF'`], default `'ADJ'`

    width[¶](#bpy.types.BevelModifier.width "Link to this definition")
    :   Bevel amount

        Type:
        :   float in [0, inf], default 0.1

    width\_pct[¶](#bpy.types.BevelModifier.width_pct "Link to this definition")
    :   Bevel amount for percentage method

        Type:
        :   float in [0, inf], default 0.1

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.BevelModifier.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass

    *classmethod* bl\_rna\_get\_subclass\_py(*id*, *default=None*, */*)[¶](#bpy.types.BevelModifier.bl_rna_get_subclass_py "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The class or default when not found.

        Return type:
        :   type
