# SpaceUVEditor(bpy\_struct)[¶](#spaceuveditor-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.SpaceUVEditor(*bpy\_struct*)[¶](#bpy.types.SpaceUVEditor "Link to this definition")
:   UV editor data for the image editor space

    custom\_grid\_subdivisions[¶](#bpy.types.SpaceUVEditor.custom_grid_subdivisions "Link to this definition")
    :   Number of grid units in UV space that make one UV Unit

        Type:
        :   int array of 2 items in [1, 5000], default (10, 10)

    display\_stretch\_type[¶](#bpy.types.SpaceUVEditor.display_stretch_type "Link to this definition")
    :   Type of stretch to display

        * `ANGLE`
          Angle – Angular distortion between UV and 3D angles.
        * `AREA`
          Area – Area distortion between UV and 3D faces.

        Type:
        :   enum in [`'ANGLE'`, `'AREA'`], default `'ANGLE'`

    edge\_display\_type[¶](#bpy.types.SpaceUVEditor.edge_display_type "Link to this definition")
    :   Display style for UV edges

        * `OUTLINE`
          Outline – Display white edges with black outline.
        * `DASH`
          Dash – Display dashed black-white edges.
        * `BLACK`
          Black – Display black edges.
        * `WHITE`
          White – Display white edges.

        Type:
        :   enum in [`'OUTLINE'`, `'DASH'`, `'BLACK'`, `'WHITE'`], default `'OUTLINE'`

    grid\_shape\_source[¶](#bpy.types.SpaceUVEditor.grid_shape_source "Link to this definition")
    :   Specify source for the grid shape

        * `DYNAMIC`
          Dynamic – Dynamic grid.
        * `FIXED`
          Fixed – Manually set grid divisions.
        * `PIXEL`
          Pixel – Grid aligns with pixels from image.

        Type:
        :   enum in [`'DYNAMIC'`, `'FIXED'`, `'PIXEL'`], default `'DYNAMIC'`

    lock\_bounds[¶](#bpy.types.SpaceUVEditor.lock_bounds "Link to this definition")
    :   Constraint to stay within the image bounds while editing

        Type:
        :   boolean, default False

    pixel\_round\_mode[¶](#bpy.types.SpaceUVEditor.pixel_round_mode "Link to this definition")
    :   Round UVs to pixels while editing

        * `DISABLED`
          Disabled – Don’t round to pixels.
        * `CORNER`
          Corner – Round to pixel corners.
        * `CENTER`
          Center – Round to pixel centers.

        Type:
        :   enum in [`'DISABLED'`, `'CORNER'`, `'CENTER'`], default `'DISABLED'`

    show\_faces[¶](#bpy.types.SpaceUVEditor.show_faces "Link to this definition")
    :   Display faces over the image

        Type:
        :   boolean, default False

    show\_grid\_over\_image[¶](#bpy.types.SpaceUVEditor.show_grid_over_image "Link to this definition")
    :   Show the grid over the image

        Type:
        :   boolean, default True

    show\_metadata[¶](#bpy.types.SpaceUVEditor.show_metadata "Link to this definition")
    :   Display metadata properties of the image

        Type:
        :   boolean, default False

    show\_modified\_edges[¶](#bpy.types.SpaceUVEditor.show_modified_edges "Link to this definition")
    :   Display edges after modifiers are applied

        Type:
        :   boolean, default False

    show\_pixel\_coords[¶](#bpy.types.SpaceUVEditor.show_pixel_coords "Link to this definition")
    :   Display UV coordinates in pixels rather than from 0.0 to 1.0

        Type:
        :   boolean, default False

    show\_stretch[¶](#bpy.types.SpaceUVEditor.show_stretch "Link to this definition")
    :   Display faces colored according to the difference in shape between UVs and their 3D coordinates (blue for low distortion, red for high distortion)

        Type:
        :   boolean, default False

    show\_uv[¶](#bpy.types.SpaceUVEditor.show_uv "Link to this definition")
    :   Display overlay of UV layer

        Type:
        :   boolean, default False

    stretch\_opacity[¶](#bpy.types.SpaceUVEditor.stretch_opacity "Link to this definition")
    :   Opacity of the UV Stretch overlay

        Type:
        :   float in [0, 1], default 0.0

    tile\_grid\_shape[¶](#bpy.types.SpaceUVEditor.tile_grid_shape "Link to this definition")
    :   How many tiles will be shown in the background

        Type:
        :   int array of 2 items in [1, 100], default (1, 1)

    use\_live\_unwrap[¶](#bpy.types.SpaceUVEditor.use_live_unwrap "Link to this definition")
    :   Continuously unwrap the selected UV island while transforming pinned vertices

        Type:
        :   boolean, default False

    uv\_face\_opacity[¶](#bpy.types.SpaceUVEditor.uv_face_opacity "Link to this definition")
    :   Opacity of faces in UV overlays

        Type:
        :   float in [0, 1], default 0.0

    uv\_opacity[¶](#bpy.types.SpaceUVEditor.uv_opacity "Link to this definition")
    :   Opacity of UV overlays

        Type:
        :   float in [0, 1], default 0.0

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.SpaceUVEditor.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        :   The RNA type or default when not found.
