# MovieTrackingSettings(bpy\_struct)[¶](#movietrackingsettings-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.MovieTrackingSettings(*bpy\_struct*)[¶](#bpy.types.MovieTrackingSettings "Link to this definition")
:   Match moving settings

    clean\_action[¶](#bpy.types.MovieTrackingSettings.clean_action "Link to this definition")
    :   Cleanup action to execute

        * `SELECT`
          Select – Select unclean tracks.
        * `DELETE_TRACK`
          Delete Track – Delete unclean tracks.
        * `DELETE_SEGMENTS`
          Delete Segments – Delete unclean segments of tracks.

        Type:
        :   enum in [`'SELECT'`, `'DELETE_TRACK'`, `'DELETE_SEGMENTS'`], default `'SELECT'`

    clean\_error[¶](#bpy.types.MovieTrackingSettings.clean_error "Link to this definition")
    :   Effect on tracks which have a larger re-projection error

        Type:
        :   float in [0, inf], default 0.0

    clean\_frames[¶](#bpy.types.MovieTrackingSettings.clean_frames "Link to this definition")
    :   Effect on tracks which are tracked less than the specified amount of frames

        Type:
        :   int in [0, inf], default 0

    default\_correlation\_min[¶](#bpy.types.MovieTrackingSettings.default_correlation_min "Link to this definition")
    :   Default minimum value of correlation between matched pattern and reference that is still treated as successful tracking

        Type:
        :   float in [0, 1], default 0.0

    default\_frames\_limit[¶](#bpy.types.MovieTrackingSettings.default_frames_limit "Link to this definition")
    :   Every tracking cycle, this number of frames are tracked

        Type:
        :   int in [0, 32767], default 0

    default\_margin[¶](#bpy.types.MovieTrackingSettings.default_margin "Link to this definition")
    :   Default distance from image boundary at which marker stops tracking

        Type:
        :   int in [0, 300], default 0

    default\_motion\_model[¶](#bpy.types.MovieTrackingSettings.default_motion_model "Link to this definition")
    :   Default motion model to use for tracking

        * `Perspective`
          Perspective – Search for markers that are perspectively deformed (homography) between frames.
        * `Affine`
          Affine – Search for markers that are affine-deformed (t, r, k, and skew) between frames.
        * `LocRotScale`
          Location, Rotation & Scale – Search for markers that are translated, rotated, and scaled between frames.
        * `LocScale`
          Location & Scale – Search for markers that are translated and scaled between frames.
        * `LocRot`
          Location & Rotation – Search for markers that are translated and rotated between frames.
        * `Loc`
          Location – Search for markers that are translated between frames.

        Type:
        :   enum in [`'Perspective'`, `'Affine'`, `'LocRotScale'`, `'LocScale'`, `'LocRot'`, `'Loc'`], default `'Loc'`

    default\_pattern\_match[¶](#bpy.types.MovieTrackingSettings.default_pattern_match "Link to this definition")
    :   Track pattern from given frame when tracking marker to next frame

        * `KEYFRAME`
          Keyframe – Track pattern from keyframe to next frame.
        * `PREV_FRAME`
          Previous frame – Track pattern from current frame to next frame.

        Type:
        :   enum in [`'KEYFRAME'`, `'PREV_FRAME'`], default `'KEYFRAME'`

    default\_pattern\_size[¶](#bpy.types.MovieTrackingSettings.default_pattern_size "Link to this definition")
    :   Size of pattern area for newly created tracks

        Type:
        :   int in [5, 1000], default 0

    default\_search\_size[¶](#bpy.types.MovieTrackingSettings.default_search_size "Link to this definition")
    :   Size of search area for newly created tracks

        Type:
        :   int in [5, 1000], default 0

    default\_weight[¶](#bpy.types.MovieTrackingSettings.default_weight "Link to this definition")
    :   Influence of newly created track on a final solution

        Type:
        :   float in [0, 1], default 0.0

    distance[¶](#bpy.types.MovieTrackingSettings.distance "Link to this definition")
    :   Distance between two bundles used for scene scaling

        Type:
        :   float in [-inf, inf], default 1.0

    object\_distance[¶](#bpy.types.MovieTrackingSettings.object_distance "Link to this definition")
    :   Distance between two bundles used for object scaling

        Type:
        :   float in [0.001, 10000], default 1.0

    refine\_intrinsics\_focal\_length[¶](#bpy.types.MovieTrackingSettings.refine_intrinsics_focal_length "Link to this definition")
    :   Refine focal length during camera solving

        Type:
        :   boolean, default False

    refine\_intrinsics\_principal\_point[¶](#bpy.types.MovieTrackingSettings.refine_intrinsics_principal_point "Link to this definition")
    :   Refine principal point during camera solving

        Type:
        :   boolean, default False

    refine\_intrinsics\_radial\_distortion[¶](#bpy.types.MovieTrackingSettings.refine_intrinsics_radial_distortion "Link to this definition")
    :   Refine radial coefficients of distortion model during camera solving

        Type:
        :   boolean, default False

    refine\_intrinsics\_tangential\_distortion[¶](#bpy.types.MovieTrackingSettings.refine_intrinsics_tangential_distortion "Link to this definition")
    :   Refine tangential coefficients of distortion model during camera solving

        Type:
        :   boolean, default False

    speed[¶](#bpy.types.MovieTrackingSettings.speed "Link to this definition")
    :   Limit speed of tracking to make visual feedback easier (this does not affect the tracking quality)

        * `FASTEST`
          Fastest – Track as fast as possible.
        * `DOUBLE`
          Double – Track with double speed.
        * `REALTIME`
          Realtime – Track with realtime speed.
        * `HALF`
          Half – Track with half of realtime speed.
        * `QUARTER`
          Quarter – Track with quarter of realtime speed.

        Type:
        :   enum in [`'FASTEST'`, `'DOUBLE'`, `'REALTIME'`, `'HALF'`, `'QUARTER'`], default `'FASTEST'`

    use\_default\_blue\_channel[¶](#bpy.types.MovieTrackingSettings.use_default_blue_channel "Link to this definition")
    :   Use blue channel from footage for tracking

        Type:
        :   boolean, default False

    use\_default\_brute[¶](#bpy.types.MovieTrackingSettings.use_default_brute "Link to this definition")
    :   Use a brute-force translation-only initialization when tracking

        Type:
        :   boolean, default False

    use\_default\_green\_channel[¶](#bpy.types.MovieTrackingSettings.use_default_green_channel "Link to this definition")
    :   Use green channel from footage for tracking

        Type:
        :   boolean, default False

    use\_default\_mask[¶](#bpy.types.MovieTrackingSettings.use_default_mask "Link to this definition")
    :   Use a Grease Pencil data-block as a mask to use only specified areas of pattern when tracking

        Type:
        :   boolean, default False

    use\_default\_normalization[¶](#bpy.types.MovieTrackingSettings.use_default_normalization "Link to this definition")
    :   Normalize light intensities while tracking (slower)

        Type:
        :   boolean, default False

    use\_default\_red\_channel[¶](#bpy.types.MovieTrackingSettings.use_default_red_channel "Link to this definition")
    :   Use red channel from footage for tracking

        Type:
        :   boolean, default False

    use\_keyframe\_selection[¶](#bpy.types.MovieTrackingSettings.use_keyframe_selection "Link to this definition")
    :   Automatically select keyframes when solving camera/object motion

        Type:
        :   boolean, default False

    use\_tripod\_solver[¶](#bpy.types.MovieTrackingSettings.use_tripod_solver "Link to this definition")
    :   Use special solver to track a stable camera position, such as a tripod

        Type:
        :   boolean, default False

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.MovieTrackingSettings.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass

    *classmethod* bl\_rna\_get\_subclass\_py(*id*, *default=None*, */*)[¶](#bpy.types.MovieTrackingSettings.bl_rna_get_subclass_py "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        :   The class or default when not found.

        :   type
