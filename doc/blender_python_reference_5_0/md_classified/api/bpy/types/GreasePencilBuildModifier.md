# GreasePencilBuildModifier(Modifier)[¶](#greasepencilbuildmodifier-modifier "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`Modifier`](Modifier.md#bpy.types.Modifier "bpy.types.Modifier")

*class* bpy.types.GreasePencilBuildModifier(*Modifier*)[¶](#bpy.types.GreasePencilBuildModifier "Link to this definition")
:   Animate strokes appearing and disappearing

    concurrent\_time\_alignment[¶](#bpy.types.GreasePencilBuildModifier.concurrent_time_alignment "Link to this definition")
    :   How should strokes start to appear/disappear

        * `START`
          Align Start – All strokes start at same time (i.e. short strokes finish earlier).
        * `END`
          Align End – All strokes end at same time (i.e. short strokes start later).

        Type:
        :   enum in [`'START'`, `'END'`], default `'START'`

    fade\_factor[¶](#bpy.types.GreasePencilBuildModifier.fade_factor "Link to this definition")
    :   Defines how much of the stroke is fading in/out

        Type:
        :   float in [0, 1], default 0.0

    fade\_opacity\_strength[¶](#bpy.types.GreasePencilBuildModifier.fade_opacity_strength "Link to this definition")
    :   How much strength fading applies on top of stroke opacity

        Type:
        :   float in [0, 1], default 0.0

    fade\_thickness\_strength[¶](#bpy.types.GreasePencilBuildModifier.fade_thickness_strength "Link to this definition")
    :   How much strength fading applies on top of stroke thickness

        Type:
        :   float in [0, 1], default 0.0

    frame\_end[¶](#bpy.types.GreasePencilBuildModifier.frame_end "Link to this definition")
    :   End Frame (when Restrict Frame Range is enabled)

        Type:
        :   float in [-1.04857e+06, 1.04857e+06], default 125.0

    frame\_start[¶](#bpy.types.GreasePencilBuildModifier.frame_start "Link to this definition")
    :   Start Frame (when Restrict Frame Range is enabled)

        Type:
        :   float in [-1.04857e+06, 1.04857e+06], default 1.0

    invert\_layer\_filter[¶](#bpy.types.GreasePencilBuildModifier.invert_layer_filter "Link to this definition")
    :   Invert layer filter

        Type:
        :   boolean, default False

    invert\_layer\_pass\_filter[¶](#bpy.types.GreasePencilBuildModifier.invert_layer_pass_filter "Link to this definition")

        Type:
        :   boolean, default False

    invert\_material\_filter[¶](#bpy.types.GreasePencilBuildModifier.invert_material_filter "Link to this definition")
    :   Invert material filter

        Type:
        :   boolean, default False

    invert\_material\_pass\_filter[¶](#bpy.types.GreasePencilBuildModifier.invert_material_pass_filter "Link to this definition")
    :   Invert material pass filter

        Type:
        :   boolean, default False

    layer\_pass\_filter[¶](#bpy.types.GreasePencilBuildModifier.layer_pass_filter "Link to this definition")
    :   Layer pass filter

        Type:
        :   int in [0, 100], default 0

    length[¶](#bpy.types.GreasePencilBuildModifier.length "Link to this definition")
    :   Maximum number of frames that the build effect can run for (unless another GP keyframe occurs before this time has elapsed)

        Type:
        :   float in [1, 1.04857e+06], default 100.0

    material\_filter[¶](#bpy.types.GreasePencilBuildModifier.material_filter "Link to this definition")
    :   Material used for filtering

        Type:
        :   [`Material`](Material.md#bpy.types.Material "bpy.types.Material")

    material\_pass\_filter[¶](#bpy.types.GreasePencilBuildModifier.material_pass_filter "Link to this definition")
    :   Material pass

        Type:
        :   int in [0, 100], default 0

    mode[¶](#bpy.types.GreasePencilBuildModifier.mode "Link to this definition")
    :   How strokes are being built

        * `SEQUENTIAL`
          Sequential – Strokes appear/disappear one after the other, but only a single one changes at a time.
        * `CONCURRENT`
          Concurrent – Multiple strokes appear/disappear at once.
        * `ADDITIVE`
          Additive – Builds only new strokes (assuming ‘additive’ drawing).

        Type:
        :   enum in [`'SEQUENTIAL'`, `'CONCURRENT'`, `'ADDITIVE'`], default `'SEQUENTIAL'`

    object[¶](#bpy.types.GreasePencilBuildModifier.object "Link to this definition")
    :   Object used as build starting position

        Type:
        :   [`Object`](Object.md#bpy.types.Object "bpy.types.Object")

    open\_fading\_panel[¶](#bpy.types.GreasePencilBuildModifier.open_fading_panel "Link to this definition")
    :   Type:

    open\_frame\_range\_panel[¶](#bpy.types.GreasePencilBuildModifier.open_frame_range_panel "Link to this definition")

        :   boolean, default False

    percentage\_factor[¶](#bpy.types.GreasePencilBuildModifier.percentage_factor "Link to this definition")
    :   Defines how much of the stroke is visible

        Type:
        :   float in [0, 1], default 0.0

    speed\_factor[¶](#bpy.types.GreasePencilBuildModifier.speed_factor "Link to this definition")
    :   Multiply recorded drawing speed by a factor

        Type:
        :   float in [0, 100], default 1.2

    speed\_maxgap[¶](#bpy.types.GreasePencilBuildModifier.speed_maxgap "Link to this definition")

        Type:
        :   float in [0, 100], default 0.5

    start\_delay[¶](#bpy.types.GreasePencilBuildModifier.start_delay "Link to this definition")
    :   Number of frames after each GP keyframe before the modifier has any effect

        Type:
        :   float in [0, 1.04857e+06], default 0.0

    target\_vertex\_group[¶](#bpy.types.GreasePencilBuildModifier.target_vertex_group "Link to this definition")
    :   Output Vertex group

        Type:
        :   string, default “”, (never None)

    time\_mode[¶](#bpy.types.GreasePencilBuildModifier.time_mode "Link to this definition")
    :   Use drawing speed, a number of frames, or a manual factor to build strokes

        * `DRAWSPEED`
          Natural Drawing Speed – Use recorded speed multiplied by a factor.
        * `FRAMES`
          Number of Frames – Set a fixed number of frames for all build animations.
        * `PERCENTAGE`
          Percentage Factor – Set a manual percentage to build.

        Type:
        :   enum in [`'DRAWSPEED'`, `'FRAMES'`, `'PERCENTAGE'`], default `'FRAMES'`

    transition[¶](#bpy.types.GreasePencilBuildModifier.transition "Link to this definition")
    :   How are strokes animated (i.e. are they appearing or disappearing)

        * `GROW`
          Grow – Show points in the order they occur in each stroke (e.g. for animating lines being drawn).
        * `SHRINK`
          Shrink – Hide points from the end of each stroke to the start (e.g. for animating lines being erased).
        * `FADE`
          Vanish – Hide points in the order they occur in each stroke (e.g. for animating ink fading or vanishing after getting drawn).

        Type:
        :   enum in [`'GROW'`, `'SHRINK'`, `'FADE'`], default `'GROW'`

    tree\_node\_filter[¶](#bpy.types.GreasePencilBuildModifier.tree_node_filter "Link to this definition")
    :   Layer name

        Type:
        :   string, default “”, (never None)

    use\_fading[¶](#bpy.types.GreasePencilBuildModifier.use_fading "Link to this definition")

        Type:
        :   boolean, default False

    use\_percentage[¶](#bpy.types.GreasePencilBuildModifier.use_percentage "Link to this definition")

        Type:
        :   boolean, default False

    use\_restrict\_frame\_range[¶](#bpy.types.GreasePencilBuildModifier.use_restrict_frame_range "Link to this definition")
    :   Only modify strokes during the specified frame range

        Type:
        :   boolean, default False
