# BakeSettings(bpy\_struct)[¶](#bakesettings-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.BakeSettings(*bpy\_struct*)[¶](#bpy.types.BakeSettings "Link to this definition")
:   Bake data for a Scene data-block

    cage\_extrusion[¶](#bpy.types.BakeSettings.cage_extrusion "Link to this definition")
    :   Inflate the active object by the specified distance for baking. This helps matching to points nearer to the outside of the selected object meshes.

        Type:
        :   float in [0, inf], default 0.0

    cage\_object[¶](#bpy.types.BakeSettings.cage_object "Link to this definition")
    :   Object to use as cage instead of calculating the cage from the active object with cage extrusion

        Type:
        :   [`Object`](Object.md#bpy.types.Object "bpy.types.Object")

    displacement\_space[¶](#bpy.types.BakeSettings.displacement_space "Link to this definition")
    :   Choose displacement space for baking

        * `OBJECT`
          Object – Bake the displacement in object space.
        * `TANGENT`
          Tangent – Bake the displacement in tangent space.

        Type:
        :   enum in [`'OBJECT'`, `'TANGENT'`], default `'OBJECT'`

    filepath[¶](#bpy.types.BakeSettings.filepath "Link to this definition")
    :   Image filepath to use when saving externally

        Type:
        :   string, default “”, (never None, blend relative `//` prefix supported)

    height[¶](#bpy.types.BakeSettings.height "Link to this definition")
    :   Vertical dimension of the baking map

        Type:
        :   int in [4, 10000], default 512

    image\_settings[¶](#bpy.types.BakeSettings.image_settings "Link to this definition")
    :   Type:
        :   [`ImageFormatSettings`](ImageFormatSettings.md#bpy.types.ImageFormatSettings "bpy.types.ImageFormatSettings"), (readonly, never None)

    margin[¶](#bpy.types.BakeSettings.margin "Link to this definition")
    :   Extends the baked result as a post process filter

        Type:
        :   int in [0, 32767], default 16

    margin\_type[¶](#bpy.types.BakeSettings.margin_type "Link to this definition")
    :   Algorithm to extend the baked result

        Type:
        :   enum in [Bake Margin Type Items](../../../appendix/bpy_types_enum_items/bake_margin_type_items.md#rna-enum-bake-margin-type-items), default `'ADJACENT_FACES'`

    max\_ray\_distance[¶](#bpy.types.BakeSettings.max_ray_distance "Link to this definition")
    :   The maximum ray distance for matching points between the active and selected objects. If zero, there is no limit.

        Type:
        :   float in [0, inf], default 0.0

    normal\_b[¶](#bpy.types.BakeSettings.normal_b "Link to this definition")
    :   Axis to bake in blue channel

        Type:
        :   enum in [Normal Swizzle Items](../../../appendix/bpy_types_enum_items/normal_swizzle_items.md#rna-enum-normal-swizzle-items), default `'POS_X'`

    normal\_g[¶](#bpy.types.BakeSettings.normal_g "Link to this definition")
    :   Axis to bake in green channel

        Type:
        :   enum in [Normal Swizzle Items](../../../appendix/bpy_types_enum_items/normal_swizzle_items.md#rna-enum-normal-swizzle-items), default `'POS_X'`

    normal\_r[¶](#bpy.types.BakeSettings.normal_r "Link to this definition")
    :   Axis to bake in red channel

        Type:
        :   enum in [Normal Swizzle Items](../../../appendix/bpy_types_enum_items/normal_swizzle_items.md#rna-enum-normal-swizzle-items), default `'POS_X'`

    normal\_space[¶](#bpy.types.BakeSettings.normal_space "Link to this definition")
    :   Choose normal space for baking

        Type:
        :   enum in [Normal Space Items](../../../appendix/bpy_types_enum_items/normal_space_items.md#rna-enum-normal-space-items), default `'TANGENT'`

    pass\_filter[¶](#bpy.types.BakeSettings.pass_filter "Link to this definition")
    :   Passes to include in the active baking pass

        Type:
        :   enum set in [Bake Pass Filter Type Items](../../../appendix/bpy_types_enum_items/bake_pass_filter_type_items.md#rna-enum-bake-pass-filter-type-items), default {`'COLOR'`, `'DIFFUSE'`, `'DIRECT'`, `'EMIT'`, `'GLOSSY'`, `'INDIRECT'`, `'TRANSMISSION'`}, (readonly)

    save\_mode[¶](#bpy.types.BakeSettings.save_mode "Link to this definition")
    :   Where to save baked image textures

        Type:
        :   enum in [Bake Save Mode Items](../../../appendix/bpy_types_enum_items/bake_save_mode_items.md#rna-enum-bake-save-mode-items), default `'INTERNAL'`

    target[¶](#bpy.types.BakeSettings.target "Link to this definition")
    :   Where to output the baked map

        Type:
        :   enum in [Bake Target Items](../../../appendix/bpy_types_enum_items/bake_target_items.md#rna-enum-bake-target-items), default `'IMAGE_TEXTURES'`

    type[¶](#bpy.types.BakeSettings.type "Link to this definition")
    :   Choose shading information to bake into the image

        * `NORMALS`
          Normals – Bake normals.
        * `DISPLACEMENT`
          Displacement – Bake displacement.
        * `VECTOR_DISPLACEMENT`
          Vector Displacement – Bake vector displacement.

        Type:
        :   enum in [`'NORMALS'`, `'DISPLACEMENT'`, `'VECTOR_DISPLACEMENT'`], default `'NORMALS'`

    use\_automatic\_name[¶](#bpy.types.BakeSettings.use_automatic_name "Link to this definition")
    :   Automatically name the output file with the pass type (external only)

        Type:
        :   boolean, default False

    use\_cage[¶](#bpy.types.BakeSettings.use_cage "Link to this definition")
    :   Cast rays to active object from a cage

        Type:
        :   boolean, default False

    use\_clear[¶](#bpy.types.BakeSettings.use_clear "Link to this definition")
    :   Clear Images before baking (internal only)

        Type:
        :   boolean, default True

    use\_lores\_mesh[¶](#bpy.types.BakeSettings.use_lores_mesh "Link to this definition")
    :   Calculate heights against unsubdivided low resolution mesh

        Type:
        :   boolean, default False

    use\_multires[¶](#bpy.types.BakeSettings.use_multires "Link to this definition")
    :   Bake directly from multires object

        Type:
        :   boolean, default False

    use\_pass\_color[¶](#bpy.types.BakeSettings.use_pass_color "Link to this definition")
    :   Color the pass

        Type:
        :   boolean, default True

    use\_pass\_diffuse[¶](#bpy.types.BakeSettings.use_pass_diffuse "Link to this definition")
    :   Add diffuse contribution

        Type:
        :   boolean, default True

    use\_pass\_direct[¶](#bpy.types.BakeSettings.use_pass_direct "Link to this definition")
    :   Add direct lighting contribution

        Type:
        :   boolean, default True

    use\_pass\_emit[¶](#bpy.types.BakeSettings.use_pass_emit "Link to this definition")
    :   Add emission contribution

        Type:
        :   boolean, default True

    use\_pass\_glossy[¶](#bpy.types.BakeSettings.use_pass_glossy "Link to this definition")
    :   Add glossy contribution

        Type:
        :   boolean, default True

    use\_pass\_indirect[¶](#bpy.types.BakeSettings.use_pass_indirect "Link to this definition")
    :   Add indirect lighting contribution

        Type:
        :   boolean, default True

    use\_pass\_transmission[¶](#bpy.types.BakeSettings.use_pass_transmission "Link to this definition")
    :   Add transmission contribution

        Type:
        :   boolean, default True

    use\_selected\_to\_active[¶](#bpy.types.BakeSettings.use_selected_to_active "Link to this definition")
    :   Bake shading on the surface of selected objects to the active object

        Type:
        :   boolean, default False

    use\_split\_materials[¶](#bpy.types.BakeSettings.use_split_materials "Link to this definition")
    :   Split external images per material (external only)

        Type:
        :   boolean, default False

    view\_from[¶](#bpy.types.BakeSettings.view_from "Link to this definition")
    :   Source of reflection ray directions

        * `ABOVE_SURFACE`
          Above Surface – Cast rays from above the surface.
        * `ACTIVE_CAMERA`
          Active Camera – Use the active camera’s position to cast rays.

        Type:
        :   enum in [`'ABOVE_SURFACE'`, `'ACTIVE_CAMERA'`], default `'ABOVE_SURFACE'`

    width[¶](#bpy.types.BakeSettings.width "Link to this definition")
    :   Horizontal dimension of the baking map

        Type:
        :   int in [4, 10000], default 512

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.BakeSettings.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass

    *classmethod* bl\_rna\_get\_subclass\_py(*id*, *default=None*, */*)[¶](#bpy.types.BakeSettings.bl_rna_get_subclass_py "Link to this definition")
