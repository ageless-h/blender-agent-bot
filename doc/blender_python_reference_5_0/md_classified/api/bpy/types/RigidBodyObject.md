# RigidBodyObject(bpy\_struct)[¶](#rigidbodyobject-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.RigidBodyObject(*bpy\_struct*)[¶](#bpy.types.RigidBodyObject "Link to this definition")
:   Settings for object participating in Rigid Body Simulation

    angular\_damping[¶](#bpy.types.RigidBodyObject.angular_damping "Link to this definition")
    :   Amount of angular velocity that is lost over time

        Type:
        :   float in [0, 1], default 0.1

    collision\_collections[¶](#bpy.types.RigidBodyObject.collision_collections "Link to this definition")
    :   Collision collections rigid body belongs to

        Type:
        :   boolean array of 20 items, default (False, False, False, False, False, False, False, False, False, False, False, False, False, False, False, False, False, False, False, False)

    collision\_margin[¶](#bpy.types.RigidBodyObject.collision_margin "Link to this definition")
    :   Threshold of distance near surface where collisions are still considered (best results when non-zero)

        Type:
        :   float in [0, 1], default 0.04

    collision\_shape[¶](#bpy.types.RigidBodyObject.collision_shape "Link to this definition")
    :   Collision Shape of object in Rigid Body Simulations

        Type:
        :   enum in [Rigidbody Object Shape Items](../../../appendix/bpy_types_enum_items/rigidbody_object_shape_items.md#rna-enum-rigidbody-object-shape-items), default `'BOX'`

    deactivate\_angular\_velocity[¶](#bpy.types.RigidBodyObject.deactivate_angular_velocity "Link to this definition")
    :   Angular Velocity below which simulation stops simulating object

        Type:
        :   float in [0, inf], default 0.5

    deactivate\_linear\_velocity[¶](#bpy.types.RigidBodyObject.deactivate_linear_velocity "Link to this definition")
    :   Linear Velocity below which simulation stops simulating object

        Type:
        :   float in [0, inf], default 0.4

    enabled[¶](#bpy.types.RigidBodyObject.enabled "Link to this definition")
    :   Rigid Body actively participates to the simulation

        Type:
        :   boolean, default False

    friction[¶](#bpy.types.RigidBodyObject.friction "Link to this definition")
    :   Resistance of object to movement

        Type:
        :   float in [0, inf], default 0.5

    kinematic[¶](#bpy.types.RigidBodyObject.kinematic "Link to this definition")
    :   Allow rigid body to be controlled by the animation system

        Type:
        :   boolean, default False

    linear\_damping[¶](#bpy.types.RigidBodyObject.linear_damping "Link to this definition")
    :   Amount of linear velocity that is lost over time

        Type:
        :   float in [0, 1], default 0.04

    mass[¶](#bpy.types.RigidBodyObject.mass "Link to this definition")
    :   How much the object ‘weighs’ irrespective of gravity

        Type:
        :   float in [0.001, inf], default 1.0

    mesh\_source[¶](#bpy.types.RigidBodyObject.mesh_source "Link to this definition")
    :   Source of the mesh used to create collision shape

        * `BASE`
          Base – Base mesh.
        * `DEFORM`
          Deform – Deformations (shape keys, deform modifiers).
        * `FINAL`
          Final – All modifiers.

        Type:
        :   enum in [`'BASE'`, `'DEFORM'`, `'FINAL'`], default `'BASE'`

    restitution[¶](#bpy.types.RigidBodyObject.restitution "Link to this definition")
    :   Tendency of object to bounce after colliding with another (0 = stays still, 1 = perfectly elastic)

        Type:
        :   float in [0, inf], default 0.0

    type[¶](#bpy.types.RigidBodyObject.type "Link to this definition")
    :   Role of object in Rigid Body Simulations

        Type:
        :   enum in [Rigidbody Object Type Items](../../../appendix/bpy_types_enum_items/rigidbody_object_type_items.md#rna-enum-rigidbody-object-type-items), default `'ACTIVE'`

    use\_deactivation[¶](#bpy.types.RigidBodyObject.use_deactivation "Link to this definition")
    :   Enable deactivation of resting rigid bodies (increases performance and stability but can cause glitches)

        Type:
        :   boolean, default True

    use\_deform[¶](#bpy.types.RigidBodyObject.use_deform "Link to this definition")
    :   Rigid body deforms during simulation

        Type:
        :   boolean, default False

    use\_margin[¶](#bpy.types.RigidBodyObject.use_margin "Link to this definition")
    :   Use custom collision margin (some shapes will have a visible gap around them)

        Type:
        :   boolean, default False

    use\_start\_deactivated[¶](#bpy.types.RigidBodyObject.use_start_deactivated "Link to this definition")
    :   Deactivate rigid body at the start of the simulation

        Type:
        :   boolean, default False

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.RigidBodyObject.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass

    *classmethod* bl\_rna\_get\_subclass\_py(*id*, *default=None*, */*)[¶](#bpy.types.RigidBodyObject.bl_rna_get_subclass_py "Link to this definition")
    :   Parameters:

        Returns:
        :   The class or default when not found.
