# MovieStrip(Strip)[¶](#moviestrip-strip "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`Strip`](Strip.md#bpy.types.Strip "bpy.types.Strip")

*class* bpy.types.MovieStrip(*Strip*)[¶](#bpy.types.MovieStrip "Link to this definition")
:   Sequence strip to load a video

    alpha\_mode[¶](#bpy.types.MovieStrip.alpha_mode "Link to this definition")
    :   Representation of alpha information in the RGBA pixels

        * `STRAIGHT`
          Straight – RGB channels in transparent pixels are unaffected by the alpha channel.
        * `PREMUL`
          Premultiplied – RGB channels in transparent pixels are multiplied by the alpha channel.

        Type:
        :   enum in [`'STRAIGHT'`, `'PREMUL'`], default `'STRAIGHT'`

    animation\_offset\_end[¶](#bpy.types.MovieStrip.animation_offset_end "Link to this definition")
    :   Animation end offset (trim end)

        Type:
        :   int in [0, inf], default 0

    :   Animation start offset (trim start)

        Type:
        :   int in [0, inf], default 0

    color\_multiply[¶](#bpy.types.MovieStrip.color_multiply "Link to this definition")
    :   Type:
        :   float in [0, 20], default 1.0

    color\_saturation[¶](#bpy.types.MovieStrip.color_saturation "Link to this definition")
    :   Adjust the intensity of the input’s color

        :   float in [0, 20], default 1.0
