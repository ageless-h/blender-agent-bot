# PointLight(Light)[¶](#pointlight-light "Link to this heading")

base classes — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct"), [`ID`](ID.md#bpy.types.ID "bpy.types.ID"), [`Light`](Light.md#bpy.types.Light "bpy.types.Light")

*class* bpy.types.PointLight(*Light*)[¶](#bpy.types.PointLight "Link to this definition")
:   Omnidirectional point Light

    energy[¶](#bpy.types.PointLight.energy "Link to this definition")
    :   Light energy emitted over the entire area of the light in all directions, in units of radiant power (W)

        Type:
        :   float in [-inf, inf], default 10.0

    shadow\_buffer\_clip\_start[¶](#bpy.types.PointLight.shadow_buffer_clip_start "Link to this definition")
    :   Shadow map clip start, below which objects will not generate shadows

        Type:
        :   float in [1e-06, inf], default 0.05

    shadow\_filter\_radius[¶](#bpy.types.PointLight.shadow_filter_radius "Link to this definition")
    :   Blur shadow aliasing using Percentage Closer Filtering

        Type:
        :   float in [0, inf], default 1.0

    shadow\_jitter\_overblur[¶](#bpy.types.PointLight.shadow_jitter_overblur "Link to this definition")
    :   Apply shadow tracing to each jittered sample to reduce under-sampling artifacts

        Type:
        :   float in [0, 100], default 10.0

    shadow\_maximum\_resolution[¶](#bpy.types.PointLight.shadow_maximum_resolution "Link to this definition")
    :   Minimum size of a shadow map pixel. Higher values use less memory at the cost of shadow quality.

        Type:
        :   float in [0, inf], default 0.001

    shadow\_soft\_size[¶](#bpy.types.PointLight.shadow_soft_size "Link to this definition")
    :   Light size for ray shadow sampling (Raytraced shadows)

        Type:
        :   float in [0, inf], default 0.0

    use\_absolute\_resolution[¶](#bpy.types.PointLight.use_absolute_resolution "Link to this definition")
    :   Limit the resolution at 1 unit from the light origin instead of relative to the shadowed pixel

        Type:
        :   boolean, default False

    use\_shadow\_jitter[¶](#bpy.types.PointLight.use_shadow_jitter "Link to this definition")
    :   Enable jittered soft shadows to increase shadow precision (disabled in viewport unless enabled in the render settings). Has a high performance impact.

        Type:
        :   boolean, default False

    use\_soft\_falloff[¶](#bpy.types.PointLight.use_soft_falloff "Link to this definition")
    :   Apply falloff to avoid sharp edges when the light geometry intersects with other objects

        Type:
        :   boolean, default True

    inline\_shader\_nodes()[¶](#bpy.types.PointLight.inline_shader_nodes "Link to this definition")
    :   Get the inlined shader nodes of this light. This preprocesses the node tree
        to remove nested groups, repeat zones and more.

        Returns:
        :   The inlined shader nodes.

        Return type:
        :   [`bpy.types.InlineShaderNodes`](InlineShaderNodes.md#bpy.types.InlineShaderNodes "bpy.types.InlineShaderNodes")

    *classmethod* bl\_rna\_get\_subclass(*id*, *default=None*, */*)[¶](#bpy.types.PointLight.bl_rna_get_subclass "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass

    *classmethod* bl\_rna\_get\_subclass\_py(*id*, *default=None*, */*)[¶](#bpy.types.PointLight.bl_rna_get_subclass_py "Link to this definition")
    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The class or default when not found.

        Return type:
        :   type
