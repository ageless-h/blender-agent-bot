# MaterialGPencilStyle(bpy\_struct)[¶](#materialgpencilstyle-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.MaterialGPencilStyle(*bpy\_struct*)[¶](#bpy.types.MaterialGPencilStyle "Link to this definition")
:   alignment\_mode[¶](#bpy.types.MaterialGPencilStyle.alignment_mode "Link to this definition")
    :   Defines how align Dots and Boxes with drawing path and object rotation

        * `PATH`
          Path – Follow stroke drawing path and object rotation.
        * `OBJECT`
          Object – Follow object rotation only.
        * `FIXED`
          Fixed – Do not follow drawing path or object rotation and keeps aligned with viewport.

        Type:
        :   enum in [`'PATH'`, `'OBJECT'`, `'FIXED'`], default `'PATH'`

    alignment\_rotation[¶](#bpy.types.MaterialGPencilStyle.alignment_rotation "Link to this definition")
    :   Additional rotation applied to dots and square texture of strokes. Only applies in texture shading mode.

        Type:
        :   float in [-1.5708, 1.5708], default 0.0

    color[¶](#bpy.types.MaterialGPencilStyle.color "Link to this definition")
    :   Type:
        :   float array of 4 items in [0, 1], default (0.0, 0.0, 0.0, 0.0)

    fill\_color[¶](#bpy.types.MaterialGPencilStyle.fill_color "Link to this definition")
    :   Color for filling region bounded by each stroke

        Type:
        :   float array of 4 items in [0, 1], default (0.0, 0.0, 0.0, 0.0)

    fill\_image[¶](#bpy.types.MaterialGPencilStyle.fill_image "Link to this definition")
    :   Type:
        :   [`Image`](Image.md#bpy.types.Image "bpy.types.Image")

    fill\_style[¶](#bpy.types.MaterialGPencilStyle.fill_style "Link to this definition")
    :   Select style used to fill strokes

        * `SOLID`
          Solid – Fill area with solid color.
        * `GRADIENT`
          Gradient – Fill area with gradient color.
        * `TEXTURE`
          Texture – Fill area with image texture.

        Type:
        :   enum in [`'SOLID'`, `'GRADIENT'`, `'TEXTURE'`], default `'SOLID'`

    flip[¶](#bpy.types.MaterialGPencilStyle.flip "Link to this definition")
    :   Flip filling colors

        Type:
        :   boolean, default False

    ghost[¶](#bpy.types.MaterialGPencilStyle.ghost "Link to this definition")
    :   Display strokes using this color when showing onion skins

        Type:
        :   boolean, default False

    gradient\_type[¶](#bpy.types.MaterialGPencilStyle.gradient_type "Link to this definition")
    :   Select type of gradient used to fill strokes

        * `LINEAR`
          Linear – Fill area with gradient color.
        * `RADIAL`
          Radial – Fill area with radial gradient.

        Type:
        :   enum in [`'LINEAR'`, `'RADIAL'`], default `'LINEAR'`

    hide[¶](#bpy.types.MaterialGPencilStyle.hide "Link to this definition")
    :   Set color Visibility

        Type:
        :   boolean, default False

    is\_fill\_visible[¶](#bpy.types.MaterialGPencilStyle.is_fill_visible "Link to this definition")
    :   True when opacity of fill is set high enough to be visible

        Type:
        :   boolean, default False, (readonly)

    is\_stroke\_visible[¶](#bpy.types.MaterialGPencilStyle.is_stroke_visible "Link to this definition")
    :   True when opacity of stroke is set high enough to be visible

        Type:
        :   boolean, default False, (readonly)

    lock[¶](#bpy.types.MaterialGPencilStyle.lock "Link to this definition")
    :   Protect color from further editing and/or frame changes

        Type:
        :   boolean, default False

    mix\_color[¶](#bpy.types.MaterialGPencilStyle.mix_color "Link to this definition")
    :   Color for mixing with primary filling color

        Type:
        :   float array of 4 items in [0, 1], default (0.0, 0.0, 0.0, 0.0)

    mix\_factor[¶](#bpy.types.MaterialGPencilStyle.mix_factor "Link to this definition")
    :   Mix Factor

        Type:
        :   float in [0, 1], default 0.0

    mix\_stroke\_factor[¶](#bpy.types.MaterialGPencilStyle.mix_stroke_factor "Link to this definition")
    :   Mix Stroke Factor

        Type:
        :   float in [0, 1], default 0.0

    mode[¶](#bpy.types.MaterialGPencilStyle.mode "Link to this definition")
    :   Select line type for strokes

        * `LINE`
          Line – Draw strokes using a continuous line.
        * `DOTS`
          Dots – Draw strokes using separated dots.
        * `BOX`
          Squares – Draw strokes using separated squares.

        Type:
        :   enum in [`'LINE'`, `'DOTS'`, `'BOX'`], default `'LINE'`

    pass\_index[¶](#bpy.types.MaterialGPencilStyle.pass_index "Link to this definition")
    :   Index number for the “Color Index” pass

        Type:
        :   int in [0, 32767], default 0

    pixel\_size[¶](#bpy.types.MaterialGPencilStyle.pixel_size "Link to this definition")
    :   Texture Pixel Size factor along the stroke

        Type:
        :   float in [1, 5000], default 0.0

    show\_fill[¶](#bpy.types.MaterialGPencilStyle.show_fill "Link to this definition")
    :   Show stroke fills of this material

        Type:
        :   boolean, default False

    show\_stroke[¶](#bpy.types.MaterialGPencilStyle.show_stroke "Link to this definition")
    :   Show stroke lines of this material

        Type:
        :   boolean, default False

    stroke\_image[¶](#bpy.types.MaterialGPencilStyle.stroke_image "Link to this definition")
    :   Type:
        :   [`Image`](Image.md#bpy.types.Image "bpy.types.Image")

    stroke\_style[¶](#bpy.types.MaterialGPencilStyle.stroke_style "Link to this definition")
    :   Select style used to draw strokes

        * `SOLID`
          Solid – Draw strokes with solid color.
        * `TEXTURE`
          Texture – Draw strokes using texture.

        Type:
        :   enum in [`'SOLID'`, `'TEXTURE'`], default `'SOLID'`

    texture\_angle[¶](#bpy.types.MaterialGPencilStyle.texture_angle "Link to this definition")
    :   Texture Orientation Angle

        Type:
        :   float in [-inf, inf], default 0.0

    texture\_clamp[¶](#bpy.types.MaterialGPencilStyle.texture_clamp "Link to this definition")
    :   Do not repeat texture and clamp to one instance only

        Type:
        :   boolean, default False

    texture\_offset[¶](#bpy.types.MaterialGPencilStyle.texture_offset "Link to this definition")
    :   Shift Texture in 2d Space

        Type:
        :   [`mathutils.Vector`](../../mathutils/index.md#mathutils.Vector "mathutils.Vector") of 2 items in [-inf, inf], default (0.0, 0.0)

    texture\_scale[¶](#bpy.types.MaterialGPencilStyle.texture_scale "Link to this definition")
    :   Scale Factor for Texture

        Type:
        :   [`mathutils.Vector`](../../mathutils/index.md#mathutils.Vector "mathutils.Vector") of 2 items in [-inf, inf], default (0.0, 0.0)

    use\_fill\_holdout[¶](#bpy.types.MaterialGPencilStyle.use_fill_holdout "Link to this definition")
    :   Remove the color from underneath this stroke by using it as a mask

        Type:
        :   boolean, default False

    use\_overlap\_strokes[¶](#bpy.types.MaterialGPencilStyle.use_overlap_strokes "Link to this definition")
    :   Disable stencil and overlap self intersections with alpha materials

        Type:
        :   boolean, default False

    use\_stroke\_holdout[¶](#bpy.types.MaterialGPencilStyle.use_stroke_holdout "Link to this definition")
    :   Remove the color from underneath this stroke by using it as a mask

        Type:
        :   boolean, default False
