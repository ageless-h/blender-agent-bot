# FieldSettings(bpy\_struct)[¶](#fieldsettings-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.FieldSettings(*bpy\_struct*)[¶](#bpy.types.FieldSettings "Link to this definition")
:   Field settings for an object in physics simulation

    apply\_to\_location[¶](#bpy.types.FieldSettings.apply_to_location "Link to this definition")
    :   Affect particle’s location

        Type:
        :   boolean, default False

    apply\_to\_rotation[¶](#bpy.types.FieldSettings.apply_to_rotation "Link to this definition")
    :   Affect particle’s dynamic rotation

        Type:
        :   boolean, default False

    distance\_max[¶](#bpy.types.FieldSettings.distance_max "Link to this definition")
    :   Maximum distance for the field to work

        Type:
        :   float in [0, inf], default 0.0

    distance\_min[¶](#bpy.types.FieldSettings.distance_min "Link to this definition")
    :   Minimum distance for the field’s falloff

        Type:
        :   float in [0, 1000], default 0.0

    falloff\_power[¶](#bpy.types.FieldSettings.falloff_power "Link to this definition")
    :   How quickly strength falls off with distance from the force field

        Type:
        :   float in [0, 10], default 0.0

    falloff\_type[¶](#bpy.types.FieldSettings.falloff_type "Link to this definition")
    :   Type:
        :   enum in [`'CONE'`, `'SPHERE'`, `'TUBE'`], default `'SPHERE'`

    flow[¶](#bpy.types.FieldSettings.flow "Link to this definition")
    :   Convert effector force into air flow velocity

        Type:
        :   float in [-inf, inf], default 0.0

    guide\_clump\_amount[¶](#bpy.types.FieldSettings.guide_clump_amount "Link to this definition")
    :   Amount of clumping

        Type:
        :   float in [-1, 1], default 0.0

    guide\_clump\_shape[¶](#bpy.types.FieldSettings.guide_clump_shape "Link to this definition")
    :   Shape of clumping

        Type:
        :   float in [-0.999, 0.999], default 0.0

    guide\_free[¶](#bpy.types.FieldSettings.guide_free "Link to this definition")
    :   Guide-free time from particle life’s end

        Type:
        :   float in [0, 0.99], default 0.0

    guide\_kink\_amplitude[¶](#bpy.types.FieldSettings.guide_kink_amplitude "Link to this definition")
    :   The amplitude of the offset

        Type:
        :   float in [0, 10], default 0.0

    guide\_kink\_axis[¶](#bpy.types.FieldSettings.guide_kink_axis "Link to this definition")
    :   Which axis to use for offset

        Type:
        :   enum in [Axis Xyz Items](../../../appendix/bpy_types_enum_items/axis_xyz_items.md#rna-enum-axis-xyz-items), default `'X'`

    guide\_kink\_frequency[¶](#bpy.types.FieldSettings.guide_kink_frequency "Link to this definition")
    :   The frequency of the offset (1/total length)

        Type:
        :   float in [0, 10], default 0.0

    guide\_kink\_shape[¶](#bpy.types.FieldSettings.guide_kink_shape "Link to this definition")
    :   Adjust the offset to the beginning/end

        Type:
        :   float in [-0.999, 0.999], default 0.0

    guide\_kink\_type[¶](#bpy.types.FieldSettings.guide_kink_type "Link to this definition")
    :   Type of periodic offset on the curve

        Type:
        :   enum in [`'NONE'`, `'BRAID'`, `'CURL'`, `'RADIAL'`, `'ROLL'`, `'ROTATION'`, `'WAVE'`], default `'NONE'`

    guide\_minimum[¶](#bpy.types.FieldSettings.guide_minimum "Link to this definition")
    :   The distance from which particles are affected fully

        Type:
        :   float in [-inf, inf], default 0.0

    harmonic\_damping[¶](#bpy.types.FieldSettings.harmonic_damping "Link to this definition")
    :   Damping of the harmonic force

        Type:
        :   float in [-inf, inf], default 0.0

    inflow[¶](#bpy.types.FieldSettings.inflow "Link to this definition")
    :   Inwards component of the vortex force

        Type:
        :   float in [-inf, inf], default 0.0

    linear\_drag[¶](#bpy.types.FieldSettings.linear_drag "Link to this definition")
    :   Drag component proportional to velocity

        Type:
        :   float in [-inf, inf], default 0.0

    noise[¶](#bpy.types.FieldSettings.noise "Link to this definition")
    :   Amount of noise for the force strength

        Type:
        :   float in [0, 10], default 0.0

    quadratic\_drag[¶](#bpy.types.FieldSettings.quadratic_drag "Link to this definition")
    :   Drag component proportional to the square of velocity

        Type:
        :   float in [-inf, inf], default 0.0

    radial\_falloff[¶](#bpy.types.FieldSettings.radial_falloff "Link to this definition")
    :   Radial falloff power (real gravitational falloff = 2)

        Type:
        :   float in [0, 10], default 0.0

    radial\_max[¶](#bpy.types.FieldSettings.radial_max "Link to this definition")
    :   Maximum radial distance for the field to work

        Type:
        :   float in [0, 1000], default 0.0

    radial\_min[¶](#bpy.types.FieldSettings.radial_min "Link to this definition")
    :   Minimum radial distance for the field’s falloff

        Type:
        :   float in [0, 1000], default 0.0

    rest\_length[¶](#bpy.types.FieldSettings.rest_length "Link to this definition")
    :   Rest length of the harmonic force

        Type:
        :   float in [0, inf], default 0.0

    seed[¶](#bpy.types.FieldSettings.seed "Link to this definition")
    :   Seed of the noise

        Type:
        :   int in [1, 128], default 0

    shape[¶](#bpy.types.FieldSettings.shape "Link to this definition")
    :   Which direction is used to calculate the effector force

        * `POINT`
          Point – Field originates from the object center.
        * `LINE`
          Line – Field originates from the local Z axis of the object.
        * `PLANE`
          Plane – Field originates from the local XY plane of the object.
        * `SURFACE`
          Surface – Field originates from the surface of the object.
        * `POINTS`
          Every Point – Field originates from all of the vertices of the object.

        Type:
        :   enum in [`'POINT'`, `'LINE'`, `'PLANE'`, `'SURFACE'`, `'POINTS'`], default `'POINT'`

    size[¶](#bpy.types.FieldSettings.size "Link to this definition")
    :   Size of the turbulence

        Type:
        :   float in [0, inf], default 0.0

    source\_object[¶](#bpy.types.FieldSettings.source_object "Link to this definition")
    :   Select domain object of the smoke simulation

        Type:
        :   [`Object`](Object.md#bpy.types.Object "bpy.types.Object")

    strength[¶](#bpy.types.FieldSettings.strength "Link to this definition")
    :   Strength of force field

        Type:
        :   float in [-inf, inf], default 0.0

    texture[¶](#bpy.types.FieldSettings.texture "Link to this definition")
    :   Texture to use as force

        Type:
        :   [`Texture`](Texture.md#bpy.types.Texture "bpy.types.Texture")

    texture\_mode[¶](#bpy.types.FieldSettings.texture_mode "Link to this definition")
    :   How the texture effect is calculated (RGB and Curl need a RGB texture, else Gradient will be used instead)

        Type:
        :   enum in [`'CURL'`, `'GRADIENT'`, `'RGB'`], default `'RGB'`

    texture\_nabla[¶](#bpy.types.FieldSettings.texture_nabla "Link to this definition")
    :   Defines size of derivative offset used for calculating gradient and curl

        Type:
        :   float in [0.0001, 1], default 0.0

    type[¶](#bpy.types.FieldSettings.type "Link to this definition")
    :   Type of field

        * `NONE`
          None.
        * `BOID`
          Boid – Create a force that acts as a boid’s predators or target.
        * `CHARGE`
          Charge – Spherical forcefield based on the charge of particles, only influences other charge force fields.
        * `GUIDE`
          Curve Guide – Create a force along a curve object.
        * `DRAG`
          Drag – Create a force that dampens motion.
        * `FLUID_FLOW`
          Fluid Flow – Create a force based on fluid simulation velocities.
        * `FORCE`
          Force – Radial field toward the center of object.
        * `HARMONIC`
          Harmonic – The source of this force field is the zero point of a harmonic oscillator.
        * `LENNARDJ`
          Lennard-Jones – Forcefield based on the Lennard-Jones potential.
        * `MAGNET`
          Magnetic – Forcefield depends on the speed of the particles.
        * `TEXTURE`
          Texture – Force field based on a texture.
        * `TURBULENCE`
          Turbulence – Create turbulence with a noise field.
        * `VORTEX`
          Vortex – Spiraling force that twists the force object’s local Z axis.
        * `WIND`
          Wind – Constant force along the force object’s local Z axis.

        Type:
        :   enum in [`'NONE'`, `'BOID'`, `'CHARGE'`, `'GUIDE'`, `'DRAG'`, `'FLUID_FLOW'`, `'FORCE'`, `'HARMONIC'`, `'LENNARDJ'`, `'MAGNET'`, `'TEXTURE'`, `'TURBULENCE'`, `'VORTEX'`, `'WIND'`], default `'NONE'`

    use\_2d\_force[¶](#bpy.types.FieldSettings.use_2d_force "Link to this definition")
    :   Apply force only in 2D

        Type:
        :   boolean, default False

    use\_absorption[¶](#bpy.types.FieldSettings.use_absorption "Link to this definition")
    :   Force gets absorbed by collision objects

        Type:
        :   boolean, default False

    use\_global\_coords[¶](#bpy.types.FieldSettings.use_global_coords "Link to this definition")
    :   Use effector/global coordinates for turbulence

        Type:
        :   boolean, default False

    use\_gravity\_falloff[¶](#bpy.types.FieldSettings.use_gravity_falloff "Link to this definition")
    :   Multiply force by 1/distance²

        Type:
        :   boolean, default False

    use\_guide\_path\_add[¶](#bpy.types.FieldSettings.use_guide_path_add "Link to this definition")
    :   Based on distance/falloff it adds a portion of the entire path

        Type:
        :   boolean, default False

    use\_guide\_path\_weight[¶](#bpy.types.FieldSettings.use_guide_path_weight "Link to this definition")
    :   Use curve weights to influence the particle influence along the curve

        Type:
        :   boolean, default False

    use\_max\_distance[¶](#bpy.types.FieldSettings.use_max_distance "Link to this definition")
    :   Use a maximum distance for the field to work

        Type:
        :   boolean, default False

    use\_min\_distance[¶](#bpy.types.FieldSettings.use_min_distance "Link to this definition")
    :   Use a minimum distance for the field’s falloff

        Type:
        :   boolean, default False

    use\_multiple\_springs[¶](#bpy.types.FieldSettings.use_multiple_springs "Link to this definition")
    :   Every point is affected by multiple springs

        Type:
        :   boolean, default False

    use\_object\_coords[¶](#bpy.types.FieldSettings.use_object_coords "Link to this definition")
    :   Use object/global coordinates for texture

        Type:
        :   boolean, default False

    use\_radial\_max[¶](#bpy.types.FieldSettings.use_radial_max "Link to this definition")
    :   Use a maximum radial distance for the field to work

        Type:
        :   boolean, default False

    use\_radial\_min[¶](#bpy.types.FieldSettings.use_radial_min "Link to this definition")
    :   Use a minimum radial distance for the field’s falloff

        Type:
        :   boolean, default False

    use\_root\_coords[¶](#bpy.types.FieldSettings.use_root_coords "Link to this definition")
    :   Texture coordinates from root particle locations

        Type:
        :   boolean, default False

    use\_smoke\_density[¶](#bpy.types.FieldSettings.use_smoke_density "Link to this definition")
    :   Adjust force strength based on smoke density

        Type:
        :   boolean, default False

    wind\_factor[¶](#bpy.types.FieldSettings.wind_factor "Link to this definition")
    :   How much the force is reduced when acting parallel to a surface, e.g. cloth

        Type:
        :   float in [0, 1], default 0.0

    z\_direction[¶](#bpy.types.FieldSettings.z_direction "Link to this definition")
    :   Effect in full or only positive/negative Z direction

        Type:
        :   enum in [`'POSITIVE'`, `'NEGATIVE'`, `'BOTH'`], default `'BOTH'`
