# Region(bpy\_struct)[¶](#region-bpy-struct "Link to this heading")

base class — [`bpy_struct`](bpy_struct.md#bpy.types.bpy_struct "bpy.types.bpy_struct")

*class* bpy.types.Region(*bpy\_struct*)[¶](#bpy.types.Region "Link to this definition")
:   Region in a subdivided screen area

    active\_panel\_category[¶](#bpy.types.Region.active_panel_category "Link to this definition")
    :   The current active panel category, may be Null if the region does not support this feature (NOTE: these categories are generated at runtime, so list may be empty at initialization, before any drawing took place)

        * `UNSUPPORTED`
          Not Supported – This region does not support panel categories.

        Type:
        :   enum in [`'UNSUPPORTED'`], default `'UNSUPPORTED'`

    alignment[¶](#bpy.types.Region.alignment "Link to this definition")
    :   Alignment of the region within the area

        * `NONE`
          None – Don’t use any fixed alignment, fill available space.
        * `TOP`
          Top.
        * `BOTTOM`
          Bottom.
        * `LEFT`
          Left.
        * `RIGHT`
          Right.
        * `HORIZONTAL_SPLIT`
          Horizontal Split.
        * `VERTICAL_SPLIT`
          Vertical Split.
        * `FLOAT`
          Float – Region floats on screen, does not use any fixed alignment.
        * `QUAD_SPLIT`
          Quad Split – Region is split horizontally and vertically.

        Type:
        :   enum in [`'NONE'`, `'TOP'`, `'BOTTOM'`, `'LEFT'`, `'RIGHT'`, `'HORIZONTAL_SPLIT'`, `'VERTICAL_SPLIT'`, `'FLOAT'`, `'QUAD_SPLIT'`], default `'NONE'`, (readonly)

    data[¶](#bpy.types.Region.data "Link to this definition")
    :   Region specific data (the type depends on the region type)

        Type:
        :   [`AnyType`](AnyType.md#bpy.types.AnyType "bpy.types.AnyType"), (readonly)

    height[¶](#bpy.types.Region.height "Link to this definition")
    :   Region height

        Type:
        :   int in [0, 32767], default 0, (readonly)

    type[¶](#bpy.types.Region.type "Link to this definition")
    :   Type of this region

        Type:
        :   enum in [Region Type Items](../../../appendix/bpy_types_enum_items/region_type_items.md#rna-enum-region-type-items), default `'WINDOW'`, (readonly)

    view2d[¶](#bpy.types.Region.view2d "Link to this definition")
    :   2D view of the region

        Type:
        :   [`View2D`](View2D.md#bpy.types.View2D "bpy.types.View2D"), (readonly, never None)

    width[¶](#bpy.types.Region.width "Link to this definition")
    :   Region width

        Type:
        :   int in [0, 32767], default 0, (readonly)

    x[¶](#bpy.types.Region.x "Link to this definition")
    :   The window relative vertical location of the region

        Type:
        :   int in [-inf, inf], default 0, (readonly)

    y[¶](#bpy.types.Region.y "Link to this definition")
    :   The window relative horizontal location of the region

        Type:
        :   int in [-inf, inf], default 0, (readonly)

    tag\_redraw()[¶](#bpy.types.Region.tag_redraw "Link to this definition")
    :   tag\_redraw

    tag\_refresh\_ui()[¶](#bpy.types.Region.tag_refresh_ui "Link to this definition")
    :   tag\_refresh\_ui

    :   Parameters:
        :   **id** (*str*) – The RNA type identifier.

        Returns:
        :   The RNA type or default when not found.

        Return type:
        :   [`bpy.types.Struct`](Struct.md#bpy.types.Struct "bpy.types.Struct") subclass
