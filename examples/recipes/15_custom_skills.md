# 15_custom_skills — 自定义技能包制作

## 目标

把高频工作流程沉淀为可复用技能包（Skill）。

## 前置条件

- 明确一个高频重复场景
- 拥有基础 Prompt 设计经验

## 操作步骤

1. 识别重复任务：输入、处理、输出三个环节。
2. 定义技能边界：能做什么、不能做什么。
3. 编写技能文档：场景、参数、示例、异常处理。
4. 使用 3 个真实样本回归验证输出质量。
5. 迭代版本号并维护变更日志。

## 验收标准

- 技能可在 3 个不同场景复用
- 输入参数明确，错误提示可理解
- 输出结构稳定且便于自动化串联

## 常见问题

- 目标过大：拆分为多个小技能后组合
- 结果不稳定：收紧输入约束与输出模板
