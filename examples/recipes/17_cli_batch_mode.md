# 17_cli_batch_mode — CLI 批处理实战

## 目标

通过命令行在无界面模式下批量执行 Blender 任务。

## 前置条件

- Blender 支持命令行调用
- 已准备好任务脚本或 Pipeline 配置

## 操作步骤

1. 先在单文件上验证命令行参数正确性。
2. 设计批量输入清单（文件列表、输出目录、参数集）。
3. 使用 shell/powershell 脚本循环调用 Blender CLI。
4. 为每个任务写日志并记录退出码。
5. 汇总失败任务并自动重试一次。

## 验收标准

- 批处理可稳定跑完目标任务集
- 每个任务有独立日志可回溯
- 失败重试机制可用

## 常见问题

- 路径问题：统一使用绝对路径并转义空格
- 执行中断：增加超时与断点续跑策略
