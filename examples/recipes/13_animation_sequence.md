# 13_animation_sequence — 复杂动画序列

## 目标

组织多镜头、多对象的动画序列并保持节奏连贯。

## 前置条件

- 已具备关键帧基础
- 确认分镜（shot list）

## 操作步骤

1. 先建立分镜时间表（Shot01/02/03 的起止帧）。
2. 按镜头分集合管理对象，减少误操作。
3. 在 Dope Sheet 统一关键帧节奏，再到 Graph Editor 调细节。
4. 使用 Marker 标记关键事件（入场、转场、动作点）。
5. 输出 Playblast 做节奏评估，必要时整体重定时。

## 验收标准

- 镜头衔接自然，无突兀跳变
- 关键动作点与音乐/叙事节奏对齐
- 序列可稳定预览并可批量渲染

## 常见问题

- 节奏拖沓：缩短非关键信息镜头时长
- 关键帧杂乱：按镜头与对象分层管理轨道

## 可复用 Prompt
