# 16_cursor_workflow — Cursor + Blender 工作流

## 目标

把代码侧自动化（Cursor）与 Blender 场景操作串联成高效闭环。

## 前置条件

- 已安装 Cursor 与 Blender
- 熟悉基本命令行操作

## 操作步骤

1. 在 Cursor 中维护场景配置与脚本模板。
2. 将可重复操作抽成脚本（如批量命名、批量导出）。
3. 通过终端调用脚本并在 Blender 执行。
4. 把输出结果（日志、截图）回写到项目文档。
5. 形成“修改配置 → 执行脚本 → 结果回查”的循环。

## 验收标准

- 至少 1 个任务从手工变为脚本化
- 执行结果可追踪（日志/版本）
- 工作流可由他人按文档复现

## 常见问题

- 脚本耦合过重：拆分编排层与执行层
- 结果不可追踪：统一输出目录与时间戳
