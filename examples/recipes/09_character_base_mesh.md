# 09_character_base_mesh — 角色基础网格

## 目标

创建一个可继续雕刻和绑定的角色基础网格。

## 前置条件

- 具备基础人体比例概念
- 准备正/侧视参考图（推荐）

## 操作步骤

1. 从低面数立方体开始，应用 `Mirror` 保持左右对称。
2. 先做大体块（头、胸腔、骨盆、四肢）再做连接。
3. 在肩、肘、膝、髋关节预留可变形边环。
4. 通过 `Subdivision` 检查形体连续性，不急于细节雕刻。
5. 输出 base mesh 并冻结版本号（如 `char_base_v01.blend`）。

## 验收标准

- 整体比例可读，轮廓稳定
- 关节处拓扑连续，不出现极端极点
- 可直接进入下一步雕刻或绑定

## 常见问题

- 体块比例失衡：先重看正侧轮廓再调局部
- 边流混乱：减少无意义切线，优先保留主流向
