# 05_animation_basics — 动画基础

## 目标

制作一个 2 秒的基础物体位移动画并输出预览。

## 前置条件

- 任意对象（建议 `Cube_Main`）
- 时间轴帧率设为 24 FPS

## 操作步骤

1. 将时间轴定位到第 `1` 帧，按 `I` 插入 `Location` 关键帧。
2. 跳到第 `48` 帧（2 秒），将对象沿 X 轴移动 3 米。
3. 再按 `I` 插入 `Location` 关键帧。
4. 打开 Graph Editor，把插值改为 `Linear`（用于测试）或 `Bezier`（更自然）。
5. 空格播放动画并检查运动轨迹。

## 验收标准

- 对象从起点平滑移动到终点
- 时间线中至少有两个有效关键帧
- 预览动画时无跳帧或突变

## 常见问题

- 对象不动：检查是否在同一属性上打帧（Location）
- 动画过冲：检查插值曲线手柄

## 可复用 Prompt

> 帮我给物体做一个 2 秒位移动画，并告诉我如何控制节奏。
